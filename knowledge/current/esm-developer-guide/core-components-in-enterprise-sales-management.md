# Core Components in Enterprise Sales Management

Enterprise Sales Management provides a library of reusable core components that are used as the building blocks for this implementation. Core components handle specific functions used across the various flows built in the solution. You can leverage these core components when you build customized solutions in your projects.

Enterprise Sales Management is built on top of the latest ​Salesforce Industries Communications, Media, and Energy (CME) release, including:

-   Cards, Lightning Web Components, the Newport Design System (NDS), Templates, and Omniscripts for extending the UI and guided flows

-   APIs and Integration Procedures for retrieving and processing data

-   CME Communications data model, which includes extensions for Enterprise Sales Management

-   Enterprise Product Catalog for commercial and technical catalog modeling

-   Configure, Price, Quote for cart operations, validation, pricing, order creation, and configuration

-   Contract Lifecycle Management for contract-based negotiation, ordering, and updates

-   Industries Order Management for order decomposition and orchestration


-   **[APIs for Enterprise Sales Management](atlas.en-us.esm_developer_guide.meta/esm_developer_guide/comms_t_apis_for_enterprise_sales_management_64311.htm)**
    You can use the Enterprise Sales Management APIs to create and manage enterprise quotes by calling corresponding Integration Procedures.
-   **[Integration Procedures for Enterprise Sales Management](atlas.en-us.esm_developer_guide.meta/esm_developer_guide/comms_t_integration_procedures_for_enterprise_salesmanagement_65388.htm)**
    Every Enterprise Sales Management org includes a number of Integration Procedures that are used by Enterprise Sales Management.
-   **[Lightning Web Components for Enterprise Sales Management](atlas.en-us.esm_developer_guide.meta/esm_developer_guide/comms_t_lightning_web_components_for_enterprise_salesmanagement_65423.htm)**
    Enterprise Sales Management Lightning Web Components (LWCs) supplement Vlocity Lightning Web Components by incorporating calls to the Enterprise Sales Management SDK, which makes calls to Enterprise Sales Management APIs, Cart-Based APIs, Digital Commerce cacheable APIs, and Integration Procedures.
-   **[Omniscripts for Enterprise Sales Management](atlas.en-us.esm_developer_guide.meta/esm_developer_guide/comms_t_omniscripts_for_enterprise_sales_management_65607.htm)**
    Enterprise Sales Management uses Omniscripts to implement key business processes, such as to create a quote or generate a proposal document.
-   **[SDK for Enterprise Sales Management](atlas.en-us.esm_developer_guide.meta/esm_developer_guide/comms_t_sdk_for_enterprise_sales_management_65685.htm)**
    The Enterprise Sales Management Software Development Toolkit (SDK) is a JavaScript library that abstracts and simplifies the use of Enterprise Sales Management, Cart-Based, and Digital Commerce APIs.
