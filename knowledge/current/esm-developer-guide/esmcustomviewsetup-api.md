# ESM\_CustomViewSetup API

Store information about the custom views created for the Summary page.

This API is used to determine the fields shown on the Summary page for each option selected in the **Displaying** list. It also determines the OmniScript that is launched by the **Enrich Quote** button for each option selected in the **Displaying** list.

The information stored by this API includes:

-   The list of read-only fields shown on the Summary page.

-   The list of fields  displayed on the Summary page for each option selected in the **Displaying** list.

-   The name of the  OmniScript launched for each option selected in the **Displaying** list. The relevant OmniScript is launched when you click **Enrich Quote**.
