# Einstein Discovery REST Resources

REST API resources are sometimes called endpoints.

-   **[Smart Data Discovery Resource](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_smart_data_discovery.htm)**
    Lists the top-level resources available for Einstein Discovery.
-   **[Model Resources](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_models_overview.htm)**
    Einstein Discovery models are sophisticated, custom mathematical constructs that are used to predict particular outcomes.
-   **[Metrics Resource](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_metrics.htm)**
    Returns the metrics collection for Einstein Discovery.
-   **[Narrative Resource](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_narrative.htm)**
    Returns the narrative data for an Einstein Discovery story.
-   **[Prediction Definition Resources](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_prediction_definition_overview.htm)**
    A prediction definition specifies what the model is trying to predict and the Salesforce entity associated with the prediction. Each prediction definition has a unique id. Only certain attributes of a prediction definition can be modified.
-   **[Predict Jobs Resources](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_predict_jobs_overview.htm)**
    Predict jobs run bulk scoring jobs for predictions. Bulk scoring jobs enable scoring predictions on multiple records at a time. For example, after deploying an updated model, use bulk scoring to refresh all prediction scores.
-   **[Predict Resource](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_predict.htm)**
    Make an Einstein Discovery prediction.
-   **[Predict History Resource](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_predict_history.htm)**
    Query the history of Einstein Discovery predictions.
-   **[Refresh Jobs Resources](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_refresh_jobs_overview.htm)**
    Refresh jobs run refresh Einstein Discovery models.
-   **[Stories Resources](atlas.en-us.bi_dev_guide_rest_sdd.meta/bi_dev_guide_rest_sdd/bi_sdd_resources_stories_overview.htm)**
    Einstein Discovery stories are used to analyze your data. The insights from stories show you what happened in your data, why it happened, and what could happen.
