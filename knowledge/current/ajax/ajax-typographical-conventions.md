# AJAX Typographical Conventions

# AJAX Typographical Conventions

Topics about the AJAX Toolkit use the following typographical conventions:

| Convention | Description |
| --- | --- |
| <script src="/soap/ajax/66.0/connection.js" type="text/javascript"></script> | In an example, Courier font indicates items that you should type the information as shown. This includes sample code, literals, methods, calls, functions, and events from a variety of languages. |
| sforce.connection.header_option_name="value"; | In an example or syntax statement, italics represent variables. You supply the actual value. |
