# COA\_ServiceOrderCloneInput Properties

The following are properties for COA\_ServiceOrderCloneInput.

-   **[serviceOrderId](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderCloneInput_serviceOrderId.htm)**
    Specifies the ID of the order you are cloning. This field is required.
-   **[cloneProducts](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderCloneInput_cloneProducts.htm)**
    Indicates whether to clone the original order’s line items. If true, the line items are cloned. This field is required.
