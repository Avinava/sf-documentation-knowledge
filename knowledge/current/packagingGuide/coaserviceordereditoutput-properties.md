# COA\_ServiceOrderEditOutput Properties

The following are properties for COA\_ServiceOrderEditOutput.

-   **[isSuccess](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderEditOutput_isSuccess.htm)**
    Indicates the success of the edit operation. If true, the order was successfully edited.
-   **[responseMessages](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderEditOutput_responseMessages.htm)**
    Holds response messages generated by the edit operation.
-   **[serviceOrderId](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderEditOutput_serviceOrderId.htm)**
    References the order ID passed in by the edit operation.
