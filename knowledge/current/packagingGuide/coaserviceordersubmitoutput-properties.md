# COA\_ServiceOrderSubmitOutput Properties

The following are properties for COA\_\_ServiceOrderSubmitOutput.

-   **[isSuccess](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderSubmitOutput_isSuccess.htm)**
    Indicates the success of the submit operation. If true, the order was successfully submitted.
-   **[responseMessages](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderSubmitOutput_responseMessages.htm)**
    Holds response messages generated by the submit operation.
-   **[serviceOrderId](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderSubmitOutput_serviceOrderId.htm)**
    References the order ID passed in by the submit operation.
