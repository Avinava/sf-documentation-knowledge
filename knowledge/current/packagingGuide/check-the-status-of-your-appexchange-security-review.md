# Check the Status of Your AppExchange Security Review

Find the status of your security review in the AppExchange Partner Console. Status updates appear after you submit the solution for review.


| User Permissions Needed |
| --- |
| To access the AppExchange Partner Console: | Manage Listings |

1.  Log in to the [Salesforce Partner Community](https://partners.salesforce.com/ "HTML (New Window)").
2.  To launch the Partner Console, click **Publishing**.
3.  Click **Technologies** | **Solutions**.
4.  Locate the security review column. If your solution isn’t subject to security review, the column is omitted.
5.  View the review status (1).

    ![A screenshot of the Solutions page in the Partner Console highlighting the status field in the security review column](/docs/resources/img/en-us/260.0?doc_id=appexchange%2Fsecurity_review%2Fimages%2Fsecurity_reviews_on_solutions_tab.png&folder=packagingGuide)

6.  To see additional status information for an in-progress review, click **Check Status**.

    The security review wizard’s Overview page loads. It contains detailed status, feedback from the review teams, and recommended action items.
