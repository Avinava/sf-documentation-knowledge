# COA\_ServiceOrderRecallOutput Properties

The following are properties for COA\_ServiceOrderRecallOutput.

-   **[isSuccess](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderRecallOutput_isSuccess.htm)**
    Indicates the success of the recall operation. If true, the order was successfully recalled.
-   **[responseMessages](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderRecallOutput_responseMessages.htm)**
    Holds response messages generated by the recall operation.
-   **[serviceOrderId](atlas.en-us.packagingGuide.meta/packagingGuide/apex_COA_ServiceOrderRecallOutput_serviceOrderId.htm)**
    References the order ID passed in by the recall operation.
