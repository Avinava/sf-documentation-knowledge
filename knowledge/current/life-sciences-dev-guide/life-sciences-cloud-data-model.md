# Life Sciences Cloud Data Model

Life Sciences Cloud provides a rich set of objects to store and access specialized Life Sciences information.

To verify the complete list of fields for an object, you can use a describe call from the API, or inspect with an appropriate tool, for example, inspecting the WSDL or using a schema viewer.

-   **[Clinical Engagement](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/life_sciences_clinical_engagement.htm)**
    These objects are available for Life Sciences Cloud for Clinical Engagement.
-   **[Customer Engagement](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/life_sciences_customer_engagement_data_model.htm)**
    These objects are available for Life Sciences Cloud for Customer Engagement.
-   **[Patient Engagement](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/life_sciences_patient_engagement_data_model.htm)**
    These objects are available for Life Sciences Cloud for Patient Engagement.
-   **[MedTech Commercial Engagement](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/life_sciences_medtec_commercial_engagement.htm)**
    These objects are available for Life Sciences Cloud for MedTech Commercial Engagement.
