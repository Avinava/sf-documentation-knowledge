# ConnectApi Namespace

The ConnectApi namespace (also called Connect in Apex) provides classes for accessing the same data available in Connect REST API. Use Connect in Apex to create custom experiences in Salesforce.

For information about working with the ConnectApi classes, see [Connect in Apex](https://developer.salesforce.com/docs/atlas.en-us.260.0.apexcode.meta/apexcode/connectAPI_overview.htm).

-   **[LifeSciEmailTemplate Methods](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/apex_LifeSciEmailTemplate_methods.htm)**
    These are the LifeSciEmailTemplate methods for Life Sciences Cloud.
-   **[LifeSciPresentation Methods](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/apex_LifeSciPresentation_methods.htm)**
    These are the LifeSciPresentation methods for Life Sciences Cloud.
-   **[LifeSciPresPageProductGuidance Methods](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/apex_LifeSciPresPageProductGuidance_methods.htm)**
    These are the LifeSciPresPageProductGuidance methods for Life Sciences Cloud.
-   **[Input Classes](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/ls_apex_connect_api_input_classes.htm)**
    These are the Apex input classes for Life Sciences Cloud.
-   **[Output Classes](atlas.en-us.life_sciences_dev_guide.meta/life_sciences_dev_guide/ls_apex_connect_api_output_classes.htm)**
    These are the Apex output classes for Life Sciences Cloud.
