# Work with the Console APIs for Open CTI

There are console-specific methods that you can use to interact with Open CTI. Use the Salesforce Console Integration Toolkit JavaScript APIs to interact with Salesforce Classic console apps.

For more information, see the [Methods for Computer-Telephony Integration (CTI)](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_console.meta/api_console/sforce_api_console_methods_cti.htm "HTML (New Window)") in the Salesforce Console Developer Guide.
