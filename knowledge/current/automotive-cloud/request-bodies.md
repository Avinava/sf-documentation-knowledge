# Request Bodies

Here’s a list of Automotive Cloud Business API request bodies.

-   **[Additional Criteria Input](atlas.en-us.automotive_cloud.meta/automotive_cloud/connect_requests_additional_criteria.htm)**
    Input representation of the additional event criteria generated during inbound events.
-   **[Field List](atlas.en-us.automotive_cloud.meta/automotive_cloud/connect_requests_field_list.htm)**
    Input representation of a field match that contains the field name and value.
-   **[Inbound Event Input](atlas.en-us.automotive_cloud.meta/automotive_cloud/connect_requests_inbound_event_input.htm)**
    Input representation of the request to process incoming events from a telematics provider.
-   **[Inventory Actions Query Input](atlas.en-us.automotive_cloud.meta/automotive_cloud/connect_requests_inventory_actions_query.htm)**
    Input representation to perform the inventory actions query.
-   **[Inventory Actions Item Input](atlas.en-us.automotive_cloud.meta/automotive_cloud/connect_requests_inventory_actions_input_item.htm)**
    Input representation to perform the inventory product transfer.
-   **[Transformation Input](atlas.en-us.automotive_cloud.meta/automotive_cloud/auto_request_body_transformation_input.htm)**
    Input representation for converting an input object type to an output object type.
