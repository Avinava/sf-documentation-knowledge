| Property | Type | Description | Filter Group and Version | Available Version |

| errorCode | String | An error code with the information about the error, for example, INSUFFICIENT_PRIVILEGES. | Small, 29.0 | 23.0 |
| message | String | Description of the error. | Small, 29.0 | 23.0 |
