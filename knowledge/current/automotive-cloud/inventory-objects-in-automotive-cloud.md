# Inventory Objects in Automotive Cloud

Automotive Cloud provides access to standard objects that you can use to organize products, track their movement, and monitor suppliers. You can access these objects when Automotive Cloud is enabled in your org.

| Available in: Enterprise, Unlimited, and Developer Editions. |
| --- |
