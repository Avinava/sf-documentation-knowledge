# Action Plans in Automotive Cloud

Use the Action Plans data model to capture commonly repeated tasks and metrics for partner visits. You can also use action plans for vehicles, asset milestones, and asset account and contact participants.

| Available in: Enterprise, Unlimited, and Developer Editions. |
| --- |
