# Action Launcher in Automotive Cloud

Add actions in the Action Launcher Lightning web component so that your users can search for and launch an action easily. The Action Launcher can be used on record pages like Vehicle, Asset, or Account, and in the Service Console for Automotive app.

For more information about the developer resources of Action Launcher, see [Action Launcher](https://developer.salesforce.com/docs/atlas.en-us.260.0.industries_reference.meta/industries_reference/action_launcher.htm).
