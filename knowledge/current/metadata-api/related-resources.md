# Related Resources

The Salesforce developer website provides a full suite of developer toolkits, sample code, sample SOAP messages, community-based support, and other resources to help you with your development projects. Be sure to visit [https://developer.salesforce.com/page/Getting\_Started](https://developer.salesforce.com/page/Getting_Started) for more information, or visit [https://developer.salesforce.com/signup](https://developer.salesforce.com/signup) to sign up for a free Developer Edition account.

You can visit these websites to find out more about Salesforce applications:

-   [Salesforce Developers](https://developer.salesforce.com/page/Tools "HTML (New Window)") provides useful information for developers.
-   [Salesforce](https://salesforce.com) for information about the Salesforce application.
-   [Lightning Platform AppExchange](https://appexchange.salesforce.com/) for access to apps created for Salesforce.
-   [Trailblazer Community](https://trailhead.salesforce.com/en/trailblazer-community/feed) for services to ensure Salesforce customer success.
