# CustomObjectTranslation Language Support: End-User Languages

Not every language supports all the possible values for the fields in [CustomObjectTranslation](atlas.en-us.api_meta.meta/api_meta/meta_customobjecttranslation.htm "This metadata type allows you to translate custom objects for a variety of languages."). Use this appendix to determine which field values a language supports.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=api_meta)

#### Note

Salesforce offers three levels of language support: fully supported languages, end-user languages, and platform-only languages. This appendix provides information only for end-user languages.
