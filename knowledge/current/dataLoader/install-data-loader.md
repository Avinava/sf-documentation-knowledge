# Install Data Loader

Use Data Loader to insert, update, delete, and export Salesforce records.

| Available in: both Salesforce Classic (not available in all orgs) and Lightning Experience |
| --- |
| Available in: Enterprise, Performance, Unlimited, and Developer editions |


-   **[Download and Install Data Loader](atlas.en-us.dataLoader.meta/dataLoader/loader_install_mac.htm)**
    Data Loader is available for MacOS and Windows operating systems. Download to install and configure the app on your local machine.
-   **[Considerations for Installing Data Loader](atlas.en-us.dataLoader.meta/dataLoader/installing_the_data_loader.htm)**
    Before you download and install Data Loader, understand the installation and login considerations. Each release of Data Loader has its own list of supported operating systems and hardware requirements.
-   **[Uninstall Data Loader](atlas.en-us.dataLoader.meta/dataLoader/loader_uninstall.htm)**
    You can uninstall Data Loader from your Windows or macOS machine.
