# sf

This section contains information about the sf commands for the analytics plug-in.

This version of the sf command reference includes details about version 1.4.16 of the sf analytics executable of the Salesforce CLI @salesforce/analytics plug-in.

-   **[analytics enable](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_enable_unified.htm)**
    Enable Analytics in the target org.
-   **[analytics query](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_query_unified.htm)**
    Execute an Analytics query.
-   **[app Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_app_unified.htm#bi_cli_reference_analytics_app_unified)**
    Create and manage Analytics apps.
-   **[asset Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_asset_unified.htm#bi_cli_reference_analytics_asset_unified)**
    Manage Analytics asset publishing.
-   **[autoinstall Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_autoinstall_unified.htm#bi_cli_reference_analytics_autoinstall_unified)**
    Manage Analytics auto-install requests.
-   **[dashboard Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_dashboard_unified.htm#bi_cli_reference_analytics_dashboard_unified)**
    Manage Analytics dashboards.
-   **[dataflow Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_dataflow_unified.htm#bi_cli_reference_analytics_dataflow_unified)**
    Manage Analytics dataflows and dataflow jobs.
-   **[dataset Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_dataset_unified.htm#bi_cli_reference_analytics_dataset_unified)**
    Manage Analytics datasets.
-   **[generate Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_generate_unified.htm#bi_cli_reference_analytics_generate_unified)**
    Generate simple Analytics assets.
-   **[lens Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_lens_unified.htm#bi_cli_reference_analytics_lens_unified)**
    Manage Analytics lenses.
-   **[recipe Commands](atlas.en-us.bi_dev_guide_cli_reference.meta/bi_dev_guide_cli_reference/bi_cli_reference_analytics_recipe_unified.htm#bi_cli_reference_analytics_recipe_unified)**
