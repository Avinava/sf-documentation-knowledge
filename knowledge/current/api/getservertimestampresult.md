# getServerTimestampResult

Returns a GetServerTimestampResult object.

The [getServerTimestamp()](atlas.en-us.api.meta/api/sforce_api_calls_getservertimestamp.htm "Retrieves the current system timestamp (Coordinated Universal Time (UTC) time zone) from the API.") call returns a GetServerTimestampResult object, which has the following properties:

| Name | Type | Description |
| --- | --- | --- |
| timestamp | dateTime | System timestamp of the API when the getServerTimestamp() call was executed. |
