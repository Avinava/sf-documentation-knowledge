# Message Segment: Field Change Name

Represents an inline name of a field change, like “Name” or “Description.” Making this a separate segment makes it possible to give special user interface treatments to field names, for example, making a name field bold.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| text | String | Formatted text of the change. Possible values include:Old name of the fieldNew name of the field | Small, 29.0 | 23.0 |
| type | String | Value is FieldChangeName | Small, 29.0 | 23.0 |
