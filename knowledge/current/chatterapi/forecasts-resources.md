# Forecasts Resources

Get information about forecast users, managers, and the forecast hierarchy.

Available resources are:

| Resource | Description |
| --- | --- |
| /connect/forecasting/ancestors/forecastingTypeId | Get a list of managers or territories between the context user and a specified user or territory in the forecast hierarchy. |
| /connect/forecasting/jump-to-user/forecastingTypeId | Get a list of users and territories whose forecasts the context user has access to view. |
| /connect/forecasting/my-forecasts-list/forecastingTypeId | Get the appropriate list of forecasts for a given user’s dropdown menu on the forecasts page. |
| /connect/forecasting/user-access/userId/​forecastingTypeId | Get information about the context user’s access to the user to which you need access. |

-   **[Forecasting Jump-To-User-Or-Territory](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_forecasting_jump_to_user_or_territory.htm)**
    Get a list of users and territories whose forecasts the context user has access to view.
-   **[Forecasting Manager Parents by User Role or Territory ID](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_forecasting_manager_parent_by_user_or_territory_id.htm)**
    Get a list of managers or territories between the context user and a specified user or territory in the forecast hierarchy.
-   **[Forecasting My Forecasts List](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_forecasting_my_forecasts_list.htm)**
    Get the appropriate list of forecasts for a given user’s dropdown menu on the forecasts page.
-   **[Forecasting User Access by Forecasting Type](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_forecasting_user_access_by_forecasting_type.htm)**
    Get information about the context user’s access to the user to which you need access.
