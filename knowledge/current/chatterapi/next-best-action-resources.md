# Next Best Action Resources

Get a recommendation, execute a recommendation strategy, and manage reactions to recommendations.

Community users can't access this resource. Portal and guest users can access strategies only through the Suggested Actions component.

Available resources are:

| Resource | Description |
| --- | --- |
| /connect/recommendations/recommendationId | Get a recommendation. |
| /connect/recommendation-strategies/​strategyName/recommendations | Execute a strategy. |
| /connect/recommendation-strategies/reactions | Get and record user reactions to recommendations. |
| /connect/recommendation-strategies/reactions/​reactionId | Get and delete a recommendation reaction. |
