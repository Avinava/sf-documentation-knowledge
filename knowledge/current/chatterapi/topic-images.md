# Topic Images

Images associated with a topic.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| coverImageUrl | String | URL to a topic’s cover image, which appears on the topic page. Both topics and managed topics can have cover images. | Small, 32.0 | 32.0 |
| featuredImageUrl | String | URL to a managed topic’s featured image, which appears wherever you feature it, for example, on your Experience Cloud site home page. | Small, 32.0 | 32.0 |
