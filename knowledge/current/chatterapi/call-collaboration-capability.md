# Call Collaboration Capability

If a feed element has this capability, it has a recording comment.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| commentEndTime | Integer | End time of the comment on the media player, in seconds. | Small, 51.0 | 51.0 |
| commentStartTime | Integer | Start time of the comment on the media player, in seconds. | Small, 51.0 | 51.0 |
