# Topics Capability

If a feed element has this capability, the context user can add topics to it. Topics help users organize and discover conversations.

| Property Name | Type | Description | Filter and Version | Available Version |
| --- | --- | --- | --- | --- |
| canAssignTopics | Boolean | true if a topic can be assigned to the feed element, false otherwise. | Small, 32.0 | 32.0 |
| items | Topic[] | A collection of topics associated with this feed element. | Small, 32.0 | 32.0 |
