# Text Classifications

HTTP headers containing URLs associating text strings and classifications.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| httpHeaders | HTTP Header | HTTP headers for text classifications output. Each header provides a URL you can use to get the result of the classification. The URL takes a list of text strings and classifiers that each text string can be classified in. | Big, 59.0 | 59.0 |
