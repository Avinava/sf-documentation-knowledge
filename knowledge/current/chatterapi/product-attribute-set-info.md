# Product Attribute Set Info

#### Important

Where possible, we changed noninclusive terms to align with our company value of Equality. We maintained certain terms to avoid any effect on customer implementations.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| attributeInfo | Map<String, Product Attribute Info> | Map of the API name of the attribute field to the attribute metadata. | Small, 50.0 | 50.0 |
| description | String | Description of the attribute set. | Small, 50.0 | 50.0 |
| developerName | String | Developer name of the attribute set. | Small, 50.0 | 50.0 |
| id | String | ID of the attribute set. | Small, 50.0 | 50.0 |
| masterLabel | String | Label of the attribute set. | Small, 50.0 | 50.0 |
| sequence | Integer | Sequence of the attribute set for the product. | Small, 50.0 | 50.0 |
