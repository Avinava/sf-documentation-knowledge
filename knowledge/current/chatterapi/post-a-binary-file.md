# Post a Binary File

Post a binary file.

For detailed information about uploading and posting binary files, see [Inputs and Binary File Upload Examples](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm "Make HTTP POST, PATCH, and PUT requests using inputs and learn more about uploading binary files."), which includes examples for:

-   [Post a comment with a binary file](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#postComment)
-   [Post a batch of comments with binary files](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#genericBatchExampleSection)
-   [Upload and crop a user photo](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#fileUploadExample)
-   [Upload a binary file to the Files home](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#fileDataExample)
-   [Upload a file to an external repository](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#fileDataExternalRepository)

![Important](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note_important.png&folder=chatterapi)

#### Important

In version 36.0 and later, you can’t create a feed post and upload a binary file in the same request. [Upload files to Salesforce](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#fileDataExample) first, and then use the file IDs to attach one or more files to a feed post.

-   [Post a feed element with a binary file](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#feedElementUploadExample) (version 35.0 and earlier)
-   [Post a batch of feed elements with binary files](atlas.en-us.chatterapi.meta/chatterapi/intro_input.htm#feedElementBatchSection) (version 35.0 and earlier)
