# Record Summary

Record summary.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| entityLabel | Entity Label | Label of the record’s entity. | Small, 40.0 | 40.0 |
| id | String | 18-character ID of the record. | Small, 29.0 | 23.0 |
| motif | Motif | Small, medium, and large icons indicating the tab style of the record. The motif can also contain the record type’s base color. | Small, 29.0 | 27.0 |
| mySubscription | Reference | If the context user is following this record, contains information about the subscription; otherwise returns null. | Medium, 29.0 | 24.0 |
