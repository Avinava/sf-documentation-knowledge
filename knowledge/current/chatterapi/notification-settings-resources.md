# Notification Settings Resources

Get, set, and reset notification settings and notification app settings.

Available resources are:

| Resource | Description |
| --- | --- |

| /connect/notifications/settings/organization/​notificationTypeOrId | Get, set, and reset a notification setting for the org. |
| /connect/notifications/app-settings/​organization | Get notification app settings for the org. |
| /connect/notifications/app-settings/​organization/notificationTypeOrId | Get, set, and reset a notification app setting for the org. |
