# Social Customer Service Message Type Settings

Social Customer Service settings related to the behavior of specific message types.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| isEnabled | Boolean | Specifies whether the message type is sent to Social Customer Service (true) or not (false). | Small, 45.0 | 45.0 |
| messageType | String | Specifies the message type, such as PrivateMessage. | Small, 45.0 | 45.0 |
