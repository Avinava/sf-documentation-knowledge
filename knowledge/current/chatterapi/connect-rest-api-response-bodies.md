# Connect REST API Response Bodies

The successful execution of a request to a Connect REST API resource can return a response body in either JSON or XML format.

A request to a Connect REST API resource always returns an [HTTP response code](atlas.en-us.chatterapi.meta/chatterapi/connect_error_responses.htm "The response header contains an HTTP status code. If the response isn’t successful, the response body contains an error message, and if appropriate, information about the field or object where the error occurred."), whether the request was successful or not.
