# Icon

Icon.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| height | Integer | The height of the icon, in pixels | Small, 29.0 | 27.0 |
| width | Integer | The width of the icon, in pixels | Small, 29.0 | 27.0 |
| url | String | The URL of the icon. This URL is available to unauthenticated users. The URL doesn’t expire. | Small, 29.0 | 27.0 |
