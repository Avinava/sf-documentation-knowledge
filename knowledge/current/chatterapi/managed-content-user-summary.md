# Managed Content User Summary

| Property Name | Type | Description | Filter Group and Version | Available Version |

| id | String | ID of the user. | Small, 54.0 | 54.0 |
| name | String | Reserved for future use. | Small, 55.0 | 55.0 |
| resourceUrl | String | Resource URL of the user. | Small, 54.0 | 54.0 |
