# Purchase Quantity Rule

Rule that restricts the quantity of a product that can be purchased.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| increment | String | Increment value of the quantity that can be purchased. | Small, 52.0 | 52.0 |
| maximum | String | Maximum quantity that can be purchased. | Small, 52.0 | 52.0 |
| minimum | String | Minimum quantity that can be purchased. | Small, 52.0 | 52.0 |
