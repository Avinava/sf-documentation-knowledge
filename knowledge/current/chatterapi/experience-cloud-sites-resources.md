# Experience Cloud Sites Resources

Get information about Experience Cloud sites, create a site, and publish a site. Get the Experience Builder templates available to the context user. Get externally managed accounts for sites.

Available resources:

| Resource | Description |
| --- | --- |
| /connect/communities | Get a list of all Experience Cloud sites available to the context user. Create an Experience Cloud site. |
| /connect/communities/communityId | Get information about an Experience Cloud site. |
| /connect/communities/communityId/publish | Publish an Experience Cloud site. |
| /connect/communities/templates | Get the Experience Builder templates available to the context user. |
| /connect/communities/communityId​/external-managed-accounts | Get externally managed accounts available to the context user across all Experience Cloud sites. |
| /connect/communities/communityId​/preview-url/pages/pageApiName | Get a preview URL for an Experience Cloud site page. |
