# Managed Content Taxonomy Term Collection

Collection of managed content taxonomy terms.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| contentKey | String | Unique identifier of a managed content that is unchanged by org migration. | Small, 63.0 | 63.0 |
| managedContentId | String | ID of the managed content in authoring space. | Small, 63.0 | 63.0 |
| taxonomyTerms | Managed Content Taxonomy Term[] | List of taxonomy terms associated with managed content. | Small, 63.0 | 63.0 |
