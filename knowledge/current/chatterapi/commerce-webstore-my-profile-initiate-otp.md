# Commerce Webstore My Profile Initiate OTP

Initiate the one-time password (OTP) via the specified method, such as SMS or Email. The API returns a unique identifier associated with the OTP request.

This process make sures that any changes to buyer's critical information, such as email and phone number, are securely verified before the update using the [Commerce Webstore My Profile](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_commerce_webstore_myprofile.htm "Retrieve the buyer's account details. Update buyer account details. The API retrieves or updates the registered buyer’s details by identifying the buyer user’s ID within the logged-in context.") PATCH method.
