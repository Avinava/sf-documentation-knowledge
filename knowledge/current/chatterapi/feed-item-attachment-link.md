# Feed Item Attachment: Link

Link feed item attachment.

![Important](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note_important.png&folder=chatterapi)

#### Important

This response body isn’t available in version 32.0 and later. In version 32.0 and later, [Link Capability](atlas.en-us.chatterapi.meta/chatterapi/connect_responses_link_capability.htm "If a feed element has this capability, it has a link.") is used.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| title | String | Description of the link | Small, 29.0 | 23.0–31.0 |
| type | String | Link | Small, 29.0 | 27.0–31.0 |
| url | String | URL attached to the feed item (could be external) | Small, 29.0 | 23.0–31.0 |
