# Banner Capability

If this feed element has this capability, it has a banner motif and style.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| motif | Motif | A banner motif. | Small, 31.0 | 31.0 |
| style | String | Decorates a feed item with a color and set of icons.Announcement—An announcement displays in a designated location in the Salesforce UI until 11:59 p.m. on its expiration date, unless it’s deleted or replaced by another announcement. | Small, 31.0 | 31.0 |
