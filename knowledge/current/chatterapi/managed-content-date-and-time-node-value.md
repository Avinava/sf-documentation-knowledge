# Managed Content Date and Time Node Value

Managed content node of date and time type.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| dateTimeValue | String | UTC date and time value of the managed content node. | Small, 48.0 | 48.0 |
| nodeType | String | Type of managed content node. Value is DateTime. | Small, 48.0 | 48.0 |
| timeZone | String | Time zone in which the date and time is authored. | Small, 48.0 | 48.0 |
