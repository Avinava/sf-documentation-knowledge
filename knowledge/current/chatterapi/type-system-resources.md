# Type System Resources

Use Type System resources to retrieve information about the types available in your org.

Available resources are:

| Resource | Description |
| --- | --- |
| /connect/lightning-types | Get a list of all standard and custom Lightning types. |
| /connect/lightning-types/fullyQualifiedName | Get detailed information about standard and custom Lightning types. |
| /connect/lightning-model/types | Get detailed information about standard and custom Lightning types and Apex class types. |
