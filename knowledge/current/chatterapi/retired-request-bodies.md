# Retired Request Bodies

These request bodies are retired.

-   **[Attachment Input: Canvas App](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_attach_canvas.htm)**
    Canvas app attachment input.
-   **[Attachment Input: Existing Content](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_attach_existing_content.htm)**
    Existing content attachment input.
-   **[Attachment Input: Link](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_attach_link.htm)**
    Link attachment input.
-   **[Attachment Input: New File Upload](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_attach_new_file.htm)**
    New file upload attachment input.
-   **[Attachment Input: Feed Poll](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_attach_poll.htm)**
    Feed poll attachment input.
-   **[Datacloud Contact Import Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_datacloud_contact_import_input.htm)**
    Input representation of the criteria for importing records as contacts or leads.
-   **[Datacloud Order Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_datacloud_order_input.htm)**
    A list of Data.com identification numbers that are used to purchase contacts or companies by a specific user type.
-   **[Feed Poll Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_feed_poll_input.htm)**
    Feed poll input.
-   **[Promotion Association Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_promotion_association_input.htm)**
    Cart and its cart items before any promotion discounts are associated with them.
-   **[Promotion Execution Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_promotion_execution_input.htm)**
    Cart and its items, with associated promotion discounts for the items.
-   **[Social Customer Service Configuration Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_scs_config_input.htm)**
    Social Customer Service configuration options to save.
-   **[Social Customer Service Message Type Settings Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_scs_message_type_settings_input.htm)**
    Social Customer Service settings related to the behavior of specific message types.
-   **[Social Customer Service Provider Settings Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_scs_provider_settings_input.htm)**
    Social Customer Service settings related to the behavior of specific social networks.
-   **[Social Post Mass Approval Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_social_post_mass_approval_input.htm)**
    List of social post IDs and the action to approve or reject publishing them.
