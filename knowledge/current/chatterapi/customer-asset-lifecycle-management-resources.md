# Customer Asset Lifecycle Management Resources

Generate, cancel, and change lifecycle-managed assets for sales and account reps to view in Lightning Experience.

Available resources are:

| Resource | Description |
| --- | --- |
| /asset/management/assets/actions/generate | Generate an asset. |
| /asset/management/assets/assetId/actions/cancel | Cancel an asset. |
| /asset/management/assets/assetId/actions/change | Change an asset. |
