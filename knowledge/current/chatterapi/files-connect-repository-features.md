# Files Connect Repository Features

Information about the features of a Files Connect repository.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| canBrowse | Boolean | Specifies whether the repository’s folder hierarchy can be browsed (true) or not (false). | Small, 32.0 | 32.0 |
| canSearch | Boolean | Specifies whether the repository can be searched (true) or not (false). | Small, 32.0 | 32.0 |
