# Personalization Recommender Base Predicate Input

The base representation for a recommender rule predicate.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=chatterapi)

#### Note

This is a base representation and is extended by these predicate input types.

-   [Personalization Recommender Boolean Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_boolean_predicate_input.htm "Represents a predicate based on a boolean value.")
-   [Personalization Recommender Calculated Insight Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_calculated_insight_predicate_input.htm "Represents a predicate based on a calculated insight.")
-   [Personalization Recommender Date Only Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_date_only_predicate_input.htm "Represents a predicate based on a date.")
-   [Personalization Recommender Number Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_number_predicate_input.htm "Represents a predicate based on a number.")
-   [Personalization Recommender Text Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_text_predicate_input.htm "Represents a predicate based on a text string.")
-   [Personalization Recommender Field Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_field_predicate_input.htm "Represents a predicate where the RHS is another field.")
-   [Personalization Recommender Related Field Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_related_field_predicate_input.htm "Represents a predicate where the RHS is a related field.")
-   [Personalization Recommender Segment Membership Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_segment_membership_predicate_input.htm "Represents a predicate based on segment membership.")
-   [Personalization Recommender Relative To Now Predicate Input](atlas.en-us.chatterapi.meta/chatterapi/connect_requests_personalization_recommender_relative_to_now_predicate_input.htm "Represents a predicate based on a date relative to the current time.")

.

| Name | Type | Description | Required or Optional | Available Version |
| --- | --- | --- | --- | --- |
| type | Personalization Recommender Predicate Type Enum | Identifies the RHS predicate type used in this condition. | Required | 64.0 |
