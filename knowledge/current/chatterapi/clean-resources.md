# Clean Resources

Get information about active data integration rules, including data services, update metrics, and the statuses of rules for a record.

Available resources are:

| Resource | Description |
| --- | --- |
| /clean/data-services | Get the data services associated with an organization’s active data integration rules. |
| /clean/data-services/dataServiceId/metrics | Get metrics for the data services associated with an organization’s active data integration rules. |
| /clean/recordId/rules/statuses | Get the statuses of active data integration rules for a record. |
