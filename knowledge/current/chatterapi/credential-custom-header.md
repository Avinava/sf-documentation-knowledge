# Credential Custom Header

Credential custom header.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| headerName | String | Header name. | Small, 57.0 | 57.0 |
| headerValue | String | Header value that can contain formulas. | Small, 57.0 | 57.0 |
| id | String | ID of the customer header parameter. | Small, 58.0 | 58.0 |
| sequenceNumber | Integer | Sequence number of the header. The sequence number determines the order of the header. | Small, 57.0 | 57.0 |
