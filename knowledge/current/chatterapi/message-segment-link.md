# Message Segment: Link

Represents an inline link to a URL.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| text | String | Plain text of the URL. | Small, 29.0 | 23.0 |
| type | String | Value is Link. | Small, 29.0 | 23.0 |
| url | String | URL value of the link. | Small, 29.0 | 23.0 |
