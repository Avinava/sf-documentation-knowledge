# Interactions Capability

If a feed element has this capability, it has information about user interactions.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| count | Long | The number of individual views, likes, and comments on a feed post.NoteThis count appears in the UI under the feed post as the number of views, for example, “5 views.” | Small, 37.0 | 37.0 |
