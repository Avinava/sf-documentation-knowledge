# Priming Records Stats

Statistics for records primed for briefcases.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| ruleCountTotal | Integer | Total number of rules across all briefcases. | Small, 54.0 | 54.0 |
| recordCountTotal | Integer | Maximum number of records that can be returned across all briefcases. | Small, 54.0 | 54.0 |
| ruleCountServed | Integer | Total number of rules processed in the current relay (including all relay requests made so far). | Small, 54.0 | 54.0 |
| recordCountServed | Integer | Maximum number of records that can be returned by processed rules. | Small, 54.0 | 54.0 |
