# Conversation Summary

Conversation summary.

| Property | Type | Description | Filter Group and Version | Available Version |

| latestMessage | Message | The most recent message in this conversation | Small, 29.0 | 23.0 |
| members | User Summary[] | Collection of users participating in the conversation | Small, 29.0 | 23.0 |
