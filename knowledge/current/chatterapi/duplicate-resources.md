# Duplicate Resources

Get a directory of duplicate resources, job definitions, and jobs. Update a job status, run a job, and delete job results.

Available resources:

| Resource | Description |
| --- | --- |
| /dedupe | Get a directory of the duplicate resources. |
| /dedupe/job-definitions | Get duplicate job definitions. |
| /dedupe/job-definitions/jobDefinitionId | Get a duplicate job definition. |
| /dedupe/jobs | Get duplicate jobs or run a job. |
| /dedupe/jobs/jobId | Get or update the status of a duplicate job. |
| /dedupe/jobs/jobId/results | Delete duplicate job results. |
