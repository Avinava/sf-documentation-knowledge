# Personalization Recommenders Resources

Create, manage, and retrieve personalization recommenders.

Available resources are:

| Resource | Description |
| --- | --- |
| /personalization/personalization-recommenders | Create a personalization recommender. (POST) |
| /personalization/personalization-recommenders | Get a list of personalization recommenders. (GET) |
| /personalization/personalization-recommenders/{idOrName} | Get information about a specific personalization recommender. |
| /personalization/personalization-recommenders/{idOrName} | Update a specific personalization recommender. |
| /personalization/personalization-recommenders/{idOrName} | Delete a specific personalization recommender. |
| /personalization/personalization-recommenders/{idOrName}/jobs | Get jobs associated with a specific personalization recommender. |
