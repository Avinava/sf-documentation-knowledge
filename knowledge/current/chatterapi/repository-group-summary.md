# Repository Group Summary

Detailed description of a group entry.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| domain | String | Domain of the group. | Medium, 35.0 | 35.0 |
| email | String | Email of the group. | Small, 35.0 | 35.0 |
| groupType | String | Type of group. Values are:Everybody—Group is public to everybody.EverybodyInDomain—Group is public to everybody in the same domain.Unknown—Group type is unknown. | Small, 35.0 | 35.0 |
