# Content Taxonomy Resources

Create, get, update, delete, and search content taxonomy terms.

Available resources are:

| Resource | Description |
| --- | --- |
| /connect/content-taxonomy/terms | Get terms for a content taxonomy. |
| /connect/content-taxonomy/terms/search | Search content taxonomy terms. |
| /connect/content-taxonomy/taxonomyId/terms | Create a term within a content taxonomy. |
| /connect/content-taxonomy/taxonomyId/terms/termId | Get, update, or delete a term within a content taxonomy. |
