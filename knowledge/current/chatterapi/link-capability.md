# Link Capability

If a feed element has this capability, it has a link.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| url | String | Link URL. The URL can be to an external site. | Small, 32.0 | 32.0 |
| urlName | String | Description of the link. | Small, 32.0 | 32.0 |
