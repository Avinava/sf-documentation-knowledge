# Prompt Templates Resources

Get prompt template data and generate LLM responses using prompt templates.

Available resources are:

| Resource | Description |
| --- | --- |
| /einstein/prompt-templates/ | Get a list of prompt templates based on the specified filters. |
| /einstein/prompt-templates/promptTemplateDevName/generations | Generate a response based on the specified prompt template and input parameters. |

-   **[Einstein Prompt Template Generations](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_prompt_template.htm)**
    Generates a response based on the specified prompt template and input parameters.
-   **[Get Prompt Templates](atlas.en-us.chatterapi.meta/chatterapi/connect_resources_get_prompt_templates.htm)**
    Get a list of prompt templates based on specified filters.
