# Bookmarks Capability

If a feed element has this capability, the context user can bookmark it.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| isBookmarked​ByCurrentUser | Boolean | Indicates whether the feed element has been bookmarked by the context user (true) or not (false). | Small, 32.0 | 32.0 |
