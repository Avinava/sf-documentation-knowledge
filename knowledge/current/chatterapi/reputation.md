# Reputation

Reputation for a user.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| reputationLevel | Reputation Level | User’s reputation level. | Small, 32.0 | 32.0 |
| reputationPoints | Double | User's reputation points, which can be earned by performing different activities. | Small, 32.0 | 32.0 |
| url | String | Connect REST API URL to the reputation. | Small, 32.0 | 32.0 |
