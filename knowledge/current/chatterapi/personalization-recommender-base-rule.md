# Personalization Recommender Base Rule

The base representation for a recommender rule.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=chatterapi)

#### Note

This is a base representation and is extended by these rule types. The type property determines which rule type is represented.

-   [Personalization Recommender Abstract Field Rule](atlas.en-us.chatterapi.meta/chatterapi/connect_responses_personalization_recommender_abstract_field_rule.htm "Represents an abstract rule based on a field.")
-   [Personalization Recommender Calculated Insight Rule](atlas.en-us.chatterapi.meta/chatterapi/connect_responses_personalization_recommender_calculated_insight_rule.htm "Represents a rule based on a calculated insight.")
-   [Personalization Recommender Related Field Rule](atlas.en-us.chatterapi.meta/chatterapi/connect_responses_personalization_recommender_related_field_rule.htm "Represents a rule based on a related field.")
-   [Personalization Recommender Field Rule](atlas.en-us.chatterapi.meta/chatterapi/connect_responses_personalization_recommender_field_rule.htm "Represents a rule based on a field.")

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| type | Personalization Recommender Rule Type Enum | The type of the rule. |  | 64.0 |
