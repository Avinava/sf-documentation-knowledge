# Repository Folder Item

Description of a folder item.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| file | Repository File Summary | If the folder item is a file, the file summary. If the folder item is a folder, null. | Small, 32.0 | 32.0 |
| folder | Repository Folder Summary | If the folder item is a folder, the folder summary. If the folder item is a file, null. | Small, 32.0 | 32.0 |
| type | String | The type of folder item: folder or file. | Small, 32.0 | 32.0 |
