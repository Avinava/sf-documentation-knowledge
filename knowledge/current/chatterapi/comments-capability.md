# Comments Capability

If a feed element or comment has this capability, the context user can add a comment.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| page | Comment Page | The comments information for this feed element or comment.Threaded comments are supported in version 44.0 and later. | Small, 32.0 | 32.0 |
