# Messaging Channel

Messaging channel.

![Important](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note_important.png&folder=chatterapi)

#### Important

Where possible, we changed noninclusive terms to align with our company value of Equality. We maintained certain terms to avoid any effect on customer implementations.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| developerName | String | Unique developer name of the channel. | Small, 44.0 | 44.0 |
| id | String | ID of the Messaging channel, if it is added to the org. | Small, 44.0 | 44.0 |
| isActive | Boolean | Indicates whether the channel is currently active (true) or not (false). | Small, 44.0 | 44.0 |
| masterLabel | String | User friendly name of the channel. | Small, 44.0 | 44.0 |
| messageType | String | Type of channel. Values are:Facebook—A Facebook message.Line—A LINE message.Text—A text message. | Small, 44.0 | 44.0 |
| messagingPlatformKey | String | ID of the Messaging channel on the external service (such as Facebook), if applicable. | Small, 44.0 | 44.0 |
