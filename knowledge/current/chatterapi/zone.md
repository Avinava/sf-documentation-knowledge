# Zone

A Chatter Answers zone.

| Property | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| description | String | The description of the zone. | Big, 29.0 | 29.0 |
| id | String | The zone ID. | Small, 29.0 | 29.0 |
| isActive | Boolean | Indicates that the zone is active. | Small, 29.0 | 29.0 |
| isChatterAnswers | Boolean | Indicates that the zone is available for Chatter Answers. | Small, 29.0 | 29.0 |
| name | String | Name of the zone. | Small, 29.0 | 29.0 |
| url | String | Connect REST API resource URL to the zone. | Small, 30.0 | 30.0 |
| visibility | String | Zone visibility type. One of these values:Community—Available in an Experience Cloud site.Internal—Available internally only.Portal—Available in a portal. | Small, 29.0 | 29.0 |
| visibilityId | String | If the zone is available in a site, this property contains the ID of the site. If the zone is available to all sites, this property contains the value All. | Small, 29.0 | 29.0 |
