# getPackageName()

> For a domain that includes the package name, such as a Lightning
      Component domain or Visualforce page domain, returns the package name. For a domain that
      doesn’t contain a package name, this method returns null.

### getPackageName()

For a domain that includes the package name, such as a Lightning Component domain or Visualforce page domain, returns the package name. For a domain that doesn’t contain a package name, this method returns null.
