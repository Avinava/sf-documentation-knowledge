# localesIdKey

> Stores the default value for the user's locale, defined using two-letter International
    Organization for Standardization (ISO) codes. For example, en_US indicates English (United States). The default value is the language for the
    registration handler execution user. The execution user is specified in the Run As field in the
    authentication provider definition.

### localesIdKey

Stores the default value for the user's locale, defined using two-letter International Organization for Standardization (ISO) codes. For example, en\_US indicates English (United States). The default value is the language for the registration handler execution user. The execution user is specified in the Run As field in the authentication provider definition.
