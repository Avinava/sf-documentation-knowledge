# ConnectApi.InteractionsCapability

If a feed element has this capability, it has information about user interactions.

Subclass of [ConnectApi.FeedElementCapability](atlas.en-us.apexref.meta/apexref/apex_connectapi_output_feed_element_capability.htm "A feed element capability, which defines the characteristics of a feed element.").

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| count | Long | The number of individual views, likes, and comments on a feed post.NoteThis count appears in the UI under the feed post as the number of views, for example, “5 views.” | 37.0 |
