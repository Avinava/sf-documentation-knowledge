# ConnectApi.ProductAttributeValueMetadataRepresentation

# ConnectApi.ProductAttributeValueMetadataRepresentation

Metadata for a product attribute value.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| apiName | String | API Name of the product attribute picklist value. For example, blue__c. | 63.0 |
| colorHexCode | String | Hex code value of a color attribute. For example, #0000FF. | 63.0 |
| label | String | Label of the picklist value for a custom field product attribute. For example, Red. | 63.0 |
| variantAvailable | Boolean | Specifies whether a variation product exists (true) or not (false). | 63.0 |
