# SkillRequirement Class

# SkillRequirement Class

Contains information about the set of skills that are required to complete a particular task for a work type.

## Namespace

[LxScheduler](atlas.en-us.apexref.meta/apexref/apex_namespace_lxscheduler.htm "The LxScheduler namespace provides an interface and classes for integrating Salesforce Scheduler with external calendars.")

## Usage

The constructor for this class can’t be called directly. Create an instance of this class using the [SkillRequirementBuilder.build()](atlas.en-us.apexref.meta/apexref/apex_class_lxscheduler_SkillRequirementBuilder.htm#apex_lxscheduler_SkillRequirementBuilder_build "Returns an instance of the lxscheduler.SkillRequirement object.") method.
