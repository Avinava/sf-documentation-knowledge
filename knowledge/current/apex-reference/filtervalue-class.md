# FilterValue Class

> Returns the unique API name of the filter value. Possible
values for this name are restricted based on the data type of the
column being filtered.

### getName()

Returns the unique API name of the filter value. Possible values for this name are restricted based on the data type of the column being filtered. 

#### Syntax

public String getName()

#### Return Value

Type: [String](atlas.en-us.apexref.meta/apexref/apex_methods_system_string.htm#apex_methods_system_string "Contains methods for the String primitive data type.")
