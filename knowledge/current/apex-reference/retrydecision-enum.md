# RetryDecision Enum

Specifies the retry decision.

## Enum Values

The following are the values of the commercepayments.RetryDecision enum.

| Value | Description |
| --- | --- |
| NonRetriable | The payment operation cannot be retried. |
| Retriable | The payment operation can be retried. |
