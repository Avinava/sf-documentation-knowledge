# SortOrder Enum

Describes the sort order of a related list.

## Enum Values

The following are the values of the Metadata.SortOrder enum.

| Value | Description |
| --- | --- |
| Asc_x | Sort in ascending order. |
| Desc_x | Sort in descending order. |
