# Compression Namespace

> The Compression namespace provides classes and methods to create and extract zip
    files.

**Namespace:** `Compression`

# Compression Namespace

The Compression namespace provides classes and methods to create and extract zip files.

The following are the classes and enums in the Compression namespace.

-   **[Level Enum](atlas.en-us.apexref.meta/apexref/apex_enum_compression_Level.htm)**
    Specifies the compression level for creating a zip file.
-   **[Method Enum](atlas.en-us.apexref.meta/apexref/apex_enum_compression_Method.htm)**
    Specifies the compression method for the zip entries.
-   **[ZipEntry Class](atlas.en-us.apexref.meta/apexref/apex_class_compression_zipentry.htm#apex_class_compression_zipentry)**
    Contains methods to get and set information about a zip file entry.
-   **[ZipReader Class](atlas.en-us.apexref.meta/apexref/apex_class_compression_zipreader.htm#apex_class_compression_zipreader)**
    Contains methods to get information about zip entries and to extract content for specified zip entries from the zip file.
-   **[ZipWriter Class](atlas.en-us.apexref.meta/apexref/apex_class_compression_zipwriter.htm#apex_class_compression_zipwriter)**
    Contains methods to add zip entries, generate a zipped archive, and return the result as an Apex blob.
-   **[Compression Exceptions](atlas.en-us.apexref.meta/apexref/apex_exceptions_compression.htm)**
    The Compression namespace contains exception classes.
