# ConnectApi.ManagedContentChannelDomainRepresentation

# ConnectApi.ManagedContentChannelDomainRepresentation

Domain associated with a managed content channel.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| isLocked | Boolean | Specifies whether the domain is locked and can’t be changed (true) or not (false). | 62.0 |
| name | String | Name of the domain assigned to the channel. | 62.0 |
| value | String | Value of the domain (name or ID) associated with the channel. | 62.0 |
