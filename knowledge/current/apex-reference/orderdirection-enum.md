# OrderDirection Enum

Specifies the direction for sorting rows based on column values.

## Usage

Used by the [direction](atlas.en-us.apexref.meta/apexref/apex_class_DataSource_Order.htm#apex_DataSource_Order_direction "Direction for sorting rows based on column values.") property on the [DataSource.Order](atlas.en-us.apexref.meta/apexref/apex_class_DataSource_Order.htm#apex_class_DataSource_Order "Contains details about how to sort the rows in the result set. Equivalent to an ORDER BY statement in a SOQL query.") class.

## Enum Values

The following are the values of the DataSource.OrderDirection enum.

| Value | Description |
| --- | --- |
| ASCENDING | Sort rows in ascending order (A–Z). |
| DESCENDING | Sort rows in descending order (Z–A). |
