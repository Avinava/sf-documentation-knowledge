# ConnectApi.ExternalAuthIdentityProviderCredential

External auth identity provider credential.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| credentialName | String | Name of the external auth identity provider credential. | 62.0 |
| credentialValue | String | Value of the external auth identity provider credential. | 62.0 |
| encrypted | Boolean | Indicates whether the external auth identity provider credential is encrypted (true) or not (false). | 62.0 |
