# getSupportedEmojis()

> Get supported emojis for the org.

### getSupportedEmojis()

Get supported emojis for the org.

#### API Version

39.0

#### Requires Chatter

Yes

#### Signature

public static ConnectApi.SupportedEmojis getSupportedEmojis()

#### Return Value

Type: [ConnectApi.SupportedEmojis](atlas.en-us.apexref.meta/apexref/apex_connectapi_output_supported_emojis_output.htm "A collection of supported emoji.")

#### Usage

To get the list, emojis must be enabled in your org.
