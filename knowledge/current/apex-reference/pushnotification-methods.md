# PushNotification Methods

The following are the methods for PushNotification. All are global methods.

-   **[send(application, users)](atlas.en-us.apexref.meta/apexref/apex_Messaging_PushNotification_send.htm)**  
    Sends a push notification message to the specified users.
-   **[setPayload(payload)](atlas.en-us.apexref.meta/apexref/apex_Messaging_PushNotification_setPayload.htm)**  
    Sets the payload of the push notification message.
-   **[setTtl(ttl)](atlas.en-us.apexref.meta/apexref/apex_Messaging_PushNotification_setTtl.htm)**  
    Reserved for future use.
