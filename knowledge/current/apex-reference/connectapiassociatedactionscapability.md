# ConnectApi.AssociatedActionsCapability

# ConnectApi.AssociatedActionsCapability

If a feed element has this capability, it has platform actions associated with it.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| platformAction​Groups | List<ConnectApi.​PlatformActionGroup> | The platform action groups associated with a feed element. Platform action groups are returned in the order specified in the ConnectApi.AssociatedActions​CapabilityInput class. | 33.0 |
