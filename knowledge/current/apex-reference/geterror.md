# getError()

If an error occurred, returns a ConnectApi.ConnectApiException object providing the error code and description.

## Signature

public ConnectApi.ConnectApiException getError()

## Return Value

Type: [ConnectApi.ConnectApiException](atlas.en-us.apexref.meta/apexref/connectAPI_exceptions.htm#connectAPI_exceptions "The ConnectApi namespace contains exception classes.")
