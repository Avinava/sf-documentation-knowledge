# ConnectApi.FeedItemAttachmentInput

# ConnectApi.FeedItemAttachmentInput

Used to attach a file to a feed item.

![Important](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note_important.png&folder=apexref)

#### Important

This class isn’t available in version 32.0 and later. In version 32.0 and later, use [ConnectApi.FeedElementCapabilityInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_feed_element_capability.htm "A feed element capability.").

This class is abstract and has no public constructor. You can make an instance only of a subclass.

Superclass for:

-   [ConnectApi.CanvasAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_canvasAtt.htm "Used to attach a canvas app to a feed item.")
-   [ConnectApi.ContentAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_contentAtt.htm "Used to attach existing content to a comment or feed item.")
-   [ConnectApi.LinkAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_linkAttach.htm "Add links to a feed item.")
-   [ConnectApi.NewFileAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_NewFileAtt.htm "Attach a new file to a feed item.")
-   [ConnectApi.PollAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_PollAttachInput.htm "Attach a poll to a feed item.")
