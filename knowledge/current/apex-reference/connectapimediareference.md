# ConnectApi.MediaReference

A media reference.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| mediaUrl | String | URL to stream or download the media. | 41.0 |
| thumbnailUrl | String | If one exists, URL of the media’s thumbnail. | 41.0 |
