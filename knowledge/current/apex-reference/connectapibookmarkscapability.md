# ConnectApi.BookmarksCapability

If a feed element has this capability, the context user can bookmark it.

Subclass of [ConnectApi.FeedElementCapability](atlas.en-us.apexref.meta/apexref/apex_connectapi_output_feed_element_capability.htm "A feed element capability, which defines the characteristics of a feed element.").

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| isBookmarked​ByCurrentUser | Boolean | Indicates whether the feed element has been bookmarked by the context user (true) or not (false). | 32.0 |
