# Context Namespace

# Context Namespace

The Context namespace provides classes and methods to manage the sharing and consumption of business application data by using Context Service.

The Context namespace includes the [IndustriesContext](https://developer.salesforce.com/docs/atlas.en-us.260.0.industries_reference.meta/industries_reference/apex_class_Context_IndustriesContext.htm "HTML (New Window)") class.
