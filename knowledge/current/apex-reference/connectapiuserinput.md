# ConnectApi.UserInput

Update a user’s About Me information.

| Property | Type | Description | Available Version |
| --- | --- | --- | --- |
| aboutMe | String | The aboutMe property of a ConnectApi.UserDetail output object. This property populates the About Me section of the user profile, which is visible to all members of an Experience Cloud site or org. | 29.0 |
