# ConnectApi.DataSpaceInfoRepresentation

| Property Name | Type | Description | Available Version |

| description | String | Description of the data space. | 62.0 |
| prefix | String | Prefix of the data space. | 62.0 |
| status | DataSpaceStatusEnum | Status of the data space.Active—The data space is active.Error—The data space has an error.Processing—The data space is being processed. | 62.0 |
