# getButtonClicked()

> Returns the field ID of the button that was clicked in the preference
      form. For example, use this method to determine if the clicked button was
        Submit or Cancel.

### getButtonClicked()

Returns the field ID of the button that was clicked in the preference form. For example, use this method to determine if the clicked button was **Submit** or **Cancel**.

#### Signature

public String getButtonClicked()
