# ConnectApi.TopicImages

Images associated with a topic.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| coverImageUrl | String | URL to a topic’s cover image, which appears on the topic page. Both topics and managed topics can have cover images. | 32.0 |
| featuredImageUrl | String | URL to a managed topic’s featured image, which appears wherever you feature it, for example, on your Experience Cloud site home page. | 32.0 |
