# ConnectApi.ContentHubRepositoryFeatures

The features of a repository.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| canBrowse | Boolean | Specifies whether the repository’s folder hierarchy can be browsed (true) or not (false). | 39.0 |
| canSearch | Boolean | Specifies whether the repository can be searched (true) or not (false). | 39.0 |
