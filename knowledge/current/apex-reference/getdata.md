# getData()

> Retrieves data from the identity provider token. This data can include custom data that
    isn’t stored in the userData property.

### getData()

Retrieves data from the identity provider token. This data can include custom data that isn’t stored in the userData property.

#### Signature

public Object getData()

#### Return Value

Type: Object
