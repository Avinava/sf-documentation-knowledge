# ConnectApi.CallCollaborationCapability

# ConnectApi.CallCollaborationCapability

If a feed element has this capability, it has a recording comment.

Subclass of [ConnectApi.FeedElementCapability](atlas.en-us.apexref.meta/apexref/apex_connectapi_output_feed_element_capability.htm "A feed element capability, which defines the characteristics of a feed element.").

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| commentEndTime | Integer | End time of the comment on the media player, in seconds. | 51.0 |
| commentStartTime | Integer | Start time of the comment on the media player, in seconds. | 51.0 |
