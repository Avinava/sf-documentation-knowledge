# ConnectApi.ManagedContentBodyInput

# ConnectApi.ManagedContentBodyInput

Input representation for the body of a piece of managed content.

| Property | Type | Description | Required or Optional | Available Version |
| --- | --- | --- | --- | --- |
| nodeMap | Object | Body of the managed content version. The format must be Map<String, Object> , where map values are either primitive values like String, Integer, Double, Boolean, or another Map<String, Object>. | Required | 60.0 |
