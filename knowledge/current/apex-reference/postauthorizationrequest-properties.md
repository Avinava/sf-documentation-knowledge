# PostAuthorizationRequest Properties

Lists properties for a postauthorizaiton request.

The following are properties for a PostAuthorizationRequest.

-   **[accountId](atlas.en-us.apexref.meta/apexref/apex_commercepayments_PostAuthorizationRequest_accountId.htm)**
    The customer account that is settled when the postauthorization is performed.
-   **[amount](atlas.en-us.apexref.meta/apexref/apex_commercepayments_PostAuthorizationRequest_amount.htm)**
    The total amount of the postauthorization request.
-   **[comments](atlas.en-us.apexref.meta/apexref/apex_commercepayments_PostAuthorizationRequest_comments.htm)**
    Comments about the postauthorization. Users can enter comments to provide additional information.
-   **[currencyIsoCode](atlas.en-us.apexref.meta/apexref/apex_commercepayments_PostAuthorizationRequest_currencyIsoCode.htm)**
    The ISO currency code for the postauthorization request.
-   **[paymentMethod](atlas.en-us.apexref.meta/apexref/apex_commercepayments_PostAuthorizationRequest_paymentMethod.htm)**
    The payment method used to process the postauthorization request.
