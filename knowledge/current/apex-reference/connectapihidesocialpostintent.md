# ConnectApi.HideSocialPostIntent

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |

| managedSocialAccount | ConnectApi.​ManagedSocialAccount | Managed social account that hides the social post. | 45.0 |
