# GroupingInfo Class

> Returns the summary field that is used to sort data within
a grouping in a summary report. The value is null when data within
a grouping is not sorted by a summary field.

### getSortAggregate()

Returns the summary field that is used to sort data within a grouping in a summary report. The value is null when data within a grouping is not sorted by a summary field.

#### Syntax

public String getSortAggregate()
