# SmartDataDiscovery Class

Get predictions on Salesforce objects.

Use the ConnectApi.SmartDataDiscovery.predict method to get predictions on Salesforce objects. For more information, see [Get Predictions in Apex](https://help.salesforce.com/articleView?id=bi_edd_prediction_apex.htm&language=en_US).
