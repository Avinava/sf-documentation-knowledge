# Retired ConnectApi Input Classes

# Retired ConnectApi Input Classes

These ConnectApi input classes are retired.

-   **[ConnectApi.CanvasAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_canvasAtt.htm)**  
    Used to attach a canvas app to a feed item.
-   **[ConnectApi.ContentAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_contentAtt.htm)**  
    Used to attach existing content to a comment or feed item.
-   **[ConnectApi.DatacloudOrderInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_datacloud_order.htm)**  
    Input representation for a Datacloud order to purchase contacts or companies and retrieve purchase information.
-   **[ConnectApi.FeedItemAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_FeedItemAtt.htm)**  
    Used to attach a file to a feed item.
-   **[ConnectApi.LinkAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_linkAttach.htm)**  
    Add links to a feed item.
-   **[ConnectApi.NewFileAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_NewFileAtt.htm)**  
    Attach a new file to a feed item.
-   **[ConnectApi.PollAttachmentInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_PollAttachInput.htm)**  
    Attach a poll to a feed item.
-   **[ConnectApi.SocialPostMassApprovalInput](atlas.en-us.apexref.meta/apexref/apex_connectapi_input_social_post_mass_approval.htm)**  
    List of social post ids and the action to approve or reject publishing them.
