# Flow Namespace

The Flow namespace provides a class for advanced access to flows from Apex such as from Visualforce controllers and asynchronous Apex.

The following is the class in the Flow namespace.

-   **[Interview Class](atlas.en-us.apexref.meta/apexref/flow_interview_class.htm#flow_interview_class)**
    The Flow.Interview class provides advanced controller access to flows and the ability to start a flow.
