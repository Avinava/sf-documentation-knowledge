# ConnectApi.DigestJob

# ConnectApi.DigestJob

Represents a successfully enqueued API digest job request.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| period | ConnectApi.​DigestPeriod | Time period that’s included in a Chatter email digest. Values are:DailyDigest—The email includes up to the 50 latest posts from the previous day.WeeklyDigest—The email includes up to the 50 latest posts from the previous week. | 37.0 |
