# ConnectApi.WishlistSummary

Summary of a wishlist.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| createdDate | Datetime | Created date for the wishlist in ISO 8601 format, for example, 2011-02-25T18:24:31.000Z. | 49.0 |
| id | String | ID of the wishlist. | 49.0 |
| modifiedDate | Datetime | Last modified date of the wishlist in ISO 8601 format, for example, 2011-02-25T18:24:31.000Z. | 49.0 |
| name | String | Name of the wishlist. | 49.0 |
| wishlistProductCount | Integer | Unique product count in the wishlist. | 49.0 |
