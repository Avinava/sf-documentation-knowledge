# ConnectApi.MatchInfo

Search information related to the search result.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| isPromoted | Boolean | Specifies whether search promotion affected the result (true) or not (false). | 63.0 |
| isSpellCorrected | Boolean | Specifies whether spell correction affected the result (true) or not (false). | 63.0 |
