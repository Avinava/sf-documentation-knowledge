# ApplicationContext Methods

The following are methods for ApplicationContext.

-   **[getCanvasUrl()](atlas.en-us.apexref.meta/apexref/apex_canvas_ApplicationContext_getCanvasUrl.htm)**
    Retrieves the fully qualified URL of the canvas app.
-   **[getDeveloperName()](atlas.en-us.apexref.meta/apexref/apex_canvas_ApplicationContext_getDeveloperName.htm)**
    Retrieves the internal API name of the canvas app.
-   **[getName()](atlas.en-us.apexref.meta/apexref/apex_canvas_ApplicationContext_getName.htm)**
    Retrieves the name of the canvas app.
-   **[getNamespace()](atlas.en-us.apexref.meta/apexref/apex_canvas_ApplicationContext_getNamespace.htm)**
    Retrieves the namespace prefix of the canvas app.
-   **[getVersion()](atlas.en-us.apexref.meta/apexref/apex_canvas_ApplicationContext_getVersion.htm)**
    Retrieves the current version of the canvas app.
-   **[setCanvasUrlPath(newPath)](atlas.en-us.apexref.meta/apexref/apex_canvas_ApplicationContext_setCanvasUrl.htm)**
    Overrides the URL of the canvas app for the current request.
