# Download Developer WSDL Files (WSDL-Based APIs)

Salesforce Web Services Definition Language (WSDL) files provide API details that you use in your developer environment to make API calls.

To download WSDL files directly from your Salesforce organization:

1.  Log in to your Salesforce developer organization in your browser.
2.  From Setup, enter API in the Quick Find box, then select **API**.
3.  Download the appropriate WSDL files for the API you want to use.
    1.  If you want to use SOAP API you’ll need either the Enterprise or Partner WSDL. See [Choosing a WSDL](https://developer.salesforce.com/docs/atlas.en-us.260.0.api.meta/api/sforce_api_quickstart_intro.htm#choose_wsdl "HTML (New Window)") in the SOAP API Developer Guide to determine which WSDL to download.
    2.  If you want to use Metadata API you’ll need the Metadata WSDL. To login and authenticate with Salesforce you’ll also need either the Enterprise or Partner WSDL.
    3.  If you want to use Tooling API you’ll need the Tooling WSDL. To login and authenticate with Salesforce you’ll also need either the Enterprise or Partner WSDL.
