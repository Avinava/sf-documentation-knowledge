# AdditionalNumber

Represents an optional additional number for a call center. This additional number is visible in the call center's phone directory.

This guide only lists certain information for each object. For more information, including descriptions of the fields, see [AdditionalNumber](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_additionalnumber.htm "HTML (New Window)") in the Object Reference for the Salesforce Platform.
