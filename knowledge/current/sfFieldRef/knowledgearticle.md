# KnowledgeArticle

Provides read-only access to an article and the ability to delete the primary article. This object is available in API version 19.0 and later.

This guide only lists certain information for each object. For more information, including descriptions of the fields, see [KnowledgeArticle](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_knowledgearticle.htm "HTML (New Window)") in the Object Reference for the Salesforce Platform and Salesforce Knowledge Developer Guide.
