# SiteDetail

Represents the details of a Salesforce site or Experience Cloud site. Available in API Version 38.0 and later.

This guide only lists certain information for each object. For more information, including descriptions of the fields, see [SiteDetail](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_sitedetail.htm "HTML (New Window)") in the Object Reference for the Salesforce Platform.
