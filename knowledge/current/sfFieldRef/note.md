# Note

Represents a note, which is text associated with a custom object or a standard object, such as a Contact, Contract, or Opportunity.

This guide only lists certain information for each object. For more information, including descriptions of the fields, see [Note](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_note.htm "HTML (New Window)") in the Object Reference for the Salesforce Platform.
