# CustomBrandAsset

Represents a branding element in a custom branding scheme. For example, a color, logo image, header image, or footer text. A CustomBrandAsset can apply to an Experience Cloud site or to an org using the Salesforce mobile app. This object is available in API version 28.0 and later.

This guide only lists certain information for each object. For more information, including descriptions of the fields, see [CustomBrandAsset](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_custombrandasset.htm "HTML (New Window)") in the Object Reference for the Salesforce Platform.
