# apex:outputLabel

A label for an input or output field. Use this component to provide a label for a controller method that does not correspond to a field on a Salesforce object.

This component supports [HTML pass-through attributes](https://developer.salesforce.com/docs/atlas.en-us.260.0.pages.meta/pages/pages_html_features_pass_through_attributes.htm "html (New Window)") using the "html-" prefix. Pass-through attributes are attached to the generated <label\> tag.
