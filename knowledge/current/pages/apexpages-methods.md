# ApexPages Methods

The following are methods for ApexPages. All are instance methods.

-   **[addMessage(message)](atlas.en-us.pages.meta/pages/apex_System_ApexPages_addMessage.htm)**
    Add a message to the current page context.
-   **[addMessages(exceptionThrown)](atlas.en-us.pages.meta/pages/apex_System_ApexPages_addMessages.htm)**
    Adds a list of messages to the current page context based on a thrown exception.
-   **[currentPage()](atlas.en-us.pages.meta/pages/apex_System_ApexPages_currentPage.htm)**
    Returns the current page's PageReference.
-   **[getMessages()](atlas.en-us.pages.meta/pages/apex_System_ApexPages_getMessages.htm)**
    Returns a list of the messages associated with the current context.
-   **[hasMessages()](atlas.en-us.pages.meta/pages/apex_System_ApexPages_hasMessages.htm)**
    Returns true if there are messages associated with the current context, false otherwise.
-   **[hasMessages(severity)](atlas.en-us.pages.meta/pages/apex_System_ApexPages_hasMessages_2.htm)**
    Returns true if messages of the specified severity exist, false otherwise.
