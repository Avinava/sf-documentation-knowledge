# Resources

Here’s a list of Manufacturing Cloud Business API resources.

-   **[Sales Agreement (POST)](atlas.en-us.mfg_api_devguide.meta/mfg_api_devguide/connect_resources_sales_agreement.htm)**
    Create a sales agreement from an external source, including quote, opportunity, or custom object.
-   **[Sample Management (POST)](atlas.en-us.mfg_api_devguide.meta/mfg_api_devguide/connect_resources_sample-management.htm)**
    Creates, updates, or versions Product Requirement Specification records.
-   **[Transformations (POST)](atlas.en-us.mfg_api_devguide.meta/mfg_api_devguide/connect_resources_transformations.htm)**
    Perform the business transformation of program component forecast data (source object) to opportunities (target object). The following are some of the transformation scenarios: MfgProgramCpntFrcstFact to Opportunity, ManufacturingProgram to Opportunity, MfgProgramCpntFrcstFact to OpportunityLineItem, and Period to OpportunityLineItemSchedule.
-   **[Warranty To Supplier Claims (POST)](atlas.en-us.mfg_api_devguide.meta/mfg_api_devguide/connect_resources_warranty_to_supplier_claims.htm)**
    Warranty to Supplier Claims API clones the existing warranty claim and its hierarchy such as claim items, claim coverage, and claim coverage payment details.
