# Manufacturing Cloud Business APIs

Manufacturing Cloud Business APIs are RESTful APIs that are sometimes available as Apex classes and methods.

-   **[REST Reference](atlas.en-us.mfg_api_devguide.meta/mfg_api_devguide/manufacturing_cloud_apis_rest_references.htm)**
    You can access Manufacturing Cloud Business APIs using REST endpoints. These REST APIs follow similar conventions as Connect REST APIs.
