# Salesforce Knowledge REST APIs

You can manage articles and get support information with REST APIs.

REST API uses the same underlying [data model](atlas.en-us.knowledge_dev.meta/knowledge_dev/knowledge_development_object_model.htm "The Knowledge object model is slightly different from other feature areas because it exposes a set of abstract Salesforce objects that aren’t directly used when you create articles. These abstract objects then contain concrete derivations that you do use when creating articles.") and [standard objects](atlas.en-us.knowledge_dev.meta/knowledge_dev/knowledge_development_soap_intro.htm "You can manage articles and get support information with Salesforce Objects using the SOAP API.") as those in SOAP API. For generic REST API information see the [REST API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_rest.meta/api_rest/ "HTML (New Window)").
