# Knowledge SOAP API Calls

SOAP API calls for working with Salesforce Knowledge.

-   **[describeKnowledge()](atlas.en-us.knowledge_dev.meta/knowledge_dev/sforce_api_calls_describeknowledge.htm)**
    Retrieves the Knowledge language settings in the organization.
-   **[describeDataCategoryGroups()](atlas.en-us.knowledge_dev.meta/knowledge_dev/sforce_api_calls_describedatacategorygroups.htm)**
    Retrieves available category groups for objects specified in the request.
-   **[describeDataCategoryGroupStructures()](atlas.en-us.knowledge_dev.meta/knowledge_dev/sforce_api_calls_describedatacategorygroupstructures.htm)**
    Retrieves available category groups along with their data category structure for objects specified in the request.
-   **[search()](atlas.en-us.knowledge_dev.meta/knowledge_dev/sforce_api_calls_search.htm)**
    Executes a text search in your organization's data.
