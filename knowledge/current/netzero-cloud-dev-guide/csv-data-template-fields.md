# CSV Data Template Fields

Output representation for the field details in the CSV data template.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| apiName | String | The API name of an object. | Small, 58.0 | 58.0 |
| fieldName | String | The name of the field that contains the verification data based on the selected field type. | Small, 58.0 | 58.0 |
| fieldType | String | The data type of the field. | Small, 58.0 | 58.0 |
| required | Boolean | Determines whether the field is required or not. | Small, 58.0 | 58.0 |
