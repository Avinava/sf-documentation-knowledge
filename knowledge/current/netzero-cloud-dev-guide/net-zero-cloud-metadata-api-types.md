# Net Zero Cloud Metadata API Types

Metadata API enables you to access some types and feature settings that you can customize in the user interface. For more information about Metadata API and to find a complete reference of existing metadata types, see **[Metadata API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/ "HTML (New Window)")**.

-   **[ClauseCatgConfiguration](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_clausecatgconfiguration.htm)**
    Represents the configuration about the clause category that can be used to categorize your disclosure and compliance reports from standardized disclosure templates in a response document.
-   **[DisclosureDefinition](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_disclosuredefinition.htm)**
    Represents information that defines a disclosure type, such as details of the publisher or vendor who created or implemented the report.
-   **[DisclosureDefinitionVersion](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_disclosuredefinitionversion.htm)**
    Represents the version information about the disclosure definition.
-   **[DisclosureType](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_disclosuretype.htm)**
    Represents the types of disclosures that are done by an individual or an organization and the associated metadata.
-   **[Flow for Net Zero Cloud](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/netzero_cloud_flow_metadata_api.htm)**
    Represents the metadata associated with a flow. With Flow, you can create an application that navigates users through a series of screens to query and update records in the database. You can also execute logic and provide branching capability based on user input to build dynamic applications.
-   **[FuelType](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_fueltype.htm)**
    Represents a custom fuel type in an org.
-   **[FuelTypeSustnUom](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_fueltypesustnuom.htm)**
    Represents a mapping between the custom fuel types and their corresponding unit of measure (UOM) values defined by a customer in an org.
-   **[IndustriesSettings](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/netzero_cloud_meta_industriessettings.htm)**
    Represents settings for Net Zero Cloud.
-   **[SustnUomConversion](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_sustnuomconversion.htm)**
    Represents information about the unit of measure (UOM) conversion for the custom fuel types defined by a customer in an org.
-   **[SustainabilityUom](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/meta_sustainabilityuom.htm)**
    Represents the unit of measure (UOM) values for custom fuel types in an org. Track fuel consumption and emission results with the flexibility to add custom fuel types and UOM values.
