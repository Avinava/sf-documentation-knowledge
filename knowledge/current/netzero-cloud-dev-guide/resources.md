# Resources

Here’s a list of Net Zero Cloud Business API resources.

-   **[Allocate Scorecard Emissions](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_supplier_management_allocate.htm)**
    Allocate final emissions to procurement items. Update the AllocationStatus field on the scorecard on a successful allocation. The API allocates final emissions to procurement items based on the calculated value of the FinalScorecardEmissions field.
-   **[Annual Emission Inventory Recalculation](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_annual_emission_recalculation.htm)**
    Recalculate a record whenever you update data, such as emission factors.
-   **[Associate Scorecard Procurement Items](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_supplier_management_associate.htm)**
    Associate procurement items from supplier management to the active scorecard. Only procurement items with an unpopulated lookup can be associated with the scorecard. The API collects the procurement items that lie in between the StartDate and the EndDate as per the scorecard.
-   **[Carbon Footprint Recalculation](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_footprint_recalculation.htm)**
    Recalculate a record whenever you update data, such as emission factors.
-   **[Carbon Inventory](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_sustainability_carbon_inventory.htm)**
    Capture your organization’s carbon footprint from a source for a certain time period by creating an annual emissions inventory. Using this data as the base, achieve your emissions reduction targets and fulfill your Net Zero commitment.
-   **[Compute Data Gap Fillers](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resource_compute-datagap-fillers.htm)**
    Identify and get energy consumption records associated with the Stationary Asset Carbon Footprint object that are missing for a given period. You can fill any energy use data gaps for a stationary asset categorized as a commercial building.
-   **[Data Gap Percentage Calculation](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/netzero_connect_resource_datagap_percent_calc.htm)**
    Calculate a record whenever you update data, such as emission factors.
-   **[Date Issues](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resource_identify_date_issues.htm)**
    Identify and get energy consumption records with date issues associated with the Stationary Asset Carbon Footprint object.
-   **[Dynamic Data Gap Percentage Calculation](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/netzero_connect_resource_datagap_dynamic_percent_calc.htm)**
    Calculate the list of data gaps for stationary assets for a dynamic date range.
-   **[Object API Name](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_objectapi_name.htm)**
    Retrieve information of an object in JSON format, which can be used further to create a CSV template for uploading data.
-   **[Recalculate Building Energy Intensity](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resource_bei_recalculation.htm)**
    Recalculate the building energy intensity (BEI) based on filters such as fuel type, building type, and geolocation.
-   **[Retrieve the List of Objects](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_objects.htm)**
    Retrieve the list of objects for a specified feature.
-   **[Sync Sustainability Credit](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_sync-sustainability-credit.htm)**
    Synchronize your organization’s carbon footprint sustainability credits from the MarketPlace to NetZero Cloud. The API popluates the SustainabilityCredit, SustainabilityPurchase, and CrbnCreditProject records in Netzero Cloud.
-   **[Lock Record](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_lock_record.htm)**
    Lock a record in Salesforce object to prevent data modification after carbon footprint is calculated.
-   **[Unlock Record](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_resources_unlock_record.htm)**
    Unlock a record in Salesforce object to update data.
