# Request Bodies

Here’s a list of Net Zero Cloud Business API request bodies.

-   **[Billing Address Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_billing_address_input.htm)**
    Input representation of the sustainability purchase billing address.
-   **[Carbon Credit Project Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_crbn_credit_project_input.htm)**
    Input representation of the carbon credit project.
-   **[Data Gap Filler Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_request_sustainability_d_g_f_data_gap_filler_input.htm)**
    Input representation of the request to get missing energy consumption records.
-   **[Data Gap Percentage Calculation Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/netzero_connect_request_datagap_percent_calc.htm)**
    Input representation of Data Gap Percentage Calculation endpoint.
-   **[Date Issue Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_request_sustainability_date_issue_input.htm)**
    Input representation of the request to get energy consumption records having date issues.
-   **[Dynamic Data Gap Percentage Calculation Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/netzero_connect_request_datagap_dynamic_percent_calc.htm)**
    Input representation of Dynamic Data Gap Percentage Calculation endpoint.
-   **[Filter Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_request_filter_input.htm)**
    Input representation of the filter object with key value pair.
-   **[Scope 3 Carbon Footprint Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_scope3_carbon_footprint_input.htm)**
    Input representation of the source to calculate the scope 3 carbon footprint. A scope 3 carbon footprint covers greenhouse gas emissions from sources owned or controlled by other entities in your organization.
-   **[Stationary Asset Carbon Footprint Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_stationary_asset_carbon_footprint_input.htm)**
    Input representation of the source to calculate the carbon footprint from stationary assets, such as a commercial building or a data center.
-   **[Sustainability Carbon Inventory Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_sustainability_carbon_inventory_input.htm)**
    Input representation of the annual emissions inventory creation request.
-   **[Sync Sustainability Carbon Credit Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_sync_sustainability_carbon_credit_input.htm)**
    Input representation of the carbon footprint sustainability credits.
-   **[Sustainability Credit Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_sustainability_credit_input.htm)**
    Input representation of the sustainability credit associated with the carbon credit project.
-   **[Sustainability Purchase Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_sustainability_purchase_input.htm)**
    Input representation of the sustainability carbon credit purchase.
-   **[Vehicle Asset Carbon Footprint Input](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/connect_requests_vehicle_asset_carbon_footprint_input.htm)**
    Input representation of the source to calculate the carbon footprint from vehicle assets, such as fleet vehicles, corporate aircraft, and company shuttles.
