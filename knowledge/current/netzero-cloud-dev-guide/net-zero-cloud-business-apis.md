# Net Zero Cloud Business APIs

Net Zero Cloud Business APIs are RESTful APIs that are sometimes available as Apex classes and methods.

-   **[REST Reference](atlas.en-us.netzero_cloud_dev_guide.meta/netzero_cloud_dev_guide/net_zero_cloud_business_apis_rest_references.htm)**
    You can access Net Zero Cloud Business APIs using REST endpoints. These REST APIs follow similar conventions as Connect REST APIs.
