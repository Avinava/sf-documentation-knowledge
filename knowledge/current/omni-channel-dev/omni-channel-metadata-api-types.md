# Omni-Channel Metadata API Types

The Metadata API lets you access Omni-Channel feature settings and metadata information.

The following types are available with the Metadata API.

-   [Flow](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_visual_workflow.htm)
-   [OmniChannelSettings](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_omnichannelsettings.htm)
-   [OmniSupervisorConfig](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_omnisupervisorconfig.htm)
-   [PresenceDeclineReason](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_presencedeclinereason.htm)
-   [PresenceUserConfig](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_presenceuserconfig.htm)
-   [Queue](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_queue.htm)
-   [QueueRoutingConfig](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_queueroutingconfig.htm)
-   [ServiceChannel](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_servicechannel.htm)
-   [ServicePresenceStatus](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_servicepresencestatus.htm)
-   [WorkSkillRouting](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_workskillrouting.htm)
-   [WorkSkillRoutingAttribute](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_workskillrouting.htm)

If you need more information on the Salesforce Metadata API, see the [Metadata API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/meta_intro.htm).
