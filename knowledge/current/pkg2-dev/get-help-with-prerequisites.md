# Get Help with Prerequisites

If you need help with setting up your solution, you can request a consultation with a Platform Expert.

| Available in: both Salesforce Classic and Lightning Experience |
| --- |
| Available in: Enterprise, Performance, Unlimited, and Developer Editions |


1.  Log in to the [Salesforce Partner Community](https://partners.salesforce.com "HTML (New Window)").
2.  Click the question icon ![AppExchange Support question mark](/docs/resources/img/en-us/260.0?doc_id=appexchange%2Fimages%2Fappex_support_question_mark.png&folder=pkg2_dev) and then click **Log a Case for Help**.
3.  Provide any required details, and then click **Create Case**.
