# Referral Mobile SDK API Reference for iOS

The Referral Mobile iOS SDK interacts with the Salesforce Referral REST APIs to manage referral-related operations, such as enrolling members in referral programs, and creating referral events.

| Available in: all editions that have Referral Marketing enabled. |
| --- |


-   **[Referral Enrollment](atlas.en-us.referral_marketing.meta/referral_marketing/ios_mobile_sdk_api_referralenrollment.htm)**
    Enroll an existing or new customer to a referral promotion.
-   **[Referral event](atlas.en-us.referral_marketing.meta/referral_marketing/ios_mobile_sdk_api_referralevent.htm)**
    Create referral event records when an advocate refers a friend, or when the referred friends sign up or make a purchase.
