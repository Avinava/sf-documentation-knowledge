# Referral Mobile SDK Android Classes

The Android classes organize the Referral Mobile SDK’s functionality and data structures. When developing mobile apps using the SDK, these classes provide components, services, or features with which you can interact.

| Available in: all editions that have Referral Marketing enabled |
| --- |


-   **[ReferralsRepository](atlas.en-us.referral_marketing.meta/referral_marketing/referral_marketing_sdk_android_referralsrepository_class.htm)**
    The ReferralsRepository interface invokes and manages referral-related REST API requests. Use this class to interact with the Salesforce Referral Marketing APIs and manage referral-related operations.
