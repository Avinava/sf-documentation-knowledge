# Referral Mobile Android SDK API Reference

The Referral Mobile Android SDK interacts with the Salesforce Referral APIs to manage referral-related operations, such as enrolling members in referral programs, and creating referral events.

| Available in: all editions that have Referral Marketing enabled |
| --- |


Interact with the Referral APIs, including:

| Resource | Description |
| --- | --- |
| /referral-programs/${referralProgramName}/promotions/${promotionCode}/member-enrollments | Enroll an existing or new customer to a referral promotion. |
| /referral-program/referral-event | Create referral event records when an advocate refers a friend, or when the referred friends sign up, or make a purchase. |
