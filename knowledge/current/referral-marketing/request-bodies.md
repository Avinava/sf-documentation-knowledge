# Request Bodies

Here’s a list of Referral Marketing API request bodies.

-   **[Redeem Voucher Input](atlas.en-us.referral_marketing.meta/referral_marketing/connect_requests_industries_redeem_voucher_input.htm)**
    Input representation of the request to redeem a voucher for an advocate or their referred friend.
-   **[Person Account Input](atlas.en-us.referral_marketing.meta/referral_marketing/connect_requests_person_account_input.htm)**
    Input representation of a advocate’s person account details.
-   **[Referral Event Input](atlas.en-us.referral_marketing.meta/referral_marketing/connect_requests_referral_event_input.htm)**
    Input representation of the request to create a referral event.
-   **[Referral Advocate Enrollment Input](atlas.en-us.referral_marketing.meta/referral_marketing/connect_requests_referral_member_enrollment_input.htm)**
    Input representation of the request to enroll an advocate to a referral promotion.
-   **[Referral Advocate Profile Input](atlas.en-us.referral_marketing.meta/referral_marketing/connect_requests_referral_advocate_profile_input.htm)**
    Input representation for the Referral Advocate Profile API.
-   **[Referral Advocate Contact Input](atlas.en-us.referral_marketing.meta/referral_marketing/connect_requests_referral_advocate_contact_input.htm)**
    Input representation for Referral Advocate contact details.
