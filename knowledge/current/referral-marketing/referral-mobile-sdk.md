# Referral Mobile SDK

Referral programs help increase brand awareness and build a loyal customer base for your brand. Integrate the referral program capability in a mobile application using the Referral Mobile Software Development Kit (SDK) for Android and iOS. Add the SDK as an external dependency in your custom mobile application, or add it to the existing Loyalty Management Mobile SDK, and extend the MyNTORewards sample app to include referral promotions.

| Available in: all editions that have Referral Marketing enabled |
| --- |


-   **[Supported Versions of Tools and Components](atlas.en-us.referral_marketing.meta/referral_marketing/referral_marketing_sdk_supportedtoolsncomp.htm)**
    The Referral Mobile SDK works on iOS 15.0+ and Android 8.0+ versions. Install the tools and components required to set up the Referral Mobile SDK.
-   **[Referral Mobile SDK for iOS](atlas.en-us.referral_marketing.meta/referral_marketing/referral_marketing_sdk_ios_clone_import_parent.htm)**
    Clone and import the Referral Mobile iOS SDK to integrate the referral program feature in a mobile app.
-   **[Referral Mobile SDK for Android](atlas.en-us.referral_marketing.meta/referral_marketing/referral_marketing_sdk_android_clone_import_parent.htm)**
    Clone and import the Referral Mobile Android SDK to integrate the referral program feature in a mobile app.
