# Resources

Here’s a list of Referral Marketing API resources.

-   **[Referral Advocate Enrollments (POST)](atlas.en-us.referral_marketing.meta/referral_marketing/connect_resources_enroll_referral_member.htm)**
    Enroll an existing or new customer as an advocate of a referral promotion.
-   **[Redeem Voucher (POST)](atlas.en-us.referral_marketing.meta/referral_marketing/connect_resources_redeem_voucher_referral_member.htm)**
    Redeem a voucher for an advocate or their referred friend.
-   **[Referral Event (POST)](atlas.en-us.referral_marketing.meta/referral_marketing/connect_resources_referral_event.htm)**
    Create referral event records when an advocate refers a friend, or when referred friends sign up or make a purchase.
-   **[Referral Advocate Profile (POST)](atlas.en-us.referral_marketing.meta/referral_marketing/connect_resources_referral_advocate_profile.htm)**
    View the profile of a referral program advocate.
