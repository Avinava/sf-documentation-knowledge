# Response Bodies

Here’s a list of Referral Marketing API response bodies.

-   **[Redeem Voucher](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_industries_redeem_voucher_output.htm)**
    Output representation of the details of a redeemed voucher.
-   **[Advocate Enrollment Transaction Journal](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_member_enrollment_transaction_journal_output.htm)**
    Output representation of the details of a transaction journal created for the enrolled referral program advocate.
-   **[Referral Event](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_referral_event_output.htm)**
    Output representation of the request to create a referral event record.
-   **[Referral Advocate Enrollment](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_referral_member_enrollment_output.htm)**
    Output representation of the request to create a referral program advocate.
-   **[Referral Advocate Profile](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_referral_advocate_profile_output.htm)**
    Represents the output for Referral Advocate Profile API.
-   **[Referral Promotion](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_referral_promotion_output.htm)**
    Represents the output for Referral Promotion.
-   **[Referral Statistics](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_referral_statistics_output.htm)**
    Represents the response for Referral Statistics.
-   **[Referral Info](atlas.en-us.referral_marketing.meta/referral_marketing/connect_responses_referral_info_output.htm)**
    Represents the response for list of referrals.
