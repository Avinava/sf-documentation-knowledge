# Record Rollup Definitions Tooling API Objects

Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata.

-   **[RecordAggregationDefinition](atlas.en-us.psc_api.meta/psc_api/tooling_api_objects_recordaggregationdefinition.htm)**
    Represents a data aggregation from one object to another object to which it is connected by other objects in the data model. This object is available in API version 59.0 and later.
-   **[RecordAggregationJoinCondition](atlas.en-us.psc_api.meta/psc_api/tooling_api_objects_recordaggregationjoincondition.htm)**
    Represents a condition in a join between two record aggregation objects. This object is available in API version 59.0 and later.
-   **[RecordAggregationObject](atlas.en-us.psc_api.meta/psc_api/tooling_api_objects_recordaggregationobject.htm)**
    Represents an object in the record aggegation join sequence. This object is available in API version 59.0 and later.
-   **[RecordAggregationObjectFilter](atlas.en-us.psc_api.meta/psc_api/tooling_api_objects_recordaggregationobjectfilter.htm)**
    Represents a filter that is applied to the records of an object in the record aggregation join sequence. This object is available in API version 60.0 and later.
