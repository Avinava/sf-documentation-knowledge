# Business APIs

Return search results or retrieve unified object search data.

This table lists the available resources.

| Resource | Description |
| --- | --- |
| /connect/criteria-based-search/configurations (GET) | Retrieve the unified object search data based on the input criteria and the search configuration defined by the administrator. |
| /connect/criteria-based-search/searchable-object/results (POST) | Returns the results for search based on the searchable object configuration and search criteria configuration defined by the admin. |
