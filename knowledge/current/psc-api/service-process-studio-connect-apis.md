# Service Process Studio Connect APIs

Use the Service Process Studio Connect API to create an extended Case object that includes custom data about a customer request such as a fee reversal or address update.

-   **[Resources](atlas.en-us.psc_api.meta/psc_api/service_process_resource.htm)**
    Use Service Process Studio API resources to create, update, and retrieve information about service processes.
-   **[Request Bodies](atlas.en-us.psc_api.meta/psc_api/service_process_requests.htm)**
    Use Service Process Studio API request bodies to request service process information.
-   **[Response Bodies](atlas.en-us.psc_api.meta/psc_api/service_process_responses.htm)**
    Use Service Process Studio API response bodies to review retrieved service process information.
