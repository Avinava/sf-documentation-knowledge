# Resources

Here’s a list of Public Sector Solutions API resources.

-   **[Action Plan Templates (GET, POST)](atlas.en-us.psc_api.meta/psc_api/connect_resources_action_plan_template.htm)**
    API to create and get Action Plan Template items. An Action Plan Template has a default set of parameters required to create an Action Plan.
-   **[Benefit Disbursements (POST)](atlas.en-us.psc_api.meta/psc_api/connect_resources_benefit_disbursement.htm)**
    Create benefit disbursements for a benefit session to disburse benefits to participants enrolled in the benefit session.
-   **[Benefit Sessions of a Care Plan (GET)](atlas.en-us.psc_api.meta/psc_api/connect_resources_get_benefirassignments_careplan.htm)**
    Get the details of benefit sessions enrolled by participants in a care plan.
-   **[Cancel File Upload](atlas.en-us.psc_api.meta/psc_api/connect_resources_cancel_file_upload.htm)**
    This API sends a request to cancel the file upload process.
-   **[Care Plans (GET, PATCH)](atlas.en-us.psc_api.meta/psc_api/connect_resources_get_careplans_details.htm)**
    Retrieve the care plan details. Fetches benefit assignments and goal assignments associated with the given care plan. Update a care plan record.
-   **[Care Plans (POST)](atlas.en-us.psc_api.meta/psc_api/connect_resources_create_careplans_records.htm)**
    Create care plan records. Associate benefit assignments and goal assignments when creating a care plan.
-   **[Care Plan Definitions (GET)](atlas.en-us.psc_api.meta/psc_api/connect_resources_get_care_plan_definitions.htm)**
    Retrieve details about a specified definition record that is related to goal definition and the benefits of a care plan.
-   **[Care Plan Templates (GET)](atlas.en-us.psc_api.meta/psc_api/connect_resources_get_careplan_template_child_records.htm)**
    Retrieve the care plan template details. Fetches the care plan template benefits and care plan template goals associated with the given care plan template.
-   **[Care Plan Templates (POST)](atlas.en-us.psc_api.meta/psc_api/connect_resources_create_careplan_template_child_records.htm)**
    Create care plan template records. Associate the benefits and goals with the care plan template.
-   **[Complete File Upload](atlas.en-us.psc_api.meta/psc_api/connect_resources_complete_file_upload.htm)**
    This API sends a request to upload the files.
-   **[Initiate File Upload](atlas.en-us.psc_api.meta/psc_api/connect_resources_initiate_file_upload.htm)**
    This API sends a request to initiate a multi-part upload. It returns a response with an upload ID, which is a unique identifier for the multi-part upload. This identifier is optional for single part upload.
