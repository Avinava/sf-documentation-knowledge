# Request Bodies

Expression Set API includes these request bodies.

-   **[Business Rules Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_business_rules_input.htm)**
    Input representation of an expression set.
-   **[Context Definition Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_context_definition_input.htm)**
    Input representation of the context definitions in an expression set.
-   **[Expression Set Advanced Condition Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_advanced_condition_step.htm)**
    Input representation of an advanced condition step in an expression set.
-   **[Expression Set Aggregation Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_aggregation_step.htm)**
    Input representation of an aggregation step in an expression set.
-   **[Expression Set Assignment Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_assignment_step.htm)**
    Input representation of an assignment step in an expression set.
-   **[Expression Set Condition Criteria Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_condition_criteria.htm)**
    Input representation of a condition criteria in an expression set.
-   **[Expression Set Condition Expression Step](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_condition_expression_step.htm)**
    Input representation of an expression set condition step.
-   **[Expression Set Custom Element Parameter Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_custom_element_parameter.htm)**
    Input representation of a custom element parameter in an expression set.
-   **[Expression Set Custom Element Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_custom_element_step.htm)**
    Input representation of a custom element step in an expression set.
-   **[Expression Set DES Token Mapping Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_des_token_mapping.htm)**
    Input representation of the DES token mapping in an expression set version step.
-   **[Expression Set Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_input.htm)**
    Input representation of the expression set create and update request.
-   **[Expression Set Lookup Table Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_lookup_table_step.htm)**
    Input representation of a lookup table step in an expression set.
-   **[Expression Set Options Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_options.htm)**
    Input representation of the options for executing an expression set.
-   **[Expression Set SubExpression Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_sub_expression_step.htm)**
    Input representation of a subexpression step in an expression set.
-   **[Expression Set Version Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_version.htm)**
    Input representation of an expression set version.
-   **[Expression Set Version Step Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_version_step.htm)**
    Input representation of a step in an expression set version.
-   **[Expression Set Version Variable Input](atlas.en-us.psc_api.meta/psc_api/connect_requests_expression_set_version_variable.htm)**
    Input representation of a variable in an expression set version.
