# Public Sector Solutions Standard Invocable Actions

Use invocable actions to create benefit disbursement records for an eligible benefit assignment.

-   **[Benefit Disbursement Action](atlas.en-us.psc_api.meta/psc_api/actions_obj_benefit_disbursement.htm)**
    Create a benefit disbursement for an eligible benefit assignment. For example, create a benefit disbursement request to offer monetary benefits through direct cash assistance or subsidies.
-   **[Run Record Aggregation Batch Processing Definition Action](atlas.en-us.psc_api.meta/psc_api/actions_obj_run_record_aggregation_batch_processing_definition.htm)**
    Runs a Data Processing Engine definition to process an asynchronous batch job that creates or updates record aggregation results.
