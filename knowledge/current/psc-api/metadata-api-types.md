# Metadata API Types

Metadata API enables you to access some types and feature settings that you can customize in the user interface.

-   **[SearchableObjDataSyncInfo](atlas.en-us.psc_api.meta/psc_api/meta_searchableobjdatasyncinfo.htm)**
    Retrieve, deploy, update , create, and view searchable object data sync information for criteria-based search and filter.
-   **[SearchCriteriaConfiguration](atlas.en-us.psc_api.meta/psc_api/meta_searchcriteriaconfiguration.htm)**
    Retrieve, deploy, update , create, and view search criteria configurations for criteria-based search and filter.
-   **[SearchResultActionConfig](atlas.en-us.psc_api.meta/psc_api/meta_searchresultactionconfig.htm)**
    Represents a collection of fields to set up the actions that a user can perform on results returned by criteria-based search.
