# PublicSectrSltn Namespace

The PublicSectrSltn namespace provides classes for Public Sector Solutions.

The following are the classes in the PublicSectrSltn namespace.

-   **[AssessmentResponses Class](atlas.en-us.psc_api.meta/psc_api/apex_class_PublicSectrSltn_AssessmentResponses.htm#apex_class_PublicSectrSltn_AssessmentResponses)**
    Use this class to store dynamic assessment responses Public Sector Solutions in the Assessment and AssessmentQuestionResponse standard objects.
-   **[BenefitDisbursementAdjDetail Class](atlas.en-us.psc_api.meta/psc_api/apex_class_publicsectrsltn_BenefitDisbursementAdjDetail.htm#apex_class_publicsectrsltn_BenefitDisbursementAdjDetail)**
    This class contains the benefit disbursement adjustment details for the eligible benefit assignment.
-   **[BenefitDisbursementDetail Class](atlas.en-us.psc_api.meta/psc_api/apex_class_publicsectrsltn_BenefitDisbursementDetail.htm#apex_class_publicsectrsltn_BenefitDisbursementDetail)**
    This class contains the benefit disbursement details for an eligible benefit assignment.
