# Engagement API Responses

Here’s the list of Engagement API response bodies.

-   **[Custom Field Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_custom_field_output.htm)**
    Output representation for creating an engagement interaction, topic, or attendee record with a custom field.
-   **[Engagement Attendee Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_engagement_attendee_output.htm)**
    Output representation to create an engagement attendee record.
-   **[Engagement Interaction Create Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_engagement_interaction_create_output.htm)**
    Output representation for creating an engagement interaction record.
-   **[Engagement Interaction Fetch Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_engagement_interaction_fetch_output.htm)**
    Output representation to fetch an engagement interaction record along with the status.
-   **[Engagement Interaction IDs Create Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_engagement_interaction_ids_create_output.htm)**
    Output representation of creating an engagement interaction.
-   **[Engagement Interaction Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_engagement_interaction_output.htm)**
    Output representation to fetch an engagement interaction record.
-   **[Engagement Topic Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_engagement_topic_output.htm)**
    Output representation to create an engagement topic record.
-   **[Engagements ID Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_service_excellence_id_output.htm)**
    Output representation of IDs for the engagement interaction, engagement attendee, and engagement topic records.
-   **[Engagement Status Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_service_excellence_status_output.htm)**
    Output representation to display the status of an API request.
