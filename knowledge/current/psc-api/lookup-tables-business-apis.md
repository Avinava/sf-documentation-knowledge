# Lookup Tables Business APIs

Connect REST APIs help customers, partners, and ISVs integrate with Salesforce software and UIs.

The runtime service for Lookup Table components is exposed through Connect APIs.

With Lookup Table Connect APIs, you can:

-   [Perform a lookup on a decision matrix](atlas.en-us.psc_api.meta/psc_api/connect_resources_bre_decision_matrix.htm "Performs a lookup on decision matrix rows based on the input values provided, and returns the row’s output.").
-   [Perform a lookup on a decision table](atlas.en-us.psc_api.meta/psc_api/connect_resources_bre_decision_table.htm "Performs a lookup on a decision table.").
-   [Retrieve lookup tables](atlas.en-us.psc_api.meta/psc_api/connect_resources_bre_search_lookup_tables.htm "Retrieve lookup tables.").

-   **[Resources](atlas.en-us.psc_api.meta/psc_api/lookup_table_resources.htm)**
    Lookup Table API includes these resources.
-   **[Request Bodies](atlas.en-us.psc_api.meta/psc_api/lookup_tables_requests.htm)**
    Lookup Table API includes these request bodies.
-   **[Response Bodies](atlas.en-us.psc_api.meta/psc_api/lookup_tables_responses.htm)**
    Lookup Table API includes these response bodies.
