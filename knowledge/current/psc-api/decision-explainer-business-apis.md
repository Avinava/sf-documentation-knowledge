# Decision Explainer Business APIs

Connect REST APIs help customers, partners, and ISVs integrate with Salesforce software and UIs.

The Decision Explainer runtime service for Business Rules Engine is exposed through Connect APIs.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=psc_api)

#### Note

Use the Decision Explainer resources when your Salesforce instance has Business Rules Engine and Decision Explainer, and you want to show explanations of the results of an expression set step to external users.

With Decision Explainer connect APIs, you can:

-   [Retrieve a list of explainability message templates](atlas.en-us.psc_api.meta/psc_api/connect_resources_rules_engine_message_templates_list.htm "Retrieves a list of explainability message templates that matches the given search parameters.")
-   [Retrieve details of a specific explainability message template](atlas.en-us.psc_api.meta/psc_api/connect_resources_rules_engine_message_template_detail.htm "Retrieves the details of an explainability message template for a specified template ID.")
-   [Retrieve explainability logs for expression step elements that have explanations configured](atlas.en-us.psc_api.meta/psc_api/connect_resources_rules_engine_explainability_logs.htm "Retrieves explainability logs for expression step elements for which explanation messages are configured.")

-   **[Resources](atlas.en-us.psc_api.meta/psc_api/decision_explainer_bre_resources.htm)**
    Decision Explainer API includes these resources.
-   **[Response Bodies](atlas.en-us.psc_api.meta/psc_api/decision_explainer_bre_responses.htm)**
    Decision Explainer API includes these response bodies.
