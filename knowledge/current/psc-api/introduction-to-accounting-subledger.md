# Introduction to Accounting Subledger

Give accounting teams neatly packaged financial CRM data for record-keeping.

| Available in: Lightning Experience in Performance, Enterprise, Developer, and Unlimited editions that have Accounting Subledger enabled. |
| --- |


Accounting Subledger consolidates data between Salesforce CRM and your accounting system to help you manage your accounting department. This consolidation helps you understand how and where funds are received, paid, and allocated.

-   **[Accounting Subledger Data Model Overview](atlas.en-us.psc_api.meta/psc_api/asl_data_model_overview.htm)**
    Learn about the objects and relationships within the Accounting Subledger data model.
-   **[Accounting Subledger Standard Objects](atlas.en-us.psc_api.meta/psc_api/asl_standard_objects.htm)**
    This section lists standard objects available for use with Accounting Subledger.
-   **[Accounting Subledger Fields on Standard Objects](atlas.en-us.psc_api.meta/psc_api/asl_fields_on_standard_objects.htm)**
    This section lists Accounting Subledger fields available with standard Salesforce objects. These fields are available only in orgs where Accounting Subledger is enabled.
-   **[Accounting Subledger Metadata Types](atlas.en-us.psc_api.meta/psc_api/accountingsubledger_metadata_apis_parent.htm)**
    This section lists metadata types available for use with Accounting Subledger. Metadata API enables you to access some types and feature settings that you can customize in the user interface.
-   **[Settings](atlas.en-us.psc_api.meta/psc_api/accounting_subledger_metadata_settings.htm)**
    Represents the organization settings to control the update of non-qualifying points balance for members when points are credited.
