# Responses

Learn more about the available response bodies of Record Rollup Definitions Business APIs.

Refer to the [HTTP response code](https://developer.salesforce.com/docs/atlas.en-us.260.0.chatterapi.meta/chatterapi/connect_error_responses.htm "HTML (New Window)") to check whether the request was successful, as well as viewing the error messages for the failed requests.

-   **[Data Processing Engine Definition List Output](atlas.en-us.psc_api.meta/psc_api/connect_responses_dpe_generation_output.htm)**
    Output representation of a Record Aggregation Data Processing Engine definition generation request.
-   **[Record Rollup Results](atlas.en-us.psc_api.meta/psc_api/connect_responses_record_rollup_result_output.htm)**
    Output representation of the record rollup results.
-   **[Record Rollup Result Column](atlas.en-us.psc_api.meta/psc_api/connect_responses_record_rollup_result_column_output.htm)**
    Output representation of the consolidated view of the table column data.
-   **[Record Rollup Result Last Updated Details](atlas.en-us.psc_api.meta/psc_api/connect_responses_record_rollup_result_last_updated_details_output.htm)**
    Represents the last updated information of the record rollup results.
-   **[Record Rollup Result Row](atlas.en-us.psc_api.meta/psc_api/connect_responses_record_rollup_result_row_output.htm)**
    Output representation of the consolidated view of the table row data.
