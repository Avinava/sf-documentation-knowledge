# Salesforce Surveys Standard Invocable Actions

Send email survey invitations to leads, contacts, and users in your org based on an action. Also, send customized notifications to users about important events or updates to the records that they’re working on.

-   **[Dynamic Send Survey Invitation Actions](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/actions_obj_dynamic_send_survey_invitation.htm)**
    Send customized notifications to users about important events or updates to the records that they’re working on. For example, notify account owners when a case is created.
-   **[Perform Survey Sentiment Analysis](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/actions_obj_survey_sentiment_analysis.htm)**
    Create or update the AI Sentiment Result records. You can get insights into the sentiments underlying survey responses and save the sentiment analysis in the SentimentAnalysisResult object.
-   **[Process Data Using Generative AI Action](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/actions_obj_process_data_using_gen_ai.htm)**
    Using Einstein Generative AI, perform Natural Language Processing (NLP) to summarize text, extract key phrases, analyze sentiment, and unlock valuable insights.
-   **[Send Survey Invitation Actions](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/actions_obj_send_survey_invitation.htm)**
    Send email survey invitations to leads, contacts, and users in your org based on an action. For example, send a survey invitation when a customer support case closes.
-   **[Transform NLP Action Results Action](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/actions_obj_transform_nlp_action_result.htm)**
    Transform an AI Natural Language Processing (NLP) result created by using Einstein Generative AI into an Apex Object record.
