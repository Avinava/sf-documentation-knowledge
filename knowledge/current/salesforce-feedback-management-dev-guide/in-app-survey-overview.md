# In-App Survey Overview

Get your environment ready before you create a sample in-app survey. With the Surveys setup ready in Salesforce, prepare survey details, create and send invitations, and collect survey responses.

Here’s a high-level overview of how you can create an in-app survey.

![A representation of the basic in-app survey creation flow.](/docs/resources/img/en-us/260.0?doc_id=dev_guides%2Ffeedback_management%2Fimages%2Fsurveys_basic_workflow.png&folder=salesforce_feedback_management_dev_guide)

-   **[Set Up Your Environment](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/surveys_set_up_your_environment.htm)**
    Get your Salesforce environment ready before you create your first in-app survey.
-   **[Lightning Web Components for Surveys](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/salesforce_surveys_lightning_web_components_overview.htm)**
    Lightning Web Components (LWCs) for Surveys provide a variety of question types that you can use in a survey.
-   **[Survey Questionnaire](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/salesforce_surveys_questionnaire.htm)**
    Create a basic survey by using Survey Builder and prepare the questionnaire by using various types of survey questions.
-   **[Survey Invitations](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/salesforce_surveys_invitations.htm)**
    Learn how to create survey invitations through API requests that are invoked from an Apex class.
-   **[Survey Responses](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/salesforce_surveys_responses.htm)**
    Use API responses to show survey questions in Salesforce.
