# Connect REST API Response Bodies

The successful execution of a request to a Connect REST API resource can return a response body in either JSON or XML format.

A request to a Connect REST API resource always returns an HTTP response code, whether the request was successful or not.

-   **[Survey Description Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_description_output.htm)**
    Output representation of the response to the Start Survey request.
-   **[Survey Detail Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_detail_output.htm)**
    Output representation of the details of the survey.
-   **[Survey Email](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_email.htm)**
    Survey invitation email response.
-   **[Survey NPS Question Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_nps_question_output.htm)**
    Output representation of the details of the NPS question on the survey page.
-   **[Survey Question Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_question_output.htm)**
    Output representation of the details of the question on the survey page.
-   **[Survey Question Page Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_question_page_output.htm)**
    The details of the question page in the survey.
-   **[Survey Response Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_response_output.htm)**
    Output representation of the response to the Submit Survey request. The response includes the status of the survey response API and the Thank You page details in the survey.
-   **[Survey Selection Question Choice Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_selection_question_choice_output.htm)**
    The details of a boolean, rating, single selection, or a multiple selection question’s choice on the survey page.
-   **[Survey Selection Question Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_selection_question_output.htm)**
    Output representation of the details of the boolean, rating, single-selection, or multiple-selection question on the survey page.
-   **[Survey Short-Text Question Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_short_text_question_output.htm)**
    Output representation of the details of the short-text question on the survey page.
-   **[Survey Text Question Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_text_question_output.htm)**
    Output representation of the details of the text question on the survey page.
-   **[Survey Thank You Page Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_thank_you_page_output.htm)**
    Output representation of the details of the Thank You page in the survey.
-   **[Survey URL Buttons Output](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/connect_responses_survey_u_r_l_buttons_output.htm)**
    Output representation of the details of the URL buttons on the Thank You page in the survey.
