# Survey Questionnaire

Create a basic survey by using Survey Builder and prepare the questionnaire by using various types of survey questions.

Use Survey Builder to create a basic survey that has these questions.

-   Rating question: How do you rate the onboarding process?
-   Short text question: Would you recommend this process to your colleagues?
-   Long text question: Describe your experience in a few words.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=salesforce_feedback_management_dev_guide)

#### Note

Only basic surveys support the SFM APIs.

See [Create a Basic Survey](https://help.salesforce.com/s/articleView?id=xcloud.task_create_basic_survey.htm&type=5&language=en_US "HTML (New Window)").
