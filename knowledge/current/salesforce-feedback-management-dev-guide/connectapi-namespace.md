# ConnectApi Namespace

The ConnectApi namespace (also called Connect in Apex) provides classes for creating surveys.

The following are the classes and the interface in the ConnectApi namespace.

-   **[Surveys Class](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_ConnectAPI_Survey_static_methods.htm#apex_ConnectAPI_Survey_static_methods)**
    Send survey invitations by email.
-   **[ConnectApi Input Classes](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_connectapi_input.htm)**
    Some ConnectApi methods take arguments that are instances of ConnectApi input classes.
-   **[ConnectApi Output Classes](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_connectapi_output.htm)**
    Most ConnectApi methods return instances of ConnectApi output classes.
