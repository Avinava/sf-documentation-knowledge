# Salesforce Surveys Apex Classes

Salesforce Surveys provides the ConnectApi namespace (also called Connect in Apex) and sfdc\_surveys namespace that contains classes for creating and sending survey invitations.

-   **[sfdc\_surveys Namespace](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_namespace_sfdc_surveys.htm)**
    The sfdc\_surveys namespace provides an interface for shortening survey invitations.
-   **[ConnectApi Namespace](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/salesforce_surveys_connect_api_namespace.htm)**
    The ConnectApi namespace (also called Connect in Apex) provides classes for creating surveys.
-   **[industriesNlpSvc](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_namespace_industriesNlpSvc.htm)**
    Stores the objects used in Industries Einstein Natural Language Processing (NLP) services.
