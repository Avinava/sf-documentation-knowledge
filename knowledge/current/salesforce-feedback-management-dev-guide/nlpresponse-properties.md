# NlpResponse Properties

The following are properties for NlpResponse.

-   **[summarizationResult](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_industriesNlpSvc_NlpResponse_summarizationResult.htm)**
    Represents the property that stores the NLP Summarization result performed for an NLP Operation. NLP operation can be SurveyLongSummarization and SurveyShortSummarization.
-   **[errors](atlas.en-us.salesforce_feedback_management_dev_guide.meta/salesforce_feedback_management_dev_guide/apex_industriesNlpSvc_NlpResponse_errors.htm)**
    Represents the property to store errors that occurred as a result of the NLP Operation.
