# Retired Resources

These User Interface API resources are retired.

-   **[Get Lookup Field Suggestions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_lookup_get.htm)**
    When a user edits a lookup field, use this resource to search for and display suggestions. You can search for most recently used matches, for matching names, or for any match in a searchable field. You can also specify lookup filter bindings for dependent lookups.
-   **[Get Lookup Field Suggestions for a Specified Object](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_lookup_object_get.htm)**
    When a user edits a lookup field, use this resource to search for and display suggestions for a specified object. You can search for most recently used matches, for matching names, or for any match in a searchable field. You can also specify lookup filter bindings for dependent lookups.
-   **[Get List Views for an Object (Deprecated)](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_list_views_object.htm)**
    Get a collection of list views associated with an object.
-   **[Get List View Records and Metadata (Deprecated)](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_list_views_records_md.htm)**
    Get record data and metadata for a list view.
-   **[Get Most Recently Used List View Records (Deprecated)](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_mru_list_views_records.htm)**
    Get record data for an object’s most recently used (MRU) list view.
-   **[Get Most Recently Used List View Metadata (Deprecated)](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_mru_list_views_md.htm)**
    Get metadata for an object’s most recently viewed (MRU) list view.
-   **[Get Most Recently Used List View Records and Metadata (Deprecated)](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_mru_list_views_records_md.htm)**
    Get record data and metadata for an object’s most recently used (MRU) list view.
