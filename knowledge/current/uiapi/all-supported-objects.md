# All Supported Objects

User Interface API supports many standard objects as well as all custom and external objects.

For general information about the supported objects for UI API, see [Supported Objects](atlas.en-us.uiapi.meta/uiapi/ui_api_get_started_supported_objects.htm "User Interface API supports all custom objects and external objects and many standard objects."). This is a list of supported standard objects.

As of Summer ’23, all new standard objects are auto-enabled for UI API usage. This list might not reflect the latest object support. To check if a specific object is supported, use the [/ui-api/object-info/{objectApiName} resource](atlas.en-us.uiapi.meta/uiapi/ui_api_get_started_supported_objects.htm "User Interface API supports all custom objects and external objects and many standard objects.").
