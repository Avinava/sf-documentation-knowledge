# List View Supported Objects

List views support a subset of the standard objects that User Interface API supports.

For general information about supported objects, see [Supported Objects](atlas.en-us.uiapi.meta/uiapi/ui_api_get_started_supported_objects.htm "User Interface API supports all custom objects and external objects and many standard objects.").

This list includes the standard objects supported for list views.

As of Summer ’23, all new standard objects are auto-enabled for UI API usage. This list might not reflect the latest object support. To check if list views are supported for a specific object in UI API, use the [/ui-api/list-info/{objectApiName} resource](https://developer.salesforce.com/docs/atlas.en-us.260.0.uiapi.meta/uiapi/ui_api_resources_lists_object_get.htm).
