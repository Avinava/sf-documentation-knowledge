# Filtered Lookup Info

Metadata for a lookup filter.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| controllingFields | String[] | A collection of controlling fields when the lookup filter is dependent on the source object. | Small, 41.0 | 41.0 |
| dependent | Boolean | Indicates whether the lookup filter is dependent on the source object. | Small, 41.0 | 41.0 |
| optionalFilter | Boolean | Indicates whether the lookup filter is optional. | Small, 41.0 | 41.0 |
