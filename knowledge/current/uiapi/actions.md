# Actions

Use these resources to get data and metadata about actions displayed in the Salesforce UI. Examples include standard and custom buttons, quick actions, and productivity actions.

-   **[Get Global Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_global.htm)**
    Get the actions in the Global Actions menu.
-   **[Get Record Detail Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_action_record.htm)**
    Get the actions on record detail pages.
-   **[Get Record Edit Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_action_record_edit.htm)**
    Get the actions on a record edit page.
-   **[Get Related List Actions with URL Parameters](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_related_list.htm)**
    Get the actions on related lists on a record detail page via URL parameters.
-   **[Get Related List Actions with a Request Body](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_related_list_actions_get.htm)**
    Get the actions on related lists for a record detail page. This resource uses the POST method so you can enter request parameters in the request body instead using complex URL parameters.
-   **[Get a Batch of Related List Actions with URL Parameters](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_related_lists_batch.htm)**
    Get the actions on a batch of related lists for a record via URL parameters.
-   **[Get a Batch of Related List Actions with a Request Body](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_related_list_actions_batch_get.htm)**
    Get the actions on a batch of related lists for a record. This resource uses the POST method so you can enter request parameters in the request body instead using complex URL parameters.
-   **[Get Related List Record Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_related_list_record.htm)**
    Get the actions on records in related lists.
-   **[Get List View Header Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_list_view.htm)**
    Get the header actions on list views.
-   **[Get List View Record Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_list_view_record.htm)**
    Get the record actions on list views.
-   **[Get List View Chart Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_list_view_chart.htm)**
    Get the chart actions on list views.
-   **[Get Lightning Page Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_flexipage.htm)**
    Get the actions on Lightning pages (FlexiPages).
-   **[Get Lookup Field Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_lookup.htm)**
    Get the actions on lookup fields.
-   **[Get MRU List View Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_mru_list.htm)**
    Get the header actions on the most recently used (MRU) list view for objects.
-   **[Get Photo Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_photo.htm)**
    Get the photo actions for pages. Currently, only group and user pages support photo actions.
-   **[Get a Quick Action Layout](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_quick_action_layout.htm)**
    Get information about the record layout used by a Create a Record or Update a Record quick action.
-   **[Perform Quick Actions](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_perform_action.htm)**
    Perform a quick action that creates or updates a record.
-   **[Update a Record with a Quick Action](atlas.en-us.uiapi.meta/uiapi/ui_api_resources_update_action.htm)**
    Perform an Update a Record quick action. To perform a quick action that creates or updates a record, you can also use the POST method.
