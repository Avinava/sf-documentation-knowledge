# Delete a Dashboard

Delete a dashboard by sending a DELETE request to the Dashboard Results resource. Deleted dashboards are moved to the Recycle Bin.

## Example

This DELETE request /services/data/v34.0/analytics/dashboards/01ZD00000007S89MAE to the Dashboard Results resource deletes the dashboard and returns a 204 HTTP response code with no content in the response body.
