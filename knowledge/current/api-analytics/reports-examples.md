# Reports Examples

Learn how to run, create, edit, or delete reports with the Reports REST API. Running a report returns a fact map that describes report data. Learn how to read report fact maps.

For information about different ways to execute the Reports REST API, see Send REST Request with cURL and Authorization Through Connected Apps and OAuth 2.0.

-   **[Create a New Report](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_report_example_post_report.htm)**
    Create a new report using a POST request.
-   **[Run Reports Synchronously or Asynchronously](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_get_reportdata.htm)**

-   **[Get Report Metadata](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_get_reportmetadata.htm)**

-   **[Get a List of Report Types](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_reporttypes_example_get_reporttypes.htm)**
    Return a list of analytics notifications using a GET request.
-   **[Download Formatted Excel Reports Using the Reports REST API](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_download_excel.htm)**
    You can use the Reports REST API to request reports in printer-friendly Excel format.
-   **[List Asynchronous Runs of a Report](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_list_asyncreportruns.htm)**

-   **[Filter Reports on Demand](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_filter_reportdata.htm)**

-   **[List Recently Viewed Reports](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_list_recentreports.htm)**

-   **[Decode the Fact Map](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_factmap_example.htm)**

-   **[Get Report Data without Saving Changes to or Creating a Report](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_report_query_example.htm)**
    Run a report without creating a report or changing an existing one by making a POST request to the query resource. Get report data without filling up your org with unnecessary reports.
-   **[Save Changes to Reports](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_save_report.htm)**
    Save changes to a report by sending a PATCH request to the Report resource.
-   **[Clone Reports](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_clone_report.htm)**
    Creates a copy of a custom, standard, or public report by sending a POST request to the Report List resource.
-   **[Delete Reports](atlas.en-us.api_analytics.meta/api_analytics/sforce_analytics_rest_api_delete_report.htm)**
    Delete a report by sending a DELETE request to the Report resource. Deleted reports are moved to the Recycle Bin.
