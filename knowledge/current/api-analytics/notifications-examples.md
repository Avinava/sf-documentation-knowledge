# Notifications Examples

Learn how to refresh, create, edit, copy, and delete analytic notifications. Analytic notifications are surfaced in the Salesforce user interface as report subscriptions, dashboard subscriptions, or Analytics notifications.

-   **[Get Analytics Notifications](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_notification_example_get_notifications.htm)**
    Return a list of analytics notifications using a GET request.
-   **[Create an Analytics Notification](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_notification_example_post_notifications.htm)**
    Create an Analytics Notification using a POST request.
-   **[Save Changes to an Analytics Notification](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_notification_example_put_notifications.htm)**
    Save Changes to an Analytics Notification using a PUT request.
-   **[Delete an Analytics Notification](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_notification_example_delete_notifications.htm)**
    Delete an Analytics Notification using a DELETE request. Once deleted, the analytics notification can't be recovered.
-   **[Check Limits for Analytics Notifications](atlas.en-us.api_analytics.meta/api_analytics/analytics_api_notification_example_get_notifications_limits.htm)**
    Check analytics notification limits using a GET request.
