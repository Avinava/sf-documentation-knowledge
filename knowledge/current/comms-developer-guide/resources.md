# Resources

Here’s a list of Communications Cloud Sales API resources.

-   **[Recipient Group](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_recipient_group.htm)**
    Update a recipient group by adding or removing recipients who are associated with offers or products that share the same configuration.
-   **[Split Quote Line Item](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_split_quote_line_item.htm)**
    Enable the splitting of a quote line item associated with a recipient group into multiple quote line items—one for each recipient within the group.
-   **[Product Details (POST)](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_product_details.htm)**
    Get product details, such as attributes, hierarchy, or cardinality, for a specified product ID. This API is a composite API for Product Discovery.
-   **[Configuration (POST)](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_product_configurator_configure.htm)**
    Retrieve and update a product’s configuration from a configurator. Execute configuration rules and notify users of any violations for changes to product bundle, attributes, or product quantity within a bundle. Additionally, get pricing details for the configured bundle.
-   **[Cart to Order (POST)](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_cart_order.htm)**
    Create single order creation for all items present in the cart. An order is generated for the entire cart as a whole, rather than for individual cart items.
-   **[Products List (POST)](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_products_list.htm)**
    Get a list of products for a specified catalog, category, or subcategory. This API is a composite API for Product Discovery.
-   **[Bulk Product Details (POST)](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_bulk_product_details.htm)**
    Retrieve details for multiple products. This API is a composite API for Product Discovery.
-   **[Place Sales Transaction (POST)](atlas.en-us.comms_developer_guide.meta/comms_developer_guide/connect_resources_place_sales_transaction.htm)**
    Create a sales transaction, such as an order or a quote, with integrated pricing and configuration. Additionally, update an order or a quote, and insert and delete order or quote line items to calculate the estimated tax.
