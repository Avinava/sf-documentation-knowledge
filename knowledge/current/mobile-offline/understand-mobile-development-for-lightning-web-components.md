# Understand Mobile Development for Lightning Web Components

The best way to start building mobile-ready components is to deep dive into Lightning Web Components first.

Accelerate your mobile development journey with these resources.

-   [Build Lightning Web Components](https://trailhead.salesforce.com/en/content/learn/trails/build-lightning-web-components "HTML (New Window)")

    Start with learning how to build Lightning web components before you can build mobile-ready Lightning web components.

-   [Set Up Your Salesforce Mobile Developer Tools for Lightning Web Components](https://trailhead.salesforce.com/content/learn/projects/set-up-your-salesforce-mobile-developer-tools-for-lightning-web-components "HTML (New Window)")

    Learn how to configure your local workspace with the tools needed for developing and testing your mobile-ready Lightning web components.

-   [Quick Start: Salesforce DX](https://trailhead.salesforce.com/content/learn/projects/quick-start-salesforce-dx "HTML (New Window)") and [App Development with Salesforce DX](https://trailhead.salesforce.com/content/learn/modules/sfdx_app_dev "HTML (New Window)")

    Learn about modern development tools for developing on the Salesforce platform. These tools are essential for being successful with Lightning web components.

-   [Transform Your Business with Mobile](https://trailhead.salesforce.com/en/content/learn/trails/salesforce1_mgmt "HTML (New Window)")

    Learn how mobile apps can transform your business and improve employee productivity.

-   [Develop with Mobile SDK](https://trailhead.salesforce.com/en/content/learn/trails/mobile_sdk_intro "HTML (New Window)")

    Create your own iOS, Android, or hybrid mobile apps powered by the Salesforce Platform.

-   [Developer Intermediate](https://trailhead.salesforce.com/en/content/learn/trails/force_com_dev_intermediate "HTML (New Window)")

    Take your apps to the next level with powerful integration and mobile tools.

-   [**Video:** Salesforce Mobile Tools](https://www.youtube.com/playlist?list=PLgIMQe2PKPSJsVvAdRvJbot0KjD_XpzT5 "HTML (New Window)")

    Install and configure your mobile tools with follow-along videos.
