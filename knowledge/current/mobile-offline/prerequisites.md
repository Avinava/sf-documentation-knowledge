# Prerequisites

Ensure you’re ready for this tutorial by verifying that you have the right software installed and configured, and your org has Lightning web components enabled for mobile users.

-   [Salesforce CLI](https://developer.salesforce.com/docs/atlas.en-us.260.0.sfdx_setup.meta/sfdx_setup/sfdx_setup_intro.htm "HTML (New Window)") is installed and up-to-date.
-   [VS Code](https://code.visualstudio.com/download "HTML (New Window)") is installed and up-to-date.
-   [Salesforce Extension Pack for VS Code](https://marketplace.visualstudio.com/items?itemName=salesforce.salesforcedx-vscode "HTML (New Window)") is installed and up-to-date.

    ![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=mobile_offline)

    #### Note

    Ensure all prerequisites listed on the extension pack page are also satisfied.
