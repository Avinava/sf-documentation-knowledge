# Salesforce Scheduler Custom Objects

This section provides details on custom objects, including junction objects. The Junction Object is a custom object that enables you to link a custom object with a large number of child records and link each child record to a large number of parent objects.

Salesforce Scheduler is available for an extra cost in Lightning Experience.

Available in: Enterprise, Unlimited, Performance, and Developer editions.

-   **[ActionableListMbrInvitation](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_actionablelistmbrinvitation.htm)**
    Represents a relationship between an Actionable List Member and an Appointment Invitation. This object is available in API version 59.0 and later.
-   **[AppointmentTopicTimeSlot](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_appointmenttopictimeslot.htm)**
    Represents a lookup to a work type or a work type group for a time slot This object is available in API version 52.0 and later.
-   **[EngagementChannelType](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_engagementchanneltype.htm)**
    Represents a channel through which a customer can be reached for communication. The Engagement Channel Type object supports only the English language. This object is available in API version 48.0 and later.
-   **[EngagementChannelWorkType](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_engagementchannelworktype.htm)**
    Represents the relationship between an Engagement Channel Type object and a Work Type object for Salesforce Scheduler. This object is available in API version 56.0 and later.
-   **[ServiceTerritoryWorkType](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_serviceterritoryworktype.htm)**
    Represents the relationship between a ServiceTerritory object and a WorkType object for Salesforce Scheduler appointments. This object is available in API version 45.0 and later.
-   **[ShiftEngagementChannel](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_shiftengagementchannel.htm)**
    Represents the relationship between a Shift object and an Engagement Channel Type object for Salesforce Scheduler. This object is available in API version 56.0 and later.
-   **[ShiftWorkTopic](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_shiftworktopic.htm)**
    Represents the relationship between a Shift object and a Work Type or Work Type Group object for Salesforce Scheduler. This object is available in API version 56.0 and later.
-   **[WaitlistServiceResource](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_waitlistserviceresource.htm)**
    Represents the relationship between the Waitlist object and the Service Resource object for Salesforce Scheduler. This object is available in API version 58.0 and later.
-   **[WaitlistWorkType](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_waitlistworktype.htm)**
    Represents the relationship between the Waitlist object and the Work Type object for Salesforce Scheduler. This object is available in API version 58.0 and later.
-   **[WorkTypeGroupMember](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_worktypegroupmember.htm)**
    Represents the relationship between a work type and the work type group it belongs to. This object is available in API version 45.0 and later.
