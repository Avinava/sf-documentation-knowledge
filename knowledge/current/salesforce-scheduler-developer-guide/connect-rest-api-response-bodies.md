# Connect REST API Response Bodies

The successful execution of a request to a Connect REST API resource can return a response body in either JSON or XML format.

A request to a Connect REST API resource always returns an HTTP response code, whether the request was successful or not.

-   **[Available Territory Slots](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_available_territory_slots.htm)**
    Slot’s start and end time and available resources in the specified territory.
-   **[Available Territory Slots List](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_available_territory_slots_list.htm)**
    Available territory slots.
-   **[Available Territory Slots Output](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_available_territory_slots_output.htm)**
    Available territory slots result.
-   **[Engagement Channel Type List Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_get_engagement_channel_type_list_resul.htm)**
    Output representation of the engagement channel type result.
-   **[Engagement Channel Type Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_get_engagement_channel_type_resul.htm)**
    Output representation of the details of the engagement channel type.
-   **[Engagement Channel Type Output](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_get_engagement_channel_type_output.htm)**
    Output representation of the list of the engagement channel types.
-   **[Extended Fields](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_scheduler_extended_fields_output.htm)**
    Output representation of the extended fields for Salesforce Scheduler.
-   **[Group Appointments](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_get_group_appointments_output.htm)**
    Output representation of the response that contains the retrieved group appointments.
-   **[Group Appointments List Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_get_group_appointments_list_resul.htm)**
    Output representation of the list of group appointments.
-   **[Group Appointment Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_group_appointment_resul.htm)**
    Output representation that contains the details of a specific group appointment.
-   **[Service Appointment Output](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_service_appointment_output.htm)**
    Output of the service appointment POST method.
-   **[Service Appointment Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_service_appointment_result.htm)**
    Result of the POST method.
-   **[Service Territories Output](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_service_territories_output.htm)**
    Output for the service territories GET method.
-   **[Service Territories List Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_service_territories_list_result.htm)**
    List of service territories returned as per the query in the GET service territories call.
-   **[Service Territories Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_service_territories_result.htm)**
    Represents a service territory result.
-   **[Waitlist Analytics](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_analytics.htm)**
    Output representation of the waitlist analytics.
-   **[Waitlist Check In](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_check_in_output.htm)**
    Output representation of the waitlist check in request.
-   **[Waitlist Check In Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_check_in_result.htm)**
    Output representation of the waitlist check in result.
-   **[Waitlist Details](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_details.htm)**
    Output representation of the waitlist details.
-   **[Waitlist List Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_list_result.htm)**
    Output representation of the result of the list of waitlists.
-   **[Waitlist Participant Details](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_participant_participant.htm)**
    Output representation of the waitlist participant details.
-   **[Waitlist Participant Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_participant_result.htm)**
    Output representation of a waitlist participant result. A waitlist participant is a customer who is added to a waitlist.
-   **[Waitlist Result](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_output.htm)**
    Output representation of the waitlist result.
-   **[Waitlist Service Resource](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_service_resource.htm)**
    Output representation of the waitlist service resource. Service resources are individual users who can attend customer appointments.
-   **[Waitlist Work Type Group](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/connect_responses_waitlist_work_type_group.htm)**
    Output representation of the waitlist work type group. A work type group is a general appointment category or topic, such as a home loan or an investment.
