# Enable and Configure Concurrent Scheduling

You must enable concurrent scheduling to let users schedule multiple service appointments in the same time slot. Also, there are some things to keep in mind when you implement concurrent scheduling.

For more information, see [Enable and Configure Concurrent Scheduling](https://help.salesforce.com/s/articleView?id=platform.ls_set_up_concurrent.htm&type=5&language=en_US "HTML (New Window)") in Salesforce Scheduler Help.
