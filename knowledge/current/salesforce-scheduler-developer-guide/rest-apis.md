# REST APIs

Use Salesforce Scheduler REST APIs to get appointment time slots and available service resources based on work type groups and service territories.

#### Note

Salesforce Scheduler is built on the Salesforce platform (also known as Lighting Experience). The Salesforce platform specific API rate limits apply to the Salesforce Scheduler REST APIs. For more information on Lightning platform common limits and allocations, see [API Request Limits and Allocations](https://developer.salesforce.com/docs/atlas.en-us.260.0.salesforce_app_limits_cheatsheet.meta/salesforce_app_limits_cheatsheet/salesforce_app_limits_platform_api.htm "HTML (New Window)").
