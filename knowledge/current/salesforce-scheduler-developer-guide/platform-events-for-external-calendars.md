# Platform Events for External Calendars

Use the AppointmentSchedulingEvent to notify subscribers when an appointment schedule is added, updated, or deleted.

The following is the platform event object available within the Salesforce Scheduler platform.

-   **[AppointmentSchedulingEvent](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/sforce_api_objects_appointmentschedulingevent.htm)**
    Notifies subscribers when an appointment schedule is added, updated, or deleted. This object is available in API version 50.0 and later.
