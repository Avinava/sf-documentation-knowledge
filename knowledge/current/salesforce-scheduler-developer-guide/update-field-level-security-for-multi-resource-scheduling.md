# Update Field Level Security for Multi-Resource Scheduling

Update the field-level security settings for the Primary Resource field on the Assigned Resource object for user profiles that need to access multi-resource scheduling.

1.  From Setup, open **Object Manager**.
2.  Click **Assigned Resource** to open it.
3.  Click **Fields & Relationships**, and then select **Primary Resource**.
4.  Click **Set Field-Level Security**.
5.  Select the **Visible** checkbox for all profiles that need access to multi-resource scheduling in Salesforce Scheduler, including the System Administrator.
6.  Save your changes.
