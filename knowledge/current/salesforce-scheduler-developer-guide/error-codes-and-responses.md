# Error Codes and Responses

When using the APIs in Salesforce Scheduler, you may encounter error codes and messages under certain conditions. For each Scheduler API, it lists the potential error codes, the associated error messages, and a description of what causes the error to occur.

The following is a list of error code details within the Salesforce Scheduler platform.

-   **[Appointment Candidates (GET)](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/appointmentcandidates_get.htm)**
    Contains the error codes and error code messages related with the Salesforce Scheduler API for getAppointmentSlots (GET) resource.
-   **[Appointment Slots (GET)](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/appointmentslots_get.htm)**
    Contains the error codes and error code messages related with Salesforce Scheduler API for the getAppointmentSlots (GET) resource.
-   **[Available Territory Slots (POST)](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/availableterritoryslots.htm)**
    Contains the error codes and error code messages related with Salesforce Scheduler API for the available-territory-slots (POST) resource.
-   **[Service Appointments (PATCH)](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/serviceappointment_patch.htm)**
    Contains the error codes and error code messages related with Salesforce Scheduler API for the service-appointments (PATCH) resource.
-   **[Service Appointments (POST)](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/serviceappointment_post.htm)**
    Contains the error codes and error code messages related with Salesforce Scheduler API for the service-appointments (POST) resource.
-   **[Service Territories (GET)](atlas.en-us.salesforce_scheduler_developer_guide.meta/salesforce_scheduler_developer_guide/serviceterritories_get.htm)**
    Contains the error codes and error code messages related with Salesforce Scheduler API for the service-territories (GET) resource.
