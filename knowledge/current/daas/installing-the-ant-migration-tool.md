# Installing the Ant Migration Tool

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=daas)

#### Note

The Ant Migration Tool is retired with Spring ’24. The tool continues to function for future API versions but isn’t updated with new functionality and isn’t supported. To manage metadata changes, switch to Salesforce CLI for a modern, supported developer experience.

Before you install the Ant Migration Tool you will need Java and Ant installed on your local machine. Then you can download the Ant Migration Tool from a Salesforce organization.

1.  Install Java and Ant, as described in [Prerequisites for Using the Ant Migration Tool](atlas.en-us.daas.meta/daas/forcemigrationtool_prereq.htm).
2.  Download the Ant Migration Tool, as described in [Install the Ant Migration Tool](atlas.en-us.daas.meta/daas/forcemigrationtool_install.htm "Follow these steps to download and install the Ant Migration Tool.").
