# HealthCloudExt Namespace

The HealthCloudExt namespace provides an interface and classes for Health Cloud to integrate with a source electronic health records (EHR) system.

The HealthCloudExt namespace includes these classes and interfaces.

-   **[HealthCloudExt Namespace for Intelligent Appointment Management](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_namespace_healthcloudext_appointment_management.htm)**
    The HealthCloudExt namespace provides an interface and classes for Health Cloud to integrate with a source electronic health records (EHR) system.
-   **[HealthCloudExt Namespace for Benefit Verification](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_namespace_healthcloudext_benefit_verification.htm)**
    The HealthCloudExt namespace provides an interface and classes for Health Cloud to integrate with a source electronic health records (EHR) system.
-   **[HealthCloudExt Namespace for Unified Health Scoring](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_namespace_healthcloudext_unified_healthscore.htm)**
    The HealthCloudExt namespace provides an interface and methods for Health Cloud to store health score information in the action logs.
-   **[HealthCloudExt Namespace for Intelligent Content Management](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_reference_healthcloudext_intelligent_content_management.htm)**
    The HealthCloudExt namespace provides an interface and classes for Health Cloud to manage Intelligent Content.
