# BookingStatus Enum

Represents a possible status of a healthcare appointment.

The following are the values of the healthcloudext.BookingStatus enum.

| Canceledhealthcloudext.BookingStatus, Canceled | Indicates that a scheduled appointment has been canceled. |
| NoShowhealthcloudext.BookingStatus, NoShow | Indicates that a patient didn’t show up for a scheduled appointment. |
