# Health Cloud Fields on Standard Objects

Health Cloud adds standard and custom fields to some standard Salesforce objects to represent information about individuals. These fields are available only in orgs where Health Cloud is enabled.

-   **[Fields on Account](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_account_custom_fields.htm)**
    Health Cloud fields extend the standard Account object to represent information about patients and providers.
-   **[Fields on Contact](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_contact_custom_fields.htm)**
    Health Cloud fields extend the standard Contact object to represent information about an individual.
-   **[Fields on Event and Task (Activity)](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_task_custom_fields.htm)**
    Custom fields extend the standard Event and Task objects to represent information about an individual’s activities.
-   **[Fields on Lead](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_lead_custom_fields.htm)**
    Standard and custom fields extend the Lead object to represent information about an incoming patient.
-   **[Fields on Opportunity](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_opportunity_custom_fields.htm)**
    Standard and custom fields extend the Opportunity object to represent information about an incoming patient.
-   **[Fields on ProductRequest](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_productrequest_custom_fields.htm)**
    Health Cloud fields extend the standard ProductRequest object to associate it with care programs and to track modification history.
-   **[Health Cloud Referral Management Fields on Contact, Lead, and Opportunity](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/hc_referral_management_data_model_custom_fields.htm)**
    Standard and custom fields extend the Contact, Lead, and Opportunity objects for use in Health Cloud to represent information about referral management.
-   **[Fields on Service Appointment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/health_serviceappointment_custom_fields.htm)**
    Standard and custom fields extend the standard Service Appointment object to represent information about a healthcare appointment.
