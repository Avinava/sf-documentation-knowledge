# SuggestedResponseAssessmentService Methods

The following are methods for SuggestedResponseAssessmentService.

-   **[processPastAssessmentResponses](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_healthcloudext_SuggestedResponseAssessmentService_processPastAssessmentResponses.htm)**
    Calls the PreviousAssessmentResponses Connect API to retrieve suggestions based on prior responses.
-   **[processSuggestedResponses](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_healthcloudext_SuggestedResponseAssessmentService_processSuggestedResponses.htm)**
    Calls the prompt service to get the suggested responses.
-   **[mergeSuggestedResponses](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_healthcloudext_SuggestedResponseAssessmentService_mergeSuggestedResponses.htm)**
    Merges suggestion from both sources.
