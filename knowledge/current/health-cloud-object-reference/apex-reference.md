# Apex Reference

This Apex reference goes into detail about the built-in Apex classes, interfaces, methods, or enums for Health Cloud.

For Apex-related objects and SOAP API calls, see [Objects and SOAP API Calls and Headers for Apex](https://developer.salesforce.com/docs/atlas.en-us.260.0.api.meta/api/soap-apex.htm) in the SOAP API Developer Guide.

-   **[HealthCloudExt Namespace](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_namespace_healthcloudext.htm)**
    The HealthCloudExt namespace provides an interface and classes for Health Cloud to integrate with a source electronic health records (EHR) system.
-   **[id\_verification Namespace](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_namespace_id_verification.htm)**
    The id\_verification namespace provides an interface and classes for Health Cloud to verify the caller’s identity.
-   **[embeddedai Namespace](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/apex_namespace_embeddedai.htm)**
    The embeddedai namespace provides classes and methods to manage and represent records and data in Apex to support embedded AI features.
