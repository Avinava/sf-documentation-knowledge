# HL7 v2.3 Segments Supported in Salesforce Standard Objects

> Salesforce Standard objects support the following segments in HL7 v2.3.

# HL7 v2.3 Segments Supported in Salesforce Standard Objects

Salesforce Standard objects support the following segments in HL7 v2.3.

-   **[AL1 - Patient Allergy Information](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_AL1.htm)**
    The Patient Allergy Information (AL1) HL7 segment maps to the AllergyIntolerance object in Salesforce.
-   **[OBR - Observation Request Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_OBR.htm)**
    The Observation Request Segment (OBR) HL7 segment maps primarily to the ClinicalServiceRequest, DiagnosticSummary, and PatientMEdicalProcedure objects in Salesforce.
-   **[OBX - Observation Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_OBX.htm)**
    The Observation Segment (OBX) HL7 segment maps to the CareObservation object in Salesforce.
-   **[ORC - Common Order Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_ORC.htm)**
    The Common Order Segment (ORC) HL7 segment maps primarily to the ClinicalServiceRequest and MedicationRequest objects in Salesforce.
-   **[PID - Patient Identification](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_PID.htm)**
    The Patient Identification (PID) HL7 segment maps to the ContactPointPhone object in Salesforce.
-   **[PV1 - Patient Visit](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_PV1.htm)**
    The Patient Visit (PV1) HL7 segment maps to the Encounter object, among other objects in Salesforce.
-   **[PV2 - Patient Visit - Additional Information](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_PV2.htm)**
    The Patient Visit - Additional Information (PV2) HL7 segment maps primarily to the ClinicalEncounter object in Salesforce.
-   **[RXA - Pharmacy Administration Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_RXA.htm)**
    The Pharmacy Administration Segment (RXA) HL7 segment maps primarily to the PatientImmunization and Medication objects in Salesforce.
-   **[RXC - Pharmacy Component Order Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_RXC.htm)**
    The Pharmacy Component Order Segment (RXC) HL7 segment maps to the Medication and MedicationRequestobjects in Salesforce.
-   **[RXE - Pharmacy Encoded Order Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_RXE.htm)**
    The Pharmacy Encoded Order Segment (RXE) HL7 segment maps primarily to the Medication object in Salesforce.
-   **[RXR - Pharmacy Route Segment](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/map_hl7_RXR.htm)**
    The Pharmacy Route Segment (RXR) HL7 segment maps primarily to the PatientImmunization and PatientMedicationDosage objects in Salesforce.
