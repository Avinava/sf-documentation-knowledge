# Health Cloud FHIR APIs

Use Salesforce Healthcare API to securely connect and interact with a system that uses Fast Health Interoperability Resources (FHIR) APIs.

The FHIR standard defined by Health Level Seven (HL7) enables health care systems to work together seamlessly. Transforming data to FHIR allows you to quickly connect data sources like Electronic Medical Record (EMR) systems and research databases. FHIR also enables rapid data exchange in mobile and web apps.

To learn more, see the [Salesforce Health Care API](https://developer.salesforce.com/docs/industries/health/overview "HTML (New Window)") guide.
