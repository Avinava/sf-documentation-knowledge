# Appointment Management

Book, cancel or update an appointment for a patient.

These APIs are a part of the Intelligent Appointment Management (IAM) and calls into the system, while the [Intelligent Appointment Management Operations API](http://developer.salesforce.com/docs/atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/healthcare_appointment_booking_operations.htm) calls out to an external system.

-   **[Book Appointment (POST)](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/connect_resources_appointment_management_book_appointment.htm)**
    Create or book a health appointment.
-   **[Cancel Appointment (PATCH)](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/connect_resources_appointment_management_cancel_appointment.htm)**
    Cancel a health appointment.
-   **[Update Appointment (PUT)](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/connect_resources_appointment_management_update_appointment.htm)**
    Update an appointment.
-   **[Get Slots (POST)](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/connect_appointment_management_get_slots.htm)**
    Get slots associated with the provider or asset.
-   **[Get Resources (POST)](atlas.en-us.health_cloud_object_reference.meta/health_cloud_object_reference/connect_resources_get_resources.htm)**
    Search for healthcare care providers or assets using the backend search engine and data.
