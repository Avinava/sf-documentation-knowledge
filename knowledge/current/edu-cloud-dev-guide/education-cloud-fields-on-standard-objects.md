# Education Cloud Fields on Standard Objects

This section lists Education Cloud fields available with standard Salesforce objects. These fields are available only in orgs where Education Cloud is enabled.

| Available in: Lightning Experience in Performance, Enterprise, Developer, and Unlimited editions that have Education Cloud enabled. |
| --- |


-   **[ActionPlanTemplate](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_actionplantemplate.htm)**
    Represents the instance of an action plan template. This object is available in API version 60.0 and later.
-   **[ActionPlanTemplateAssignment](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_actionplantemplateassignment.htm)**
    Represents a junction between an action plant template's version and the target object (Care Plan Template, Benefit, and Goal Definition) associated with them. This object is available in API version 60.0 and later.
-   **[Assessment](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_assessment.htm)**
    Stores the header data for an assessment. This object is available in API version 63.0 and later.
-   **[AssessmentEnvelope](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_assessmentenvelope.htm)**
    Represents information about an envelope that contains the assessments related to a learner. This object is available in API version 62.0 and later.
-   **[AssessmentEnvelopeItem](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_assessmentenvelopeitem.htm)**
    Represents information about an item in an envelope that contains the assessments related to a learner. This object is available in API version 62.0 and later.
-   **[AssessmentQuestion](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_assessmentquestion.htm)**
    Represents the container object that stores the questions required for an assessment. This object is available in API version 63.0 and later.
-   **[AssessmentQuestionResponse](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_assessmentquestionresponse.htm)**
    Stores the responses submitted to an assessment. This object is available in API version 63.0 and later.
-   **[Award](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_award.htm)**
    Represents a person's or organization's professional awards. This object is available in API version 64.0 and later.
-   **[Benefit](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_benefit.htm)**
    Represents information about benefits associated with a program. This object is available in API version 57.0 and later.
-   **[BenefitAssignment](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_benefitassignment.htm)**
    Represents the enrollment information of an individual to a benefit. This object is available in API version 57.0 and later.
-   **[BenefitDisbursement](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_benefitdisbursement.htm)**
    Represents the allocation of an enrollee's benefit that can be made as monetary or non-monetary with different frequencies. This object is available in API version 57.0 and later.
-   **[BenefitType](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_benefittype.htm)**
    Represents information about the type of benefits that can be applied to an individual or a group. This object is available in API version 60.0 and later.
-   **[BusinessOperationsProcess](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_businessoperationsprocess.htm)**
    Represents a business operations process. This object is available in API version 63.0 and later.
-   **[BusinessProfile](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_businessprofile.htm)**
    Represents details about the business on the license or permit application. This object is available in API version 64.0 and later.
-   **[ComplianceControl](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_compliancecontrol.htm)**
    Represents a compliance control. This object is available in API version 63.0 and later.
-   **[CourseOffering](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_courseoffering.htm)**
    Represents an instance of a training course with location and date details of training courses. This object is available in API version 57.0 and later.
-   **[CourseOfferingRelationship](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_courseofferingrelationship.htm)**
    Represents a junction between a course offering and a related course offering. This object is available in API version 65.0 and later.
-   **[DocumentChecklistItem](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_documentchecklistitem.htm)**
    Represents a checklist item for a file documentation upload. This object is available in API version 66.0 and later.
-   **[GiftCommitmentSchedule](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_giftcommitmentschedule.htm)**
    Represents the schedule for fulfilling the commitment. This object is available in API version 64.0 and later.
-   **[GoalAssignment](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_goalassignment.htm)**
    Represents the assignment of a goal. This object is available in API version 57.0 and later.
-   **[GoalDefinition](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_goaldefinition.htm)**
    The definition of a care plan goal in the reusable PGI library that’s a part of Integrated Care Management. When instantiated, GoalDefinition records create GoalAssignment records that serve as goals in care plans. This object is available in API version 57.0 and later.
-   **[IndividualApplication](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_individualapplication.htm)**
    Represents information about the preferences of an applicant during program enrollment. This object is available in API version 57.0 and later.
-   **[Location](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_location.htm)**
    Represents details about a facility, building, room, or region. This object is available in API version 57.0 and later.
-   **[PersonCompetency](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_personcompetency.htm)**
    Represents the skills and competencies of a person. This object is available in API version 64.0 and later.
-   **[PersonEducation](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_personeducation.htm)**
    Represents information about the academic standing of an applicant. This object is available in API version 57.0 and later.
-   **[PersonEmployment](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_personemployment.htm)**
    Represents information about a person’s employment. This object is available in API version 57.0 and later.
-   **[PersonExamination](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_personexamination.htm)**
    Represents the examinations taken by a person. This object is available in API version 57.0 and later.
-   **[PersonLifeEvent](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_personlifeevent.htm)**
    Represents the life events of an Individual. Eg: Marriage, Birth of a baby, Birthday, Engagement, Divorce This object is available in API version 59.0 and later.
-   **[PreliminaryApplicationRef](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_preliminaryapplicationref.htm)**
    Represents details about saved applications. This object is available in API version 57.0 and later.
-   **[Program](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_program.htm)**
    Represents information about the enrollment and disbursement of benefits in a program. This object is available in API version 57.0 and later.
-   **[RecurrenceSchedule](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_recurrenceschedule.htm)**
    Represents a recurrence schedule. This object is available in API version 62.0 and later.
-   **[RegulatoryCode](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_regulatorycode.htm)**
    Represents the regulation code enforced by the regulatory body. This object is available in API version 64.0 and later.
-   **[ServiceTerritoryMember](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_serviceterritorymember.htm)**
    Represents a service resource who can be assigned in a service territory in Field Service, Salesforce Scheduler, Workforce Engagement, or Education Cloud. This object is available in API version 57.0 and later.
-   **[Waitlist](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_waitlist.htm)**
    Represents a queue to which drop in customers who visit the branch without an already scheduled appointment are added. This object is available in API version 61.0 and later.
