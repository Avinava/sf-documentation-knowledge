# Education Cloud Tooling API Objects

Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata.

For more information about Tooling API objects and to find a complete reference of all the supported objects, see [Introducing Tooling API](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_tooling.meta/api_tooling/intro_api_tooling.htm).

| Available in: Lightning Experience in Performance, Enterprise, Developer, and Unlimited editions that have Education Cloud enabled. |
| --- |


-   **[CourseWaitlistConfig](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_coursewaitlistconfig.htm)**
    Represents the configuration for Course Waitlists. This object is available in API version 66.0 and later.
-   **[LearningAchievementConfig](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_learningachievementconfig.htm)**
    Represents the mapping details between a Learning Achievement type and a Learning Achievement record type. This object is available in API version 59.0 and later.
-   **[LocationUse](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/sforce_api_objects_locationuse.htm)**
    Records the general purpose of the Location. This object is available in API version 57.0 and later.
