# Fundraising

This guide provides information about the objects and APIs that Fundraising uses.

| Available in: Lightning Experience |
| --- |
| Available in: Enterprise Unlimited and Developer Editions. |


-   **[Fundraising Standard Objects](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/npc_fundraising_standard_objects.htm)**
    Fundraising data model provides objects and fields to manage gifts and donors for your nonprofit or education organization.
-   **[Fundraising Fields on Other Objects](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/npc_fundraising_fields_standard_objects.htm)**
    Fundraising includes fields that are available on other Salesforce objects. These fields are available only in orgs with Nonprofit Cloud or Education Cloud when Fundraising is enabled.
-   **[Fundraising Business APIs](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/npc_fundraising_business_api_rest_reference.htm)**
    You can access Salesforce Fundraising APIs using REST endpoints. These REST APIs follow similar conventions as Connect REST APIs.
-   **[Fundraising Invocable Actions](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/fundraising_actions_parent.htm)**
    Use actions to add more functionality to your applications. Choose from standard actions, such as processing gift entries, update processed gift entries, manage gift default designations, and manage gift transaction designations.
-   **[Fundraising Metadata API Types](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/fundraising_metadata_api_parent.htm)**
    Metadata API enables you to access some types and feature settings that you can customize in the user interface.
-   **[Fundraising Tooling API Objects](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/npc_fundraising_tooling_api_object.htm)**
    Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata.
