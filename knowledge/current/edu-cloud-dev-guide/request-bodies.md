# Request Bodies

Learn more about the available request body of Education Cloud Business API.

-   **[Achievement Contributor Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_achievement_contributo.htm)**
    Input representation of contributor for a group achievement mapping.
-   **[Action Plan Template Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_action_plan_template_inpu.htm)**
    Input representation of the action plan template for a hold.
-   **[Course Offering Details Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_course_offering_inpu.htm)**
    Input representation of the course offering request object.
-   **[Course Offering Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_course_offering_input.htm)**
    Input representation of the course offering request.
-   **[Course Offering Schedule Details Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_course_offering_schedule_inpu.htm)**
    Input representation of a single course offering schedule to create and modify within the list of course offering request.
-   **[Course Offering Schedule Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_course_offering_schedule_input.htm)**
    Input representation of the list of course offering schedules to be created or updated for the course offering.
-   **[Custom Field Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_custom_field_input.htm)**
    Input representation of custom fields for the course offering.
-   **[Create Learning Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_create_learning_input.htm)**
    Input representation of the details to create learning objects, related outcomes, and foundations.
-   **[Create Learnings Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_create_learnings_input.htm)**
    Input representation of the details of the request to create learnings.
-   **[Edu External ID Details Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_edu_external_id_detail.htm)**
    Input representation of external fields associated with learning object.
-   **[Group Achievement Mapping Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_abstract_achievement_mappin.htm)**
    Input representation of a mapping to a given achievement that contains the shared attributes across the mapping types.
-   **[Hold Setup Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_hold_setup_input.htm)**
    Input representation of the request to create a hold.
-   **[Hold Violation Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_hold_violation_input.htm)**
    Input representation of the request to assign a hold to a student.
-   **[Learning Foundation Item Lookup Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_learning_foundation_item_looku.htm)**
    Input representation of the lookup details for a learning foundation item.
-   **[Learning Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_learnin.htm)**
    Input representation of the details that define an instructional object, which can be made available as a course, program, or an on-site experience for a contact.
-   **[Mentoring Benefit Assignment Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_mentoring_benefit_assignment_input.htm)**
    Input representation of the request to add a provider offering’s contact ID to the benefit assignment record.
-   **[Requirement Update Item Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_requirement_update_ite.htm)**
    Input representation of the learning requirements, such as prerequisites, corequisites, and recommendations during the update.
-   **[Single Achievement Mapping Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_concrete_achievement_mappin.htm)**
    Input representation of a mapping that’s associated with an existing learning achievement record.
-   **[Skill Achievement Mapping Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_skill_achievement_mappin.htm)**
    Input representation of a mapping that’s associated with an existing skill or must be created as a learning achievement.
-   **[Update Learning Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_update_learning_input.htm)**
    Input representation of the details of the learning along with related outcomes, and foundations for the update.
-   **[Update Learnings Input](atlas.en-us.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/connect_requests_update_learnings_input.htm)**
    Input representation of the request details to update learning records.
