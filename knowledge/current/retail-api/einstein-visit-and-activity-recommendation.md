# Einstein Visit and Activity Recommendation

This section provides a list of objects and fields that are related to Einstein Visit and Activity Recommendation.

-   **[Einstein Visit Recommendation Standard Objects](atlas.en-us.retail_api.meta/retail_api/industries_einstein_visit_recommendation_api_overview.htm)**
    Einstein Visit Recommendation data model provides you objects and fields to contain data about visit recommendation requests and visit recommendations. Einstein Visit Recommendation supports sales managers and field reps in scheduling visits to the right stores, at the right time.

Einstein Visit Activity Recommendation supports sales managers and field reps in scheduling tasks and assessment indicators for visits.
