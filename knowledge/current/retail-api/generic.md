## **mapValues**

This rule reads the Maps\_To\_\_c property of the workflow input structure records in order to map the input attribute described into the output field in the Maps\_To\_\_c property.

-   Description: Maps values from the input to the output structure without any processing.
-   Classname: Promotion2BoApiCoreWorkflowSteps
-   Method: mapValues.
