# Business Object API Workflows

The following workflows are provided in the default configuration.

| Workflow Name | Description |
| --- | --- |
| create | Creates promotions from simple promotion input. Can be also used to update promotions but all promotion required fields must be included in the promotion input. |
| copy | Copies the received Promotion |
| update | Updates promotion values from simple promotion input. |
| derive | Derives the received promotion to the target Account and Promotion Template |
