# Response Bodies

Here’s a list of Consumer Goods Business API response bodies.

-   **[Recommendation Decisions Output](atlas.en-us.retail_api.meta/retail_api/connect_responses_recommendation_decisions_output.htm)**
    Details of recommendation decision processed for the input recommendations.
-   **[Save Recommendation Decision Details](atlas.en-us.retail_api.meta/retail_api/connect_responses_save_recommendation_decision_detail.htm)**
    Details of a recommendation for which the recommendation decision is taken.
-   **[Save Recommendation Decision Status](atlas.en-us.retail_api.meta/retail_api/connect_responses_status.htm)**
    Output representation of the status of the save recommendation decision.
-   **[Begin Promotion Import Output](atlas.en-us.retail_api.meta/retail_api/connect_begin_promotion_imports_output.htm)**
    Output representation of the promotion import request.
-   **[Promotion Import Output](atlas.en-us.retail_api.meta/retail_api/connect_ingest_promotions_output.htm)**
    Output representation of the promotion import output request.
-   **[Import Status Output](atlas.en-us.retail_api.meta/retail_api/connect_responses_ingest_promotion_output.htm)**
    Represents the details of the ingest promotions import status.
