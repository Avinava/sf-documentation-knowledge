# Custom Objects for Direct Store Delivery

Consumer Goods Cloud data model provides custom objects and fields to support Direct Store Delivery.

-   **[cgcloud\_\_Route\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__route__c.htm)**
    Stores the details of the route object. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Route\_Template\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__route_template__c.htm)**
    Stores the details of the route template object that's to be used for routes. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour__c.htm)**
    Stores the details of the tour object. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_Check\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour_check__c.htm)**
    Stores the details of the tour check object that's used as part of tour execution for tour checking. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_Object\_Reference\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour_object_reference__c.htm)**
    Stores the details of the object references of a tour. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_Template\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour_template__c.htm)**
    Stores the details of the tour template object that's to be used for tours. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_Template\_Object\_Reference\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour_template_object_reference__c.htm)**
    Stores the details of the object references of a tour template. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_Template\_Tour\_Check\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour_template_tour_check__c.htm)**
    Stores the details of the relationship between tour template and tour check, which is used to define the relevant tour checks on a tour. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Tour\_Tour\_Check\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__tour_tour_check__c.htm)**
    Stores the details of the relationship between tour and tour check. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Vehicle\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__vehicle__c.htm)**
    Stores the details of the vehicle object. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Vehicle\_Warehouse\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__vehicle_warehouse__c.htm)**
    This object stores the relationship between a vehicle and a warehouse. This object is available in API version 54.0 and later.
-   **[cgcloud\_\_Route\_Account\_\_c](atlas.en-us.retail_api.meta/retail_api/sforce_api_objects_cgcloud__route_account__c.htm)**
    Stores the details of the relationship between routes and customers. This object is available in API version 54.0 and later.
