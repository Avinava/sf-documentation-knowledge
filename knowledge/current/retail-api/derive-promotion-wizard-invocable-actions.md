# Derive Promotion Wizard Invocable Actions

This class contains support for Data Source providers used in the DerivePromotion wizard.

-   **[getPromotionTemplates](atlas.en-us.retail_api.meta/retail_api/derive_promo_getpromotiontemplates.htm)**
    This method returns the list of possible Promotion Templates for derive workflow.
-   **[getAccounts](atlas.en-us.retail_api.meta/retail_api/derive_promo_getaccounts.htm)**
    This method returns the list of accounts valid for the target derived promotion.
-   **[isPromotionCommitted](atlas.en-us.retail_api.meta/retail_api/derive_promo_ispromotioncommitted.htm)**
    This method validates if the given promotion is on Committed phase.
-   **[isPromotionTemplateDerivable](atlas.en-us.retail_api.meta/retail_api/derive_promo_is_promo_template_deriveable.htm)**
    This method validates if the given promotion has a derivable promotion template.
