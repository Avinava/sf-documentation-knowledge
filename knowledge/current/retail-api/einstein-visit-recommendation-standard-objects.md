# Einstein Visit Recommendation Standard Objects

Einstein Visit Recommendation data model provides you objects and fields to contain data about visit recommendation requests and visit recommendations. Einstein Visit Recommendation supports sales managers and field reps in scheduling visits to the right stores, at the right time.

-   **[AiVisitRecommendation](atlas.en-us.retail_api.meta/retail_api/industries_einstein_objects_aivisitrecommendation.htm)**
    Represents the details of a visit recommendation. This object is available in API version 50.0 and later.
-   **[AiVisitRecommendRequest](atlas.en-us.retail_api.meta/retail_api/industries_einstein_objects_aivisitrecommendrequest.htm)**
    Represents the details of a visit recommendation request. This object is available in API version 50.0 and later.
