# RTR Report Classes

The RTR report-related APEX classes provide programmatic access to the TPM Real Time Reporting (RTR) report data.

Use these classes to:

-   Execute RTR reports.
-   Specify filters for the report execution.
-   Retrieve data from a Flatlist component.
-   Retrieve data from a ScoreCard component.
-   Create unit tests for the RTR report-related features and generate mock report results to test the different paths of your code.

-   **[RTRReportResult](atlas.en-us.retail_api.meta/retail_api/RTRReportResult_class.htm)**
    The RTRReportResult Apex class provides methods to access and process RTR reports.
-   **[RTRReportResult.ReportComponent](atlas.en-us.retail_api.meta/retail_api/ReportComponent_class.htm)**
    This class represents an RTR Report UI component.
-   **[RTRReportResult.ScoreCard](atlas.en-us.retail_api.meta/retail_api/ScoreCard_class.htm)**
    This class represents an RTR Report UI ScoreCard component.
-   **[RTRReportResult.Flatlist](atlas.en-us.retail_api.meta/retail_api/Flatlist_class.htm)**
    This class represents an RTR Report UI Flatlist component.
-   **[RTRReportResult.FlatlistRow](atlas.en-us.retail_api.meta/retail_api/FlatlistRow_class.htm)**
    This class represents a single row in an RTR Report UI Flatlist component.
-   **[RTRReportResult.FlatlistRowIterable](atlas.en-us.retail_api.meta/retail_api/FlatListRowIterable_class.htm)**
    This class represents a set of rows in an RTR Report UI Flatlist component.
-   **[RTRReportResult.MockException](atlas.en-us.retail_api.meta/retail_api/MockException_class.htm)**
    This class is used to create mock exceptions for the unit test execution.
-   **[RTRReportResult.MockRTRReportResult](atlas.en-us.retail_api.meta/retail_api/MockRTRReportResult_class.htm)**
    This class is used to create mock responses for the unit test execution.
-   **[RTRReportResult.MockScoreCard](atlas.en-us.retail_api.meta/retail_api/MockScoreCard_class.htm)**
    This class is used to create mock responses for the unit test execution.
-   **[RTRReportResult.MockFlatlist](atlas.en-us.retail_api.meta/retail_api/MockFlatlist_class.htm)**
    This class is used to create mock responses for the unit test execution.
-   **[RTRReportResult.MockFlatlistRow](atlas.en-us.retail_api.meta/retail_api/MockFlatlistRow_class.htm)**
    This class is used to create mock responses for the unit test execution.
