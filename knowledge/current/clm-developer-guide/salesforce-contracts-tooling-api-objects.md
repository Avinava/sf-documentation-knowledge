# Salesforce Contracts Tooling API Objects

Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata.

-   **[ContractTypeConfig](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/tooling_api_objects_contracttypeconfig.htm)**
    Represents the settings that define the characteristics and behavior of all contracts of that type. This object is available in API version 54.0 and later.
