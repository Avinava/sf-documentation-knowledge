# E-signature Notification Reminder

E-signature notification reminder settings.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| reminderDelay | Integer | Integer specifying the number of days after the recipient receives the envelope that reminder emails are sent to the recipient. | Small, 56.0 | 56.0 |
| reminderEnabled | Boolean | Indicates whether a reminder is sent. | Small, 56.0 | 56.0 |
| reminderFrequency | Integer | Integer specifying the interval in days between reminder emails. | Small, 56.0 | 56.0 |
