# Data True-Up Resources

Keep contract documents and Salesforce records perfectly aligned by synchronizing token values between them. From the Microsoft 365 add-in, you can pull the latest data from Salesforce into the document or push the document changes back to the Salesforce record. This simple, two-way process ensures that your information is always consistent across both locations, preventing data mismatches.

The available resource is:

| Resource | Description |
| --- | --- |
| /connect/content-link/load | Fetches the content data for a specified reference ID or set of content link IDs. |
| /connect/content-link/data-sync (POST) | Synchronize and save the updated values for a specified reference ID and a set of content link IDs. |

-   **[Content Link Load (POST)](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_resources_contentlink_load.htm)**
    Fetches the content data for a specified reference ID or set of content link IDs.
-   **[Data Sync (POST)](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_resources_contentlink_data_sync.htm)**
    Synchronize and save the updated values for a specified reference ID and a set of content link IDs.
