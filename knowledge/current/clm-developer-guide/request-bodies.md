# Request Bodies

Here’s a list of Salesforce Contracts API request bodies.

-   **[Content Link Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_content_link_input.htm)**
    Input representation of the request to fetch the content token data from the ContentLink object.
-   **[Content Link Data Sync Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_content_link_data_sync_input.htm)**
    Input representation of the entire request to synchronize token data from a document to sObjects.
-   **[Content Link Details Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_content_link_details_input.htm)**
    Input representation of the request to synchronize content token data from a document to sObjects. It specifies the ID of an individual content link and its corresponding new value to be updated.
-   **[Contract Action Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_contract_action_input.htm)**
    Input to get action details for the contract.
-   **[Contract Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_contract_input.htm)**
    Input to create and update contract.
-   **[Document Template Request](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_document_template_request.htm)**
    Input representation of a document template resource.
-   **[Document Template Update Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_document_template_update_input.htm)**
    Input representation to update a document template.
-   **[Envelope Documents Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_documents_reques.htm)**
    Input to get document details for e-signature.
-   **[Envelope Email Notification Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_email_notification_reques.htm)**
    Input to get email notification details.
-   **[Envelope Email Settings Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_email_settings_reques.htm)**
    Email settings input to get email details from API.
-   **[Envelope Notification Expiration Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_notification_expirations_reques.htm)**
    Input to get notification expiration information.
-   **[Envelope Notification Reminder Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_notification_reminders_reques.htm)**
    Input to get notification reminder information for e-signature document.
-   **[Envelope Notification Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_notification_reques.htm)**
    Input to get notification information for e-signature.
-   **[Envelope Recipient Details Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_recipient_detail_reques.htm)**
    Input to get recipient details.
-   **[Envelope Recipients Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_recipients_reques.htm)**
    Input to get different recipients list.
-   **[Envelope Send Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_send_reques.htm)**
    Input request to send e-signature details from API.
-   **[Envelope Void Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_envelope_void_reques.htm)**
    Input to get void envelope related details.
-   **[External Document Creation Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_external_doc_creation_input.htm)**
    Input representation of the request to create an external document.
-   **[External Document Metadata](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_external_document_metadata.htm)**
    Input representation of external document metadata.
-   **[Extraction Context Attribute Mapping Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_extraction_context_attribute_mapping_input.htm)**
    Input representation of attribute mappings used in the extraction context.
-   **[Extraction Context Mapping Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_extraction_context_mapping_input.htm)**
    Input representation of a extraction context mapping.
-   **[Extraction Context Use Case Mapping Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_extraction_context_use_case_mapping_input.htm)**
    Input representation of the use case-mapping configuration in the extraction context.
-   **[Object Reference Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_object_reference.htm)**
    Input representation of object reference.
-   **[Section Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_section.htm)**
    Input representation of a section.
-   **[Save External Document Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_save_external_document_input.htm)**
    Input representation to save external document to salesforce.
-   **[Template Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_template_input.htm)**
    Input representation to get contract document related details.
-   **[Usages Input](atlas.en-us.clm_developer_guide.meta/clm_developer_guide/connect_requests_usages.htm)**
    Input representation of usages.
