# Service Cloud Connector API Class Objects

These objects are used by the Connector API Methods and the Base Connector API.

-   **[ActiveCallsResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_activecallsresult.htm)**
    Represents the result when getting a list of active calls.
-   **[ACWInfo](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_acwinfo.htm)**
    After-Conversation Work information passed with work-related events.
-   **[AgentConfigResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_agentconfigresult.htm)**
    Represents the result when getting rep phone configuration information.
-   **[AgentStatusInfo](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_agentstatusinfo.htm)**
    Represents the rep status.
-   **[AgentVendorStatusInfo](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_agentvendorstatusinfo.htm)**
    Represents the rep’s vendor-side status information.
-   **[AgentWork](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_agentwork.htm)**
    Represents information about the AgentWork record.
-   **[AudioDevicesResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_audiodevicesresult.htm)**
    Contains the list of audio devices (speaker and microphone) that are available to the browser.
-   **[AudioStats](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_audiostatsgroup.htm)**
    Represents audio stats, which contains an array of AudioStatsElement objects.
-   **[AudioStatsElement](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_audiostats.htm)**
    Represents audio stats element, which can include audio input channel stats and audio output channel stats.
-   **[CallInfo](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_callinfo.htm)**
    Represents information about a call such as whether it is a softphone call, call control buttons are enabled and so on, using key value pairs in a JSON object. The field values can be sent at the start of the call or during an ongoing call.
-   **[CallResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_callresult.htm)**
    Represents the result type for call-related functions.
-   **[CapabilitiesResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_capabilitiesresult.htm)**
    Represents the result type for the rep capabilities.
-   **[ConferenceResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_conferenceresult.htm)**
    Represents the result for the conference() function.
-   **[Contact](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_contact.htm)**
    Represents information about a contact.
-   **[ContactsFilter](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_contactsfilter.htm)**
    Criteria object used to filter the result set returned by getContacts.
-   **[ContactsResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_contactsresult.htm)**
    Represents the result when getting a list of Messaging or phone contacts.
-   **[customError](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_customerror.htm)**
    Represents a custom error.
-   **[GenericResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_genericresult.htm)**
    Represents a generic result type for a Connector API function.
-   **[HangupResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_hangupresult.htm)**
    Represents a result type for an ended call.
-   **[HidDevice](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_hiddevice.htm)**
    Represents a Human Interface Device (HID) such as a headset or speaker.
-   **[HoldToggleResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_holdtoggleresult.htm)**
    Represents the result when toggling the hold state for a call.
-   **[InitResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_initresult.htm)**
    Represents the result type for the init call.
-   **[LogoutResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_logoutresult.htm)**
    Represents a logout result type.
-   **[MuteToggleResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_mutetoggleresult.htm)**
    Represents the result when toggling the mute state for a call.
-   **[ParticipantResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_participantresult.htm)**
    Represents the result for the addParticipant() function.
-   **[Phone](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_phone.htm)**
    Represents information about a phone.
-   **[PhoneCall](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_phonecall.htm)**
    Represents information about a phone call.
-   **[PhoneCallAttributes](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_phonecallattributes.htm)**
    Represents the attributes for the phone call using key value pairs in a JSON object.
-   **[PhoneContactsResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_phonecontactsresult.htm)**
    Represents the result when getting a list of phone contacts.
-   **[RecordingToggleResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_recordingtoggleresult.htm)**
    Represents the result when toggling the recording for a call.
-   **[SetAgentConfigResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_setagentconfigresult.htm)**
    Represents the result for the setAgentConfig() function.
-   **[SetAgentStateResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_setagentstateresult.htm)**
    Return value for setAgentStatus; indicates whether the status update succeeded and whether a follow-up sync is required. Used with Amazon Connect.
-   **[SharedCapabilitiesResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_sharedcapabilitiesresult.htm)**
    Represents the result type for the rep capabilities that are shared across partner telephony and Messaging.
-   **[ShowStorageAccessResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_showstorageaccessresult.htm)**
    Result returned by publishEvent when the event type is SHOW\_STORAGE\_ACCESS.
-   **[SignedRecordingUrlResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_signedrecordingurlresult.htm)**
    Represents the result when getting the signed recording URL.
-   **[StateChangeResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_statechangeresult.htm)**
    For Amazon Connect, represents a change in the rep’s vendor-side state.
-   **[StatsInfo](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_statsinfo.htm)**
    Representing stats for either an audio input channel or an audio output channel. This object is used to calculate the MOS (Mean Opinion Score).
-   **[VoiceCapabilitiesResult](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_objects_voicecapabilitiesresult.htm)**
    Represents the result type for the rep capabilities for Voice.
