# Service Cloud Connector API Methods

The Connector API methods are what you implement in order to be notified when messaging or telephony events occur within the Salesforce org.

![Important](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note_important.png&folder=service_connector_api_developer_guide)

#### Important

The Connector API is for partners who are implementing Bring Your Own Channel for CCaaS or Service Cloud Voice with Partner Telephony.

-   **[Common Connector API Methods](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_api_methods_common.htm)**
    The common Connector API methods apply to all third party vendor systems, including Contact Center as a Service (CCaaS) and voice systems.
-   **[Voice Connector API Methods](atlas.en-us.service_connector_api_developer_guide.meta/service_connector_api_developer_guide/service_connector_api_methods_voice.htm)**
    The Voice Connector API methods apply to telephony or CCaaS vendor systems.
