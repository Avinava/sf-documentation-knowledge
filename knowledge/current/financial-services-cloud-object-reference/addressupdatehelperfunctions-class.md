# AddressUpdateHelperFunctions Class

The callable AddressUpdateHelperFunctions class provides data conversion functions for use in the Change Address OmniScript.

## Namespace

[fscserviceprocesses](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fscserviceprocesses.htm "The fscserviceprocesses namespace provides classes used in Service Process OmniScripts.")

## Usage

The AddressUpdateHelperFunctions class is a helper function for the GET and POST methods of the SvcCatalogRequest Connect API. The Change Address OmniScript calls Integration Procedures that call the AddressUpdateHelperFunctions methods via Remote Actions.
