# Request Bodies

Here’s a list of Financial Services Cloud Business API request bodies.

-   **[Address Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_address_input.htm)**
    Represents address details.
-   **[Asset Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_mortgage_asset_input.htm)**
    Represents details about the borrower's and co-borrower's current personal and real property.
-   **[Borrower Address Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_borrower_address_input.htm)**
    Input representation for Borrower addresses.
-   **[Borrower Declaration Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_borrower_declaration_input.htm)**
    Input representation for Borrower declarations
-   **[Borrower Employment Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_borrower_employment_input.htm)**
    Represents details about the borrower's or co-borrower's current and past employment.
-   **[Borrower Income Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_borrower_income_input.htm)**
    Input representation for Borrower income
-   **[Borrower Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_borrower_input.htm)**
    Represents details of the borrower or co-borrower for the loan application.
-   **[Case Info Wrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_case_info_wrappe.htm)**
    Input representation of a case related to the complaint.
-   **[Case Participant Info Wrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_case_participant_info_wrappe.htm)**
    Input representation of a case participant associated with the complaint.
-   **[Complaint Info Wrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_complaint_info_wrappe.htm)**
    Input representation of the complaint details.
-   **[Complaint Management Request Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_complaint_mgmt_request_input.htm)**
    Input representation of the request to create a public complaint.
-   **[Document Info Attributes Wrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_document_info_attributes_wrappe.htm)**
    Input representation of the details of a supporting document for the complaint.
-   **[Ethoca Alert Callback Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_ethoca_alert_callback_input.htm)**
    Ethoca alert input payload for transaction dispute management.
-   **[Ethoca Event](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_ethoca_event.htm)**
    Ethoca event payload.
-   **[Ethoca Links](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_ethoca_links.htm)**
    Ethoca resource link payload.
-   **[Extended Fields Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_extended_fields_input.htm)**
    Input representation for extended fields.
-   **[Filter Criteria](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_filter_criteria_input.htm)**
    Filter criteria for queries of financial goals and households.
-   **[Financial Goal Create Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_create_input.htm)**
    Input for creating a single financial goal with goal members and funding.
-   **[Financial Goal Funding Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_funding_input.htm)**
    Input for Financial Accounts that fund the Financial Goal.
-   **[Financial Goal Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_input.htm)**
    Input for creating or updating a financial goal.
-   **[Financial Goal List Create Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_list_create_input.htm)**
    Input for creating multiple financial goals with goal members.
-   **[Financial Goal Party Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_party_input.htm)**
    Input for adding or removing financial goal members.
-   **[Financial Goal Patch Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_patch_input.htm)**
    Input for updating a financial goal.
-   **[Financial Goal Query Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_financial_goal_query_input.htm)**
    Query input for financial goals and goal members based on a parent record ID. The parent can be a Person Account or Household.
-   **[Financial Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_mortgage_financial_input.htm)**
    Input details for the financial transaction.
-   **[Flexible Hierarchy Bulk Validation Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_flexible_hierarchy_bulk_validation_input.htm)**
    Input representation to validate flexible hierarchies in bulk asynchronously.
-   **[Flexible Hierarchy Bulk Validation Filter Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_flexible_hierarchy_bulk_validation_filter_input.htm)**
    Input representation for a filter criterion used in bulk validation.
-   **[Flexible Hierarchy Filters Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_flexible_hierarchy_filters_input.htm)**
    Input representation for filter conditions and expression used in bulk validation.
-   **[Generic Object Wrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_generic_object_wrappe.htm)**
    Input representation of additional details of the case, case participant, or complaint.
-   **[Liability Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_mortgage_liability_input.htm)**
    Details about the borrower and co-borrower current liabilities.
-   **[Mapper Input Representation](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_entity_mapper_input.htm)**
    Represents the input representation for the Mapper POST method.
-   **[Mortgage Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_mortgage_input.htm)**
    Represents the input for mortgage POST or PATCH method.
-   **[Property Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_mortgage_property_input.htm)**
    Represents details of the property the loan application finances.
-   **[Title Holder Input](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_requests_mortgage_title_holder_input.htm)**
    Represents details of borrowers who are applying to hold title to the property the loan application finances.
