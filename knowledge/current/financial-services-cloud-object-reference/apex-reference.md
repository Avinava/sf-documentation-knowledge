# Apex Reference

This Apex reference goes into detail about the built-in Apex ConnectApi classes and interfaces for Financial Services Cloud.

For Apex-related objects and SOAP API calls, see [Objects and SOAP API Calls and Headers for Apex](https://developer.salesforce.com/docs/atlas.en-us.260.0.api.meta/api/soap-apex.htm) in the SOAP API Developer Guide.

-   **[ConnectApi Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_classes_connect_api.htm)**
    The ConnectApi namespace (also called Connect in Apex) provides classes for accessing the same data available in Connect REST API. Use Connect in Apex to create custom experiences in Salesforce.
-   **[commonserviceexcellence Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_commonserviceexcellence.htm)**
    The commonserviceexcellence namespace provides classes used in financial account FlexCards.
-   **[disputemanagement Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_disputemanagement.htm)**
    The disputemanagement namespace classes provide services for Integration Procedures that the Transaction Dispute Management OmniScript calls.
-   **[fsccashflow Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fsccashflow.htm)**
    The fsccashflow namespace provides classes used in the FSCCashFlow Flexcards and its child Flexcards.
-   **[fsccomplaintmanagement Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fsccomplaintmanagement.htm)**
    The fsccomplaintmanagement namespace classes provide data helper functions for use in complaint management.
-   **[fschousehold Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fschousehold.htm)**
    The fschousehold namespace provides classes used in the Financial Goals and Plans FlexCards.
-   **[fscserviceprocesses Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fscserviceprocesses.htm)**
    The fscserviceprocesses namespace provides classes used in Service Process OmniScripts.
-   **[fscwmgen Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fscwmgen.htm)**
    The fscwmgen namespace provides an interface used for Record Alerts.
-   **[industriespersonalizedfinance Namespace](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_industriespersonalizedfinance.htm)**
    The industriespersonalizedfinance namespace classes provide data helper functions for use in personalized finance.
