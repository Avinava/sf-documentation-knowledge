# RecordAlertInfo Properties

The following are properties for RecordAlertInfo.

-   **[actionInfoList](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_actionInfoList.htm)**
    Details of an action taken on a record alert.
-   **[categoryFullName](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_categoryFullName.htm)**
    The category name of the record alert.
-   **[description](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_description.htm)**
    The description of the record alert.
-   **[effectiveDate](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_effectiveDate.htm)**
    The start date and time from when an active record alert is in effect.
-   **[isActive](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_isActive.htm)**
    Indicates whether a record alert is active or not.
-   **[parentId](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_parentId.htm)**
    The parent ID of the record alert.
-   **[severity](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_severity.htm)**
    Indicates the degree of impact of the record alert.
-   **[sourceSystemId](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_sourceSystemId.htm)**
    The ID of the source system that this record alert was sourced from.
-   **[subject](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_subject.htm)**
    The subject line of the record alert.
-   **[validUntilDate](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_validUntilDate.htm)**
    The date and time from when a record alert is no longer valid.
-   **[whatId](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertInfo_whatId.htm)**
    The Salesforce object record ID on which a record alert is created.
