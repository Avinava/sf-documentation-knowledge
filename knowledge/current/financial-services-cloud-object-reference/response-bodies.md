# Response Bodies

Here’s a list of Financial Services Cloud Business API response bodies.

-   **[Address Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_address_output.htm)**
    Represents address details.
-   **[Asset Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_asset_output.htm)**
    Represents details about the borrower's and co-borrower's current personal and real property.
-   **[Asset Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_asset_result.htm)**
    Asset result of the POST or PATCH request.
-   **[Borrower Address Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_address_output.htm)**
    Represents Borrower addresses.
-   **[Borrower Address Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_address_result.htm)**
    Borrower address result of the POST or PATCH request.
-   **[Borrower Declaration Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_declaration_output.htm)**
    Represents financial issues like foreclosure, lawsuits, or defaults of the applicant or co-applicant.
-   **[Borrower Declaration Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_declaration_result.htm)**
    Result containing details about borrower’s declaration.
-   **[Borrower Employment Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_employment_output.htm)**
    Represents details about the borrower's or co-borrower's current and past employment.
-   **[Borrower employment Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_employment_result.htm)**
    Borrower employment result of the POST or PATCH request.
-   **[Borrower Income Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_income_output.htm)**
    Represents details about the borrower and co-borrower current monthly income.
-   **[Borrower Income Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_income_result.htm)**
    Borrower income result of the POST or PATCH request.
-   **[Borrower Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_output.htm)**
    Represents details of the borrower or co-borrower for the loan application.
-   **[Borrower Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_borrower_result.htm)**
    Borrower result of the POST or PATCH request.
-   **[Complaint Management Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_complaint_mgmt_output.htm)**
    Output representation of the created public complaint.
-   **[Ethoca Alert Callback Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_ethoca_alert_callback_output.htm)**
    Ethoca alert callback output for dispute management.
-   **[Extended Fields Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_extended_fields_output.htm)**
    Represents custom fields names and values.
-   **[Financial Goal Details Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goal_details_output.htm)**
    Queried financial goal details.
-   **[Financial Goal Funding Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goal_funding_output.htm)**
    Output for Financial Goal funding.
-   **[Financial Goal Party Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goal_party_output.htm)**
    Output for Financial Goal Parties.
-   **[Financial Goal Patch Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goal_patch_output.htm)**
    Updated financial goal.
-   **[Financial Goals Create Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goals_create_output.htm)**
    Container for created financial goals and goal members.
-   **[Financial Goals Create Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goals_create_result.htm)**
    Created financial goals and goal members.
-   **[Financial Goals List Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_financial_goals_list_output.htm)**
    Queried financial goals.
-   **[Financial Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_financial_output.htm)**
    Output details for the financial transaction.
-   **[Financial Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_financial_result.htm)**
    Financial result of the POST or PATCH request.
-   **[Flexible Hierarchy Bulk Validation Details](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_flexible_hierarchy_bulk_validation.htm)**
    Output representation to display the status of bulk validation request.
-   **[Flexible Hierarchy Status](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_flexible_hierarchy_status.htm)**
    Output representation to display the status of the Flexible Hierarchy API request.
-   **[Liability Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_liability_output.htm)**
    Details about the borrower and co-borrower current liabilities.
-   **[Liability Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_liability_result.htm)**
    Liability result of the POST or PATCH request.
-   **[Mapper Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_entity_mapper_result.htm)**
    Represents the Mapper result.
-   **[Mapper Result Representation](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_entity_mapper_result_representation.htm)**
    Represents the result representation of the POST method.
-   **[Mortgage Information Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_get_mortgage_result.htm)**
    Represents the output of the GET method.
-   **[Mortgage Information Response](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_get_mortgage_response.htm)**
    Response for the GET method.
-   **[Mortgage Response](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_post_mortgage_response.htm)**
    Response for the POST or PATCH method.
-   **[Mortgage Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_post_mortgage_result.htm)**
    Represents the result of the POST or PATCH method.
-   **[Property Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_property_result.htm)**
    Property result of the POST or PATCH request.
-   **[Property Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_property_output.htm)**
    Represents details of the property the loan application finances.
-   **[Record Alert](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_record_alert.htm)**
    Output representation of the details of a record alert.
-   **[Record Alert Collection](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_record_alert_collection.htm)**
    Output representation of a collection of record alerts.
-   **[Record Alert Collection Map](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_record_alert_collection_map.htm)**
    Output representation of the bulk record alerts request.
-   **[Record Alert Action Info](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_record_alert_action_info.htm)**
    Output representation of the information of the record alert action.
-   **[Record Alert Action Info Collection](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_record_alert_action_info_collection.htm)**
    Output representation of the details of an action on the record alert.
-   **[Title Holder Output](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_title_holder_output.htm)**
    Represents details of borrowers who are applying to hold title to the property the loan application finances.
-   **[Title Holder Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_mortgage_title_holder_result.htm)**
    Title holder result of the POST or PATCH request.
-   **[Retrieval Summary Result](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_retrieval_summary_result.htm)**
    Output representation of a retrieval summary definition call.
-   **[Retrieval Summary Result Node](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_responses_retrieval_summary_result_node.htm)**
