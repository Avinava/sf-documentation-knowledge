# Discovery Framework Tooling API Objects

Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata. For more information about Tooling API objects and to find a complete reference of all the supported objects, see **[Introducing Tooling API](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_tooling.meta/api_tooling/intro_api_tooling.htm)**.

-   **[AssessmentQuestionConfig](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/tooling_api_objects_assessmentquestionconfig.htm)**
    Represents the assessment question record metadata configuration. This object is available in API version 56.0 and later.
-   **[AssessmentQuestionSetConfig](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/tooling_api_objects_assessmentquestionsetconfig.htm)**
    Reqpresents the assessment question set record metadata configuration. This object is available in API version 56.0 and later.
