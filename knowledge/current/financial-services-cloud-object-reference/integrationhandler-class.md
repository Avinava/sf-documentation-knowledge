# IntegrationHandler Class

Use the callable IntegrationHandler class to fetch financial transaction data.

## Namespace

[commonserviceexcellence](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_commonserviceexcellence.htm "The commonserviceexcellence namespace provides classes used in financial account FlexCards.")

## Usage

The ServiceExcellenceAlertCard and ServiceExcellenceGenericAlertCard have Integration Procedure data sources that call the IntegrationHandler class. These FlexCards display information about financial accounts.

All of the IntegrationHandler methods reference an Integration Definition configured in Setup. Some methods reference the **Fetch Real Time Financial Account Info** preference in Setup.
