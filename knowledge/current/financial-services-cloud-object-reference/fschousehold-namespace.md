# fschousehold Namespace

The fschousehold namespace provides classes used in the Financial Goals and Plans FlexCards.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=financial_services_cloud_object_reference)

#### Note

The fschousehold namespace cannot be used in DML operations.

The following are the classes in the fschousehold namespace.

-   **[FSCFinancialAccountService Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_fschousehold_FSCFinancialAccountService.htm)**
    Use the callable FSCFinancialAccountService class to return true if the FinancialAccountMgmtPref permission is enabled in the org.
-   **[FSCGoalService Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_fschousehold_FSCGoalService.htm)**
    Use the callable FSCGoalService class to return either picklist values of a Financial Goal object or true if the user has Create, Edit, or Delete access to the FinancialGoal object.
-   **[FSCHouseholdService Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_fschousehold_FSCHouseholdService.htm)**
    Use the callable FSCHouseholdService class to accept a Person Account ID corresponding to a member of a Household and return the other Household members.
-   **[FSCPlansAndGoalsUtil Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_fschousehold_FSCPlansAndGoalsUtil.htm)**
    The FSCPlansAndGoalsUtil class provides utilities to the other Apex classes in the fschousehold namespace.
-   **[FSCPlanService Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_fschousehold_FSCPlanService.htm)**
    Use the callable FSCPlanService class to return picklist values of a Financial Plan object.
-   **[RetrievalSummaryDataRefresh Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_fschousehold_RetrievalSummaryDataRefresh.htm)**
    Use the callable RetrievalSummaryDataRefresh wrapper class to refresh retrieval summary definition associated rollups.
