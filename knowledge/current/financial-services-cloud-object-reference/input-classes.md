# Input Classes

These are Apex input classes in Financial Services Cloud.

-   **[ConnectApi.CaseInfoWrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_case_info_wrappe.htm)**
    Input representation of a case related to the complaint.
-   **[ConnectApi.CaseParticipantInfoWrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_case_participant_info_wrappe.htm)**
    Input representation of a case participant associated with the complaint.
-   **[ConnectApi.ComplaintInfoWrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_complaint_info_wrappe.htm)**
    Input representation of the complaint details.
-   **[ConnectApi.ComplaintMgmtRequestInputRepresentation](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_complaint_mgmt.htm)**
    Input representation of the request to create a public complaint.
-   **[ConnectApi.DocumentInfoAttributesWrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_document_info_attributes_wrappe.htm)**
    Input representation of the details of a supporting document for the complaint.
-   **[ConnectApi.EthocaAlertCallbackInputRepresentation](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_ethoca_alert_callback.htm)**
    Ethoca alert input class for transaction dispute management.
-   **[ConnectApi.EthocaEvent](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_ethoca_event.htm)**
    Ethoca event input class.
-   **[ConnectApi.EthocaLinks](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_ethoca_links.htm)**
    Ethoca resource link input class.
-   **[ConnectApi.GenericObjectWrapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_connectapi_input_generic_object_wrappe.htm)**
    Input representation of additional details of the case, case participant, or complaint.
