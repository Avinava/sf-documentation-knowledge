# Resources

Here’s a list of Financial Services Cloud Business API resources.

-   **[Application Mapper](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_post_mapper.htm)**
    Create financial accounts and related records from a residential loan application.
-   **[Bulk Record Alerts (Financial Accounts)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_get_financial_accounts_bulk_alerts.htm)**
    Retrieve bulk record alerts. Use to fetch bulk record alerts associated with financial accounts.
-   **[Bulk Record Alerts (Person Accounts)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_get_person_accounts_bulk_alerts.htm)**
    Retrieve bulk record alerts. Use to fetch bulk record alerts associated with person accounts.
-   **[Dispute Management Ethoca Alert Callback](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_dispute_mgmt_ethoca_alert.htm)**
    Send an Ethoca alert callback for transaction dispute management.
-   **[Financial Goals](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_financial_goals_create.htm)**
    Create financial goals, goal members, and goal funding.
-   **[Financial Goal Update](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_financial_goal_patch.htm)**
    Update a financial goal and its members and funding.
-   **[Financial Goals Query](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_financial_goal_query.htm)**
    Query financial goals, goal members, and goal funding using the parent Person Account, Household, or Party Relationship Group record.
-   **[Flexible Hierarchy Bulk Validation (POST)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_flexible_hierarchy_bulk_validate.htm)**
    Validate multiple flexible hierarchies asynchronously by providing a list of IDs, applying filter criteria, or selecting all draft hierarchies.
-   **[Mortgage Application](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_get_patch_mortgage.htm)**
    Use a single API call to get and update mortgage application information like borrowers, declarations, incomes, addresses, financial transactions, liabilities, assets, properties, and title holders. You can use both the loanApplicationId and the externalID of an external system to get a mortgage application.
-   **[Mortgage Applications](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_post_mortgage.htm)**
    Create mortgage applications using the POST method.
-   **[Public Complaint](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_post_public_complaints.htm)**
    Create a customer complaint.
-   **[Record Alerts (Financial Accounts)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_get_financial_account_alerts.htm)**
    Retrieve a list of active record alerts on financial accounts.
-   **[Record Alerts (Person Accounts)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_get_person_account_alerts.htm)**
    Retrieve a list of active record alerts on person accounts.
-   **[Retrieval Summary](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/connect_resources_retrieval_summary.htm)**
    Get the data of a retrieval summary definition by using the root record ID and the retrieval summary defTinition ID.
