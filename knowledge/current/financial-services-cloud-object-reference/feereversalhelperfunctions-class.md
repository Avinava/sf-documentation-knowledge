# FeeReversalHelperFunctions Class

The callable FeeReversalHelperFunctions class provides data conversion functions for use in the Fee Reversal OmniScript. This class accepts a list of transactions from the related Fee Reversal Case Object and the corresponding Fee Reversal amounts as inputs and calls the Fee Reversal Mule API endpoint.

## Namespace

[fscserviceprocesses](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_namespace_fscserviceprocesses.htm "The fscserviceprocesses namespace provides classes used in Service Process OmniScripts.")

## Usage

The FeeReversalHelperFunctions class is a helper function for the GET and POST methods of the SvcCatalogRequest Connect API. The Fee Reversal OmniScript calls Integration Procedures that call the FeeReversalHelperFunctions methods via Remote Actions.
