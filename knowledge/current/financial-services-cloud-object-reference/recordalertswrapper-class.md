# RecordAlertsWrapper Class

Use the callable RecordAlertsWrapper class to fetch Record Alert data. For Financial Services Cloud, you can group the results by Financial Account and Record Alert Category.

## Usage

The ServiceExcellenceAlertCard and ServiceExcellenceGenericAlertCard have Integration Procedure data sources that call the RecordAlertsWrapper class. These FlexCards display information about financial accounts.
