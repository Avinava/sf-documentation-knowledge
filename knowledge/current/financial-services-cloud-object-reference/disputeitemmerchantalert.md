# DisputeItemMerchantAlert

Represents the alert sent by the card issuer to the merchant. The alert occurs through a chargeback prevention vendor like Ethoca Merchant Alerts™ for a transaction that’s being disputed. This object is available in API version 60.0 and later.

## Supported Calls

create(), delete(), describeLayout(), describeSObjects(), getDeleted(), getUpdated(), query(), retrieve(), search(), undelete(), update(), upsert()

## Special Access Rules

These fields are available if the add-on license for Financial Services Cloud is enabled.
