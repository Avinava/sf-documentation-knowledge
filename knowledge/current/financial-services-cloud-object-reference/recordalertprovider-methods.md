# RecordAlertProvider Methods

The following are methods for RecordAlertProvider.

-   **[dismissAlert(var1)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertProvider_dismissAlert.htm)**
    Use this method to dismiss the record alert.
-   **[getAlertsByParentId(var1)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertProvider_getAlertsByParentId.htm)**
    Retrieve record alerts by the parent ID.
-   **[getAlertsByWhatId(var1)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertProvider_getAlertsByWhatId.htm)**
    Retrieve record alerts by the Salesforce object record ID.
-   **[getAlertsByWhatIdAndParentId(var1, var2)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertProvider_getAlertsByWhatIdAndParentId.htm)**
    Retrieve record alerts by both the Salesforce record ID and the related parent ID.
-   **[snoozeAlert(var1, var2)](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_fscwmgen_RecordAlertProvider_snoozeAlert.htm)**
    Use this method to snooze the record alert.
