# commonserviceexcellence Namespace

The commonserviceexcellence namespace provides classes used in financial account FlexCards.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=financial_services_cloud_object_reference)

#### Note

The commonserviceexcellence namespace cannot be used in DML operations.

The following are the classes in the commonserviceexcellence namespace.

-   **[IntegrationHandler Class](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/apex_class_commonserviceexcellence_IntegrationHandler.htm)**
    Use the callable IntegrationHandler class to fetch financial transaction data.
