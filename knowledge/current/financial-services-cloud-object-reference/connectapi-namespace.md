# ConnectApi Namespace

The ConnectApi namespace (also called Connect in Apex) provides classes for accessing the same data available in Connect REST API. Use Connect in Apex to create custom experiences in Salesforce.

For information about working with the ConnectApi classes, see [Connect in Apex](https://developer.salesforce.com/docs/atlas.en-us.260.0.apexcode.meta/apexcode/connectAPI_overview.htm).

-   **[Dispute Management Methods](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/dispute_mgmt_apex_methods.htm)**
    These are ﻿IDisputeManagementConnectFamily methods for Financial Services Cloud.
-   **[Public Complaint Methods](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/public_complaint_apex_methods.htm)**
    These are createComplaint(complaintMgmtRequestData) methods for Financial Services Cloud.
-   **[RecordAlert Methods](atlas.en-us.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/record_alerts_apex_methods.htm)**
    These are RecordAlert methods for Financial Services Cloud.

    These are Apex input classes in Financial Services Cloud.
