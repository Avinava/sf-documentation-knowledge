# Error: Bad CPU Type

You get the error Bad CPU type in executable when you try to install Salesforce CLI on macOS.

Answer: Make sure you download and install the correct .pkg file for your macOS computer. [Our download page](https://developer.salesforce.com/tools/salesforcecli) provides two macOS flavors based on the computer’s processor: Apple Silicon and Intel CPU.

To determine which type of processor your computer is using, see [this Apple Support document](https://support.apple.com/en-us/116943).
