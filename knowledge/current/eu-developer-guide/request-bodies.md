# Request Bodies

Here’s a list of E&U Cloud API request bodies.

-   **[Program Application File Input](atlas.en-us.eu_developer_guide.meta/eu_developer_guide/connect_requests_program_application_file_input.htm)**
    Input representation of the files or attachment for a customer program application.
-   **[Program Application Input](atlas.en-us.eu_developer_guide.meta/eu_developer_guide/connect_requests_program_application_input.htm)**
    Input representation of an application to a program.
-   **[Program Application Item Input](atlas.en-us.eu_developer_guide.meta/eu_developer_guide/connect_requests_program_application_item_input.htm)**
    Input representation of the item for a customer program application.
