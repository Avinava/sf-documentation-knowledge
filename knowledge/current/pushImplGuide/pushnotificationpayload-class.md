# PushNotificationPayload Class

Contains methods to create the notification message payload for an Apple device.

## Namespace

Messaging

## Usage

Apple has specific requirements for the notification payload. and this class has helper methods to create the payload. For more information on Apple’s payload parameters, search for “Apple Push Notification Service” at [https://developer.apple.com/library/mac/documentation/](https://developer.apple.com/library/mac/documentation/ "HTML (New Window)").
