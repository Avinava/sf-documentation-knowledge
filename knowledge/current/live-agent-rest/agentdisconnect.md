# AgentDisconnect

Indicates that the agent has been disconnected from the chat.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=live_agent_rest)

#### Note

Though the agent has been disconnected from the chat, the chat session is still active on the server. A new agent may accept the chat request and continue the chat.
