# Usage Management Fields on Product2

Standard and custom fields extend the standard Product2 object for use in Usage Management to represent information about products.

## Fields

| Field | Details |
| --- | --- |
| UsageModelType | TypepicklistPropertiesCreate, Filter, Group, Nillable, Restricted picklist, Sort, UpdateDescriptionThe type of usage model for a product or service.Valid values are:Anchor—The main subscription product or service. Available in API version 62.0 and later.Pack—The add-on product or service that grants additional usage resources for consumption.
