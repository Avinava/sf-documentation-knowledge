# Request Bodies

# Request Bodies

Learn more about the available Product Discovery API request bodies.

-   **[Additional Fields Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_additional_fields_input.htm)**  
    Input representation of the additional standard or custom fields to include in the response.
-   **[Bulk Product Details Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_bulk_product_details_input.htm)**  
    Input representation of the request to retrieve details of multiple products.
-   **[Catalog Details Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_catalog_details_input.htm)**  
    Input representation of the request to get the catalog details.
-   **[Catalog List Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_catalog_list_input.htm)**  
    Input representation of the request to get a list of catalogs.
-   **[Category Details Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_category_details_input.htm)**  
    Input representation of the request to get category details.
-   **[Category List Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_category_list_input.htm)**  
    Input representation of the request to get a list of categories.
-   **[Context Data Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_context_data_input.htm)**  
    Input representation of the context data.
-   **[Guided Selection Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_guided_selection_input.htm)**  
    Input representation of the guided selection details.
-   **[Guided Selection Search Term Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_guided_selection_search_term_input.htm)**  
    Input representation of the search terms of a guided selection.
-   **[Filter Criteria Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_filter_criteria_input.htm)**  
    Input representation of the criteria to filter records based on supported properties.
-   **[Filter Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_filter_input.htm)**  
    Input representation of the request to filter records.
-   **[Product Details Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_product_details_input.htm)**  
    Input representation of the request to get product details.
-   **[Product Data Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_product_data_input.htm)**  
    Input representation of the product details such as the product ID and product selling model ID.
-   **[Product List Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_product_list_input.htm)**  
    Input representation of the request to retrieve a list of products.
-   **[Products Search Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_search_products_input.htm)**  
    Input representation of the request to search products.
-   **[QOC Qualification](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_qoc_qualification_input.htm)**  
    Input representation of the qualification request.
-   **[Related Object Filter Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_related_object_filter_input.htm)**  
    Input representation of the request to filter records of a related object.
-   **[User Context Input](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_requests_user_context_input.htm)**  
    Input representation of the details with the user context.
