# Resources

# Resources

Learn more about the available Product Discovery API resources.

-   **[Catalog Details (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_catalog_details.htm)**  
    Get catalog details for a specified catalog ID. This API is a composite API for Product Discovery.
-   **[Catalog List (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_catalog_list.htm)**  
    Get a paginated list of catalogs. This API is a composite API for Product Discovery.
-   **[Categories List (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_categories_list.htm)**  
    Get a list of categories and subcategories of a specified catalog. This API is a composite API for Product Discovery.
-   **[Category Details (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_category_details.htm)**  
    Get details of a category for a specified category ID. This API is a composite API for Product Discovery.
-   **[Bulk Product Details (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_bulk_product_details.htm)**  
    Retrieve details for multiple products. This API is a composite API for Product Discovery.
-   **[Global Search (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_global_search.htm)**  
    Retrieves a list of products based on a search query or search term. This API is a composite API for Product Discovery.
-   **[Guided Selection (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_guided_product_selection.htm)**  
    Retrieve a list of products based on the response identifier or search terms of a guided selection. Guided selection captures user requirements to show suitable products.
-   **[Product Details (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_product_details.htm)**  
    Get product details, such as attributes, hierarchy, or cardinality, for a specified product ID. This API is a composite API for Product Discovery.
-   **[Products List (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_products_list.htm)**  
    Get a list of products for a specified catalog, category, or subcategory. This API is a composite API for Product Discovery.
-   **[Qualification (POST)](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_resources_cpq_qualification.htm)**  
    Run the qualification procedure on a list of product IDs. This API is a composite API for Product Discovery.
