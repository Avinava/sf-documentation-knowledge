# Response Bodies

# Response Bodies

Learn more about the available Product Discovery API response bodies.

-   **[API Status](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_api_status_output.htm)**  
    Output representation of the API status.
-   **[Bulk Product Details](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_cpq_base_bulk_details_output.htm)**  
    Output representation of the response that contains the details of multiple products.
-   **[CPQ Base Details](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_cpq_base_details_output.htm)**  
    Output representation of the catalog, category, or product details based on the request.
-   **[CPQ Base List](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_cpq_base_list_output.htm)**  
    Output representation of the list of catalogs, categories, or products based on the request.
-   **[CPQ Message](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_cpq_message_output.htm)**  
    Output representation of the API messages.
-   **[Facet Value](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_facet_value.htm)**  
    Output representation of the facet values found in the search result.
-   **[Guided Selection](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_guided_selection_output.htm)**  
    Output representation of the details of a guided selection.
-   **[Guided Selection Search Term](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_guided_selection_search_term_output.htm)**  
    Output representation of the search term details for a guided selection.
-   **[Search Products Facet](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_search_products_facet.htm)**  
    Output representation of the details of the faceted search.
-   **[User Context](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/connect_responses_user_context_output.htm)**  
    Output representation of the user context details.
