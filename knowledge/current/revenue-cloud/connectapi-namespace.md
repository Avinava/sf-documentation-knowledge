# ConnectApi Namespace

The ConnectApi namespace (also called Connect in Apex) provides classes to manage credit applications and billing scenarios.

These classes are available in the ConnectApi namespace.

-   **[BatchInvoiceApplication Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_BatchInvoiceApplication_static_methods.htm#apex_ConnectAPI_BatchInvoiceApplication_static_methods)**
    Update a batch of invoices from Draft to Posted status for a credit memo application. by using the BatchInvoiceApplication class.
-   **[Billing Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_Billing_static_methods.htm#apex_ConnectAPI_Billing_static_methods)**
    Manage billing scenarios by using the Billing class. You can convert negative invoice lines, create and apply a credit memo to an invoice, generate invoices, and recover billing schedules.
-   **[CreditMemoApply Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_CreditMemoApply_static_methods.htm#apex_ConnectAPI_CreditMemoApply_static_methods)**
    Manage credit memo applications by using the CreditMemoApply class.
-   **[CreditMemoLineApply Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_CreditMemoLineApply_static_methods.htm#apex_ConnectAPI_CreditMemoLineApply_static_methods)**
    Manage credit memo line applications by using the CreditMemoLineApply class.
-   **[HarmonizeBilling Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_HarmonizeBilling_static_methods.htm#apex_ConnectAPI_HarmonizeBilling_static_methods)**
    Update the status of the invoice from Draft to Posted by using the HarmonizeBilling class.
-   **[PaymentsBilling Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_PaymentsBilling_static_methods.htm#apex_ConnectAPI_PaymentsBilling_static_methods)**
    Use the PaymentsBilling class to allocate the balance of a payment to reduce the balance of an invoice. Additionally, revert the application of a payment line from an invoice.
-   **[SequencingWithoutAura Class](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/apex_ConnectAPI_SequencingWithoutAura_static_methods.htm#apex_ConnectAPI_SequencingWithoutAura_static_methods)**
    Manage invoice sequencing processes by using the SequencingWithoutAura class.
-   **[ConnectApi Input Classes](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/billing_apex_input_classes.htm)**
    Billing includes these Apex input classes.
-   **[ConnectApi Output Classes](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/billing_apex_output_classes.htm)**
    Billing includes these Apex output classes.
-   **[ConnectApi Enums](atlas.en-us.revenue_lifecycle_management_dev_guide.meta/revenue_lifecycle_management_dev_guide/billing_connectAPI_enums.htm)**
    Enums specific to the ConnectApi namespace.
