# Loops

Apex supports five types of procedural loops.

These types of procedural loops are supported:

-   [do {statement} while (Boolean\_condition);](atlas.en-us.apexcode.meta/apexcode/langCon_apex_loops_do_while.htm)
-   [while (Boolean\_condition) statement;](atlas.en-us.apexcode.meta/apexcode/langCon_apex_loops_while.htm)
-   [for (initialization; Boolean\_exit\_condition; increment) statement;](atlas.en-us.apexcode.meta/apexcode/langCon_apex_loops_for.htm)
-   [for (variable : array\_or\_set) statement;](atlas.en-us.apexcode.meta/apexcode/langCon_apex_loops_for_lists.htm)
-   [for (variable : \[inline\_soql\_query\]) statement;](atlas.en-us.apexcode.meta/apexcode/langCon_apex_loops_for_SOQL.htm "SOQL for loops iterate over all of the sObject records returned by a SOQL query.")

All loops allow for loop control structures:

-   break; exits the entire loop
-   continue; skips to the next iteration of the loop
