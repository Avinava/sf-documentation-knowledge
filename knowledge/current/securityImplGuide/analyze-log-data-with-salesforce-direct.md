# Analyze Log Data with Salesforce Direct

Transform your log data into clear, insightful visualizations. Explore Event Log Objects in CRM Analytics with Salesforce Direct using a variety of engaging chart types.

![Important](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note_important.png&folder=securityImplGuide)

#### Important

This feature is only available to Hyperforce customers. Log data is retained for up to 30 days.

| Available in: Salesforce Classic (not available in all orgs), and Lightning Experience |
| --- |
| Available in: Enterprise, Performance, and Unlimited EditionsRequires Salesforce Shield or Salesforce Event Monitoring add-on subscriptions. |

| User Permissions Needed |
| --- |
| To query and view event log object data: | View Event Log Object Data |

1.  From Setup, in the Quick Find box, enter Analytics Studio, and select **Analytics Studio**.
2.  To create a dashboard, click **Create** and then, from the dropdown list, select **Dashboard**.
3.  Drag the **Chart** option to your dashboard.
4.  To select a data source, click the new chart.
5.  In the data source window, go to the **Salesforce Object** tab.
6.  To see all event log objects, search for event log.
7.  Select the event log object data you want to visualize. See [Visualize Data With Charts](https://help.salesforce.com/s/articleView?id=sf.bi_visualize.htm&type=5&language=en_US).
8.  You can visualize your data in a variety of chart types using Salesforce Direct. Limit the number of panels on your dashboard to avoid any timeouts. See [Salesforce Direct Data Queries](https://help.salesforce.com/s/articleView?id=sf.bi_dashboard_step_sf_direct_limitations.htm&type=5&language=en_US) for more information.
