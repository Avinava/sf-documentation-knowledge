# Manage User Passwords

Salesforce provides each of your users with a unique username and password that they enter at each login. As an admin, you can configure several settings to ensure that your users' passwords are strong and secure.

To learn more about managing user passwords, see these topics in Salesforce Help.

-   [Set Password Policies](https://help.salesforce.com/articleView?id=xcloud.admin_password.htm&type=5&language=en_US "HTML (New Window)")
-   [Reset Passwords for Your Users](https://help.salesforce.com/articleView?id=xcloud.resetting_and_expiring_passwords.htm&type=5&language=en_US "HTML (New Window)")
-   [Expire Passwords for All Users](https://help.salesforce.com/articleView?id=xcloud.expiring_passwords.htm&type=5&language=en_US "HTML (New Window)")
