# B2B and D2C Commerce Actions

Manage B2B Commerce integrations and store checkout flow. Record and reverse tax transactions in external systems like Stripe based on order changes, and create subscription records for orders with subscription products. For more information on standard invocable actions, see **[REST API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_rest.meta/api_rest/resources_actions_invocable_standard.htm)** and **[Actions Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_action.meta/api_action/actions_intro.htm)**.
