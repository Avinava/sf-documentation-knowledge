# PromotionLimitUsage

Represents information about the total usage of a promotion limit. This object is available in API version 60.0 and later.

## Supported Calls

create(), delete(), describeLayout(), describeSObjects(), getDeleted(), getUpdated(), query(), retrieve(), search(), undelete(), update(), upsert()

## Special Access Rules

PromotionLimitUsage is only available in orgs where Global Promotions Management is enabled.
