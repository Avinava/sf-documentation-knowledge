# Resources

Here’s a list of Loyalty Management API resources.

-   **[Club Member Enrollment](atlas.en-us.loyalty.meta/loyalty/connect_resources_club_member_enrollment.htm)**
    Enroll members in a club and manage their details, including membership numbers, renewal periods, and end dates.
-   **[Club Member Profile](atlas.en-us.loyalty.meta/loyalty/connect_resources_club_member_profile.htm)**
    Get the profile details of a club member such as membership number, contact ID, and loyalty program name.
-   **[Club Membership Renewal](atlas.en-us.loyalty.meta/loyalty/connect_resources_club_membership_renewal.htm)**
    Renew the membership of club members and manage their membership details such as membership numbers, renewal periods, and end dates.
-   **[Corporate Member Enrollments](atlas.en-us.loyalty.meta/loyalty/connect_resources_enroll_corporate_member.htm)**
    Enroll corporate members in a loyalty program. Create a business account and a Corporate type loyalty program member record for the new member.
-   **[Individual Member Enrollments](atlas.en-us.loyalty.meta/loyalty/connect_resources_enroll_individual_member.htm)**
    Enroll individual members in a loyalty program. Create a person account and an Individual type loyalty program member record for the new member.
-   **[Link Program Member Partner](atlas.en-us.loyalty.meta/loyalty/connect_resources_link_program_member_partner.htm)**
    Link a loyalty program member with a specified partner program. Enable integration between loyalty members and partner ecosystems, allowing members to earn, redeem, and manage benefits across connected programs. This API simplifies account linkage, enhances transparency, and supports real-time synchronization between loyalty and partner systems.
-   **[Member Benefits](atlas.en-us.loyalty.meta/loyalty/connect_resources_member_benefits.htm)**
    Get benefits for a loyalty program member.
-   **[Member Profile](atlas.en-us.loyalty.meta/loyalty/connect_resources_member_profile.htm)**
    Get the details of a loyalty program member.
-   **[Member Engagement Trail](atlas.en-us.loyalty.meta/loyalty/connect_resources_member_engagement_trail.htm)**
    Fetches current and target values of the milestones defined in an engagement trail and the rewards received after achieving milestones.
-   **[Member Vouchers](atlas.en-us.loyalty.meta/loyalty/connect_resources_member_vouchers.htm)**
    Get vouchers that have been issued to a loyalty program member.
-   **[Redeem Voucher](atlas.en-us.loyalty.meta/loyalty/connect_resources_redeem_voucher.htm)**
    Redeem a voucher for a loyalty program member. The status of the voucher gets updated to redeemed and the use date gets updated to the current date.
-   **[Transaction History](atlas.en-us.loyalty.meta/loyalty/connect_resources_transaction_history.htm)**
    Get transaction journals of a loyalty member belonging to a loyalty program. Obtain either accrual or redemption transaction journal records.
-   **[Transaction Ledger Summary](atlas.en-us.loyalty.meta/loyalty/connect_resources_transaction_ledger.htm)**
    Retrieve an aggregate view on the transaction journal and the loyalty ledger records.
-   **[Transaction Journals Execution](atlas.en-us.loyalty.meta/loyalty/connect_resources_loyalty_program_realtime.htm)**
    Use this resource to create and process a list of transaction journals against a corresponding loyalty program process.
-   **[Transaction Journals Simulation](atlas.en-us.loyalty.meta/loyalty/connect_resources_loyalty_program_process_simulation.htm)**
    Runs a simulation to determine how transaction journals are processed against their corresponding loyalty program processes. When transaction journals meet the criteria and conditions for a program process, actions that are set up in the process are triggered for the transaction journals.
-   **[Unlink Program Member Partner](atlas.en-us.loyalty.meta/loyalty/connect_resources_unlink_program_member_partner.htm)**
    Unlinks an existing association between a loyalty program member and a partner program. The API securely removes the linkage and updates related records to ensure data consistency across loyalty and partner systems.
-   **[Loyalty Program Process](atlas.en-us.loyalty.meta/loyalty/connect_resources_process_name.htm)**
    Run a loyalty process created from a loyalty program process template.
-   **[Gamification](atlas.en-us.loyalty.meta/loyalty/loyalty_gamification_apis_reference.htm)**
    Who doesn’t like to play a game? Engage your customers more by gamifying customer activities, such as placing an order, or events, such as birthdays and anniversaries.
-   **[Promotion](atlas.en-us.loyalty.meta/loyalty/loyalty_promotion_apis_reference.htm)**
    The promotion APIs allow you to set eligibility rules and limits for a promotion, choose a promotion template, and create a promotion. You can also get the details of a specific promotion, and view a list of eligible promotions for a customer’s cart.
