# Coupon Objects in Loyalty Management

Global Promotions Management in Loyalty Management provides access to Coupon objects that you can use to create coupons and manage coupon redemption for your company's promotions. When Global Promotions Management is enabled, you don't need any additional license or user permission for these coupon objects.

For more information about the objects you can use to manage coupons for promotions:

-   [Coupon](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_coupon.htm)
-   [CouponCodeRedemption](https://developer.salesforce.com/docs/atlas.en-us.260.0.object_reference.meta/object_reference/sforce_api_objects_couponcoderedemption.htm)
