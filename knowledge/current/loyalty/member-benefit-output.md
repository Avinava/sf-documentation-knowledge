| benefitTypeId | String | The ID of the associated benefit type record. | Small, 51.0 | 51.0 |
| benefitTypeName | String | The name of the associated benefit type. | Small, 51.0 | 51.0 |
| createdRecordId | String | The ID of the record created during benefit assignment. | Small, 51.0 | 51.0 |
