# Gamification Mobile SDK API Reference for iOS

The Gamification Mobile iOS SDK interacts with the Salesforce Gamification APIs, and retrieves games assigned to a loyalty member, and the rewards won by the member after playing the game.

| Available in: all editions that have Loyalty Management enabled. |
| --- |


-   **[Assigned Games](atlas.en-us.loyalty.meta/loyalty/ios_mobile_sdk_api_geteligiblegames.htm)**
    Get a list of eligible, expired, and played games, such as Spin the Wheel, or Scratch card, for a loyalty program member, an account, or a contact.
-   **[Game reward](atlas.en-us.loyalty.meta/loyalty/ios_mobile_sdk_api_playgame.htm)**
    Get the details of the reward that’s provided to a participant after the participant plays a Spin the Wheel, or Scratch Card game.
