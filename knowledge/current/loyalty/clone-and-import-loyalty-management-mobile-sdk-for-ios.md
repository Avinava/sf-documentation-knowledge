# Clone and Import Loyalty Management Mobile SDK for iOS

Clone the Loyalty Management Mobile SDK GitHub repository for iOS to create a local copy of the repository on your development machine. The repository contains the code for Loyalty Management Mobile SDK for iOS and the MyNTORewards sample app.

| Available in: all editions that have Loyalty Management enabled |
| --- |


1.  [Clone Mobile SDK for iOS GitHub Repository](atlas.en-us.loyalty.meta/loyalty/mobile_sdk_ios_prep.htm)
    Clone the GitHub repository to copy the code base and revision history of the repository to your local computer.
2.  [Import SDK into iOS Swift Projects](atlas.en-us.loyalty.meta/loyalty/mobile_sdk_import.htm)
    After you clone the mobile SDK for iOS repository, import Loyalty Management Mobile SDK in Xcode to automatically download and manage external dependencies.
