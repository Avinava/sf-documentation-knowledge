# Gamification Mobile SDK Android Classes

The Android classes organize the Gamification Mobile SDK’s functionalities and data structures. When developing mobile apps using the SDK, these classes provide components, services, or features with which you can interact.

| Available in: all editions that have Loyalty Management enabled |
| --- |


-   **[GamificationRemoteRepository](atlas.en-us.loyalty.meta/loyalty/gamification_mobile_sdk_android_gamificationremoterepository_class.htm)**
    The GamificationRemoteRepository class invokes and manages gamification-related REST API requests. Use this class to interact with the Salesforce gamification APIs and retrieve member games, and play a game.
-   **[GameAPIClient](atlas.en-us.loyalty.meta/loyalty/gamification_mobile_sdk_android_gameapiclient_class.htm)**
    The GameAPIClient class creates a retrofit instance to invoke REST APIs.
-   **[GameAuthenticator](atlas.en-us.loyalty.meta/loyalty/gamification_mobile_sdk_android_gameauthenticator_interface.htm)**
    The GameAuthenticator interface defines the necessary methods for handling access tokens in the Salesforce API.
