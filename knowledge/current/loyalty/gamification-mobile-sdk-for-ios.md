# Gamification Mobile SDK for iOS

Clone and import the Gamification Mobile iOS SDK to integrate the gamification feature in a mobile app.

| Available in: all editions that have Loyalty Management enabled |
| --- |


-   **[Clone and Import Gamification Mobile SDK for iOS](atlas.en-us.loyalty.meta/loyalty/gamification_mobile_sdk_ios_clone_import.htm)**
    Clone the Gamification Mobile iOS SDK GitHub repository to copy the code base and revision history of the repository to your local computer. After you clone the repository, import Gamification Mobile iOS SDK in an Xcode project.
-   **[Integrate Gamification Mobile iOS SDK into Loyalty Management Mobile iOS SDK](atlas.en-us.loyalty.meta/loyalty/gamification_mobile_sdk_ios_import.htm)**
    Import the Gamification Mobile iOS SDK into the existing Loyalty Management Mobile iOS SDK by adding it as an external dependency.
-   **[Gamification Mobile SDK iOS Classes](atlas.en-us.loyalty.meta/loyalty/gamification_sdk_ios_classes_parent.htm)**
    The iOS classes organize the Gamification Mobile SDK’s functionalities and data structures. When developing mobile apps using the SDK, these classes provide components, services, or features with which you can interact.
-   **[Gamification Mobile SDK API Reference for iOS](atlas.en-us.loyalty.meta/loyalty/gamification_ios_mobile_sdk_api_reference.htm)**
    The Gamification Mobile iOS SDK interacts with the Salesforce Gamification APIs, and retrieves games assigned to a loyalty member, and the rewards won by the member after playing the game.
-   **[Game Zone UI Customization for iOS](atlas.en-us.loyalty.meta/loyalty/gamification_sdk_ios_customization.htm)**
    After you build the Gamification Mobile SDK for iOS, you can customize the game zone UI components.
