# LoyaltyVoucherConnect Methods

This is a method for LoyaltyVoucherConnect. Method is static.

-   **[redeem(programName, membershipNumber, voucherCode)](atlas.en-us.loyalty.meta/loyalty/apex_ConnectAPI_LoyaltyVoucherConnect_redeem_1.htm)**
    Redeem a voucher for a program member. The status of the voucher gets updated to redeemed and the use date gets updated to the current date.
-   **[redeem(programName, membershipNumber, voucherCode, amount)](atlas.en-us.loyalty.meta/loyalty/apex_ConnectAPI_LoyaltyVoucherConnect_redeem_2.htm)**
    Redeem partial value of a voucher for a program member. The status of the voucher gets updated to redeemed and the use date gets updated to the current date.
-   **[redeem(programName, membershipNumber, voucherCode, voucherNumber)](atlas.en-us.loyalty.meta/loyalty/apex_ConnectAPI_LoyaltyVoucherConnect_redeem_4.htm)**
    Redeem the single-use voucher for a loyalty program member. The status of the voucher gets updated to redeemed and the use date gets updated to the current date.
-   **[redeem(programName, membershipNumber, voucherCode, voucherNumber, amount)](atlas.en-us.loyalty.meta/loyalty/apex_ConnectAPI_LoyaltyVoucherConnect_redeem_3.htm)**
    Redeem partial value of a voucher for a program member. The status of the voucher gets updated to redeemed and the use date gets updated to the current date.
