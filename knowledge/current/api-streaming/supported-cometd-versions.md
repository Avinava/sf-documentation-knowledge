# Supported CometD Versions

Use CometD version 3.1.0 or later in your clients to connect to Streaming API. Earlier versions aren’t supported and could result in unexpected behavior. To prevent potential issues with old CometD versions in your clients, upgrade the CometD library to a supported version. For more information, see [https://cometd.org/](https://cometd.org/ "HTML (New Window)").
