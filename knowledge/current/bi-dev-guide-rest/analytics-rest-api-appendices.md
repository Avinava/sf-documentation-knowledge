# Analytics REST API Appendices

Use these appendices when working with enums and other shared resources.

-   **[Analytics REST API Enums](atlas.en-us.bi_dev_guide_rest.meta/bi_dev_guide_rest/bi_dev_guide_rest_enums.htm)**
    Enums specific to the /wave namespace
-   **[Analytics REST API Appendix](atlas.en-us.bi_dev_guide_rest.meta/bi_dev_guide_rest/bi_resources_appendix.htm)**
    This appendix contains the references for shared representations and enums that are used across the Analytics REST API resources. The following representation details are not REST API endpoints on their own, but describe object representations used in REST API request and response bodies.
-   **[Dashboard State Representation](atlas.en-us.bi_dev_guide_rest.meta/bi_dev_guide_rest/bi_resources_dashboard_state_rep.htm)**
    This page contains the reference for the DashboardStateRepresentation resource and all the representations that make up the dashboard state for API responses. The following representation details are not REST API endpoints on their own, but describe object representations used in REST API response bodies.
-   **[Dashboard State Input Representation](atlas.en-us.bi_dev_guide_rest.meta/bi_dev_guide_rest/bi_resources_dashboard_state_input_rep.htm)**
    This page contains the reference for the DashboardStateInputRepresentation resource and all the representations that make up the dashboard state for API POST and PATCH requests. The following representation details are not REST API endpoints on their own, but describe object representations used in REST API request bodies.
