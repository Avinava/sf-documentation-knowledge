Subscription inherits properties from the abstract [Snapshot](atlas.en-us.bi_dev_guide_rest.meta/bi_dev_guide_rest/bi_responses_snapshot.htm "An Analytics snapshot."). These base properties appear in Subscription alongside the properties defined explicitly in the following table.

| Parameter Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| source | Subscription​Source | The subscription source including type, label, and record ID. | Small, 52.0 | 52.0 |
