# Prerequisites

Complete these prerequisites before you begin the quick start.

-   Download and install the [Postman Desktop App](https://www.postman.com/downloads/ "HTML (New Window)").
-   Generate a private-public keypair (.key and .crt files) on your [Mac](https://learn.microsoft.com/en-us/viva/glint/setup/sftp-ssh-key-gen#create-an-ssh-key-pair-on-macintosh-or-linux "HTML (New Window)") or [Microsoft® Windows®](https://learn.microsoft.com/en-us/viva/glint/setup/sftp-ssh-key-gen#create-an-ssh-key-pair-on-microsoft-windows "HTML (New Window)")machine if you haven’t already done so.
-   Sign up for a [Trailhead Playground org](https://trailhead.salesforce.com/promo/orgs/analytics-de) with CRM Analytics enabled.
