# Recipe Resources

Recipes are used to prepare data, creating a dataset using transformations to manipulate the data.

Recipes prepare data for use in Analytics and the Customer Data Platform. For reference content, see the [Data Prep Recipe REST API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.salesforce_recipes_api.meta/salesforce_recipes_api/sforce_recipes_api_overview.htm).
