# Watchlist

An Analytics watchlist collection.

Properties

Watchlist inherits properties from the abstract [BaseCollection](atlas.en-us.bi_dev_guide_rest.meta/bi_dev_guide_rest/bi_responses_base_collection.htm "The base Analytics collection."). These base properties appear in Watchlist alongside the properties defined explicitly in the following table.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| watchlist​Snapshots | Watchlist​Snapshot[] | The list of watchlist snapshots available to the current user. | Small, 48.0 | 48.0 |
