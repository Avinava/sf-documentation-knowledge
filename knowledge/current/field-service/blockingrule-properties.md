# BlockingRule Properties

BlockingRule contains the following properties.

-   **[ruleName](atlas.en-us.field_service_dev.meta/field_service_dev/apex_FSL_BlockingRule_ruleName.htm)**
    The name of the rule that is blocking the slot. When there is no rule name, use the rule type.
-   **[slotsBlockedByMultipleRules](atlas.en-us.field_service_dev.meta/field_service_dev/apex_FSL_BlockingRule_slotsBlockedByMultipleRules.htm)**
    The number of slots that are blocked by a combination of this rule and others.
-   **[slotsBlockedByRule](atlas.en-us.field_service_dev.meta/field_service_dev/apex_FSL_BlockingRule_slotsBlockedByRule.htm)**
    The number of slots that are blocked only by this rule.
