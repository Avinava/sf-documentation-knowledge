# Field Service Metadata API

The following Metadata types are used in Field Service.

-   **[FieldServiceSettings](atlas.en-us.field_service_dev.meta/field_service_dev/meta_fieldservicesettings.htm)**  
    Represents an organization’s Field Service settings.
-   **[Skill](atlas.en-us.field_service_dev.meta/field_service_dev/meta_skills.htm)**  
    Represents the settings for a skill used for field service or to route chats to agents in Chat, such as the name of the skill and which agents the skills are assigned to.
-   **[TimeSheetTemplate](atlas.en-us.field_service_dev.meta/field_service_dev/meta_timesheettemplate.htm)**  
    Represents a template for creating time sheets in Field Service. This type extends the Metadata metadata type and inherits its fullName field.
