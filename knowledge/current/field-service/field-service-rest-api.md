# Field Service REST API

The following endpoints can be used to manage your field service implementation.

-   **[Field Service Flow](atlas.en-us.field_service_dev.meta/field_service_dev/fsl_dev_fieldserviceflow.htm)**
    Returns information corresponding to a field service flow. This resource is available in REST API version 42.0 and later.
-   **[Field Service Mobile Settings](atlas.en-us.field_service_dev.meta/field_service_dev/fsl_dev_fieldservicemobilesettings.htm)**
    Returns information about the Field Service mobile app settings assigned to the profile of the context user. This resource is available in REST API version 42.0 and later.
-   **[Service Report Template](atlas.en-us.field_service_dev.meta/field_service_dev/fsl_dev_servicereporttemplate.htm)**
    Returns information corresponding to one or more service report templates in field service. This resource is available in REST API version 40.0 and later.
-   **[Appointment Bundling REST APIs](atlas.en-us.field_service_dev.meta/field_service_dev/fsl_rest_sabundling.htm)**
    Use the Appointment Bundling REST APIs to create bundles, remove bundle members, start the automated process for creating bundles, unbundle one or more bundles, and update a bundle.
