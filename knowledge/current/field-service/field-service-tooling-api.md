# Field Service Tooling API

The following Tooling objects are used in Field Service.

-   **[CleanRule](atlas.en-us.field_service_dev.meta/field_service_dev/tooling_api_objects_cleanrule.htm)**  
    Represents a data integration rule that controls how a data service adds and updates data for existing records in an org.
-   **[TimeSheetTemplate](atlas.en-us.field_service_dev.meta/field_service_dev/tooling_api_objects_timesheettemplate.htm)**  
    Represents a template for creating time sheets in Field Service. Available in API version 46.0 and later.
