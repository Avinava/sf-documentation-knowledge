# Payload Signature

Payload-based signature validates the payload requests sent to your webhook system from Salesforce.

You can find the payload signature in the header of the HTTP payload received in your webhook system. For example, x-signature: uruO+yjX8KKjl72GkQbLXuoMyAilC5losEnY1Liiu30=
