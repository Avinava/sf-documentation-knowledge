# Salesforce CRM: Product Mapping

These data mappings are for the Product DLO found in the Loyalty Management bundle of the Salesforce CRM Connector for Data Cloud. A product that the company sells.

## Data Bundle

-   Loyalty Management

## DLO to DMO Mapping

These data mappings relate to the Product object included in the Salesforce CRM connector and how that data connects to DMOs. Learn more about the DLOs in the [Loyalty Management Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.248.0.loyalty.meta/loyalty/loyalty_api_overview.htm).

| DLO Field | DLO Field Data Type | DMO | DMO Field Name |
| --- | --- | --- | --- |
| Product Code | string | Product | Product Code |
| Description | textarea | Product | Description |
| Product Family | picklist | Product | Product Family |
| Product ID | id | Product | Product Id |
| Name | string | Product | Product Name |
| Product SKU | string | Product | Product SKU |
| Quantity Unit Of Measure | picklist | Product | Quantity Unit Of Measure |
