# Clone the Project from GitHub

The “hello world” sample project is part of the Canvas SDK, which is located on GitHub. From GitHub, clone the project to make a copy of it on your local machine.

1.  Log in to [https://github.com](https://github.com "HTML (New Window)").
2.  Locate the project that contains the Canvas SDK and code examples by going to: [https://github.com/forcedotcom/SalesforceCanvasFrameworkSDK](https://github.com/forcedotcom/SalesforceCanvasFrameworkSDK "HTML (New Window)").
3.  To clone the Canvas SDK repository, follow the instructions in the project’s [README.md](https://github.com/forcedotcom/SalesforceCanvasFrameworkSDK/blob/master/README.md#how-to-clone-the-sdk-repository "HTML (New Window)") file.

After you clone the project, the SalesforceCanvasFrameworkSDK directory should contain a subdirectory named src and files, such as pom.xml and README.md.
