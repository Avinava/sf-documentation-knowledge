# Using the  <select> Tag in a Canvas App Tag in a
   Canvas App

> If you use HTML <select> tags in your canvas web
    application and intend to make your canvas app available in the Salesforce mobile app, keep in
    mind that behavior can be unpredictable on some devices. Consider using a jQuery or CSS
    alternative to using <select>.

# Using the <select> Tag in a Canvas App Tag in a Canvas App

If you use HTML <select> tags in your canvas web application and intend to make your canvas app available in the Salesforce mobile app, keep in mind that behavior can be unpredictable on some devices. Consider using a jQuery or CSS alternative to using <select>.
