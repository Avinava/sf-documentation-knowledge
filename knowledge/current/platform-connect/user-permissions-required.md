# User Permissions Required

You need certain user permissions to interact with canvas apps.

The following user permissions are required to create canvas apps and view them in the Canvas App Previewer.

-   “Customize Application”
-   “Modify All Data”

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=platform_connect)

#### Note

The "Customize Application" user permission is required to see Canvas apps in the App Launcher on mobile.
