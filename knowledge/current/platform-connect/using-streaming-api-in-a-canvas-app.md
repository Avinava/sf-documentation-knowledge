# Using Streaming API in a Canvas App

# Using Streaming API in a Canvas App

Canvas provides an event and methods that enable canvas apps to listen for Streaming API notifications.

-   sfdc.streamingapi—JavaScript event that you create and associate with a Streaming API channel defined by a PushTopic. See [Using the Streaming API Event](atlas.en-us.platform_connect.meta/platform_connect/canvas_app_streaming_event_code_example.htm).
-   subscribe—Subscribes to the sfdc.streamingapi event that you define. See [Subscribing to a Streaming API Event](atlas.en-us.platform_connect.meta/platform_connect/canvas_app_streaming_subscribe_code_example.htm).
-   unsubscribe—Unsubscribes from the sfdc.streamingapi event. See [Unsubscribing from a Streaming API Event](atlas.en-us.platform_connect.meta/platform_connect/canvas_app_streaming_unsubscribe_code_example.htm).
