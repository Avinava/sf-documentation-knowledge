# Headers

These are custom HTTP request and response headers that are used for Bulk API 2.0.

-   **[Sforce Call Options Header](atlas.en-us.api_asynch.meta/api_asynch/bulk_api_2_0_headers_sforce_call_options.htm)**
    Use the Sforce-Call-Options header to specify client-specific options when accessing Bulk API 2.0 resources.
-   **[Warnings Header](atlas.en-us.api_asynch.meta/api_asynch/bulk_api_2_0_headers_warnings.htm)**
    Use the Warning header to return warnings, such as the use of a deprecated version of the API.
-   **[Content Type Header](atlas.en-us.api_asynch.meta/api_asynch/bulk_api_2_0_content_type_header.htm)**
    Use the Content-Type header to specify the format for your Bulk API 2.0 request. Set the value of this header to match the contentType of the body of your request.
-   **[Line Ending Header](atlas.en-us.api_asynch.meta/api_asynch/bulk_api_2_0_line_ending_header.htm)**
    Use the Sforce-Line-Ending header to specify how line endings are formatted.
