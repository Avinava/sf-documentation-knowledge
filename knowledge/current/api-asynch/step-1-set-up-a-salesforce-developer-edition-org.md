# Step 1: Set Up a Salesforce Developer Edition Org

This Quick Start suggests using a Developer Edition org. Sign up for a Salesforce Developer Edition org before trying Bulk API 2.0 with this Quick Start.

If you’re not already a member of the developer community, go to [developer.salesforce.com/signup](https://developer.salesforce.com/signup), and follow the instructions for signing up for a Developer Edition account.

You can also use a scratch org or sandbox to follow along with these examples.

![Note](/docs/resources/img/en-us/260.0?doc_id=images%2Ficon_note.png&folder=api_asynch)

#### Note

Developer Edition orgs have a data storage maximum of 5 MB. This limit doesn’t prevent you from working with these examples.
