# Insurance Policy Developer Journey

To manage insurance policies by using Salesforce Quoting, use the Quoting APIs to generate a quote context, which serves as the foundation for policy operations. For managing policies without Salesforce Quoting, use the Rating API to create a price-adjusted context. This price-adjusted context provides the necessary rated data to manage insurance policies without depending on a formal quote record.

-   **[Create and Manage Insurance Policies by Using Salesforce Quoting](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/manage_policy_using_quote.htm)**
    Learn how to effectively use the Insurance Policy and Quoting Connect APIs, along with the associated invocable actions, to build comprehensive user journeys by using Salesforce Flows and Omniscripts.
-   **[Create and Manage Insurance Policies Without Using Salesforce Quoting](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/manage_policy_without_using_quote.htm)**
    Learn how to use the Insurance Policy APIs and invocable actions to manage the policy lifecycle when Salesforce quoting isn't used, or when a third-party system handles the quoting process.
-   **[Cancel Policy](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/cancel_policy.htm)**
    The policy cancellation process addresses the termination of an insurance policy before its scheduled expiration date. Common reasons for canceling a policy include not paying the premiums or a direct cancellation request from the policyholder.
