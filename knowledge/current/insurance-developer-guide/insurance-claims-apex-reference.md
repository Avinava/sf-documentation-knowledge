# Insurance Claims Apex Reference

Apex is a strongly typed, object-oriented programming language that allows developers to execute flow and transaction control statements on the Salesforce Platform server, in conjunction with calls to the API. Insurance Claims Apex reference includes built-in Apex classes and interfaces, grouped by namespace.

-   **[ConnectApi Namespace](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/insurance_claim_connect_api_namespace.htm)**
    The ConnectApi namespace (also called Connect in Apex) in Insurance Claim provides classes for accessing the same data available in the Insurance Policy Administration Business APIs.
