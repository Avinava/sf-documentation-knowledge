# Insurance Quoting Standard Invocable Actions

Learn more about the standard invocable actions available with Insurance Quoting.

-   **[Add Eligible Insurance Clauses Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_add_eligible_insurance_clauses.htm)**
    Add eligible insurance clauses to a quote or context. You must specify either a quote ID or a context ID, and either an instance key or a quote line item ID along with insurance product clause details.
-   **[Create Insurance Quote Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_create_quote.htm)**
    Create an Insurance quote by using a context ID or a set of user inputs that represent quote details.
-   **[Create Insurance Rating Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_create_insurance_rating.htm)**
    Create a runtime context that contains the details to create a quote.
-   **[Generate Insurance Clauses Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_generate_insurance_clauses.htm)**
    Generate insurance clauses based on the specified quote and context information. You must specify either quote ID or context ID to generate insurance clauses.
-   **[Get Eligible Insurance Clauses Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_get_eligible_insurance_clauses.htm)**
    Get eligible insurance clauses based on quote ID and context ID. You must specify either a quote ID or a context ID, and either an instance key or a quote line item ID to get the eligible insurance clauses.
-   **[Get Insurance Quote Details Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_quote_details.htm)**
    Get the details of an insurance quote.
-   **[Get Insurance Rating Input Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_get_insurance_rating_input.htm)**
    Get the rating input data for the insurance products to support the pricing calculations and coverage analysis.
-   **[Reprice Insurance Product Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_reprice_insurance_product.htm)**
    Recalculate the price of insurance products based on user inputs.
-   **[Update Insurance Quote Action](atlas.en-us.insurance_developer_guide.meta/insurance_developer_guide/actions_obj_update_insurance_quote.htm)**
    Update an existing insurance quote by adding, updating, or deleting nodes and optionally executing pricing and rules.
