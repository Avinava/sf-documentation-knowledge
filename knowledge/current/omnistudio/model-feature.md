# Model Feature

Output representation of the feature model that is used for making predictions.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| label | String | The label of the feature model that is used for the scores prediction. | Small, 55.0 | 55.0 |
| name | String | The name of the feature model that is used for the scores prediction. | Small, 55.0 | 55.0 |
| type | String | The type of the feature model that is used for the scores prediction. | Small, 55.0 | 55.0 |
