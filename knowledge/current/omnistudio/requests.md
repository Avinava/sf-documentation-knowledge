# Requests

igital Lending APIs include these request bodies.

    Input representation of the details of the applicants associated to the application form.

    Input representation of the application form product.

-   **[Create Application Form Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_create_application_form_input.htm)**

-   **[Custom Fields Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_digital_lending_custom_fields_input.htm)**

    Input representation of the details of the additional owner of the party financial asset.
-   **[Party Financial Asset Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_party_financial_asset_input.htm)**
