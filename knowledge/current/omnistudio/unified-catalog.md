# Unified Catalog

The Unified Catalog provides a centralized, consistent framework for managing and accessing product and service definitions across business processes. It serves as a single source of truth for catalog data that can be reused by downstream systems and applications, ensuring uniformity in how offerings are defined, discovered, and consumed.

| Available in: Lightning Experience |
| --- |
| Available in: Enterprise, Unlimited, and Developer Editions |


-   **[Unified Catalog Business APIs](atlas.en-us.industries_reference.meta/industries_reference/unified_catalog_apis_overview.htm)**
    The Unified Catalog APIs provide a standardized REST interface for accessing catalog data and related business rules used in service automation workflows. These APIs enable client applications to retrieve catalog entities and eligibility rules in a consistent and versioned manner. They are designed to support scalable integrations without requiring catalog-specific logic in consuming systems.
