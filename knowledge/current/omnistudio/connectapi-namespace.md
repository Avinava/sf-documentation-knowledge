# ConnectApi Namespace

# ConnectApi Namespace

The ConnectApi namespace (also called Connect in Apex) provides classes to manage compliance procedure evaluations.

These classes are available in the ConnectApi namespace.

-   **[IndustriesCompliance Class](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_IndustriesCompliance_static_methods.htm)**  
    Evaluate a compliance procedure with a business context by using the IndustriesCompliance class.
-   **[ConnectApi Input Classes](atlas.en-us.industries_reference.meta/industries_reference/process_compliance_navigator_apex_input_classes.htm)**  
    Process Compliance Navigator includes these Apex input classes.
-   **[ConnectApi Output Classes](atlas.en-us.industries_reference.meta/industries_reference/process_compliance_navigator_apex_output_classes.htm)**  
    Process Compliance Navigator includes these Apex output classes.
