# Record Alerts Tooling API Objects

# Record Alerts Tooling API Objects

Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata. For more information about Tooling API objects and to find a complete reference of all the supported objects, see **[Introducing Tooling API](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_tooling.meta/api_tooling/intro_api_tooling.htm)**.

-   **[RecordAlertCategory](atlas.en-us.industries_reference.meta/industries_reference/tooling_api_objects_recordalertcategory.htm)**  
    Represents a category to group and present record alerts. This object is available in API version 54.0 and later.
-   **[RecordAlertDataSource](atlas.en-us.industries_reference.meta/industries_reference/tooling_api_objects_recordalertdatasource.htm)**  
    Represents an internal or external source for record alert data. This object is available in API version 54.0 and later.
