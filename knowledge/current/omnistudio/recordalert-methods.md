# RecordAlert Methods

These are RecordAlert methods.

-   **[getRecordAlerts(parentId, whatId)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_RecordAlert_getRecordAlerts_1.htm)**
    Retrieve a list of active record alerts.
-   **[performRecordAlertAction(alertId, actionParameters)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_RecordAlert_performRecordAlertAction_1.htm)**
    Snooze internal record alerts for a specified time period or dismiss them.
-   **[performActionOnExternalAlerts(sourceSystemId, actionParameters)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_RecordAlert_performActionOnExternalAlerts_1.htm)**
    Snooze external record alerts that are from an external source system for a specified time period or dismiss them.
