# Scoring Framework Metadata Types

Use the Scoring Framework metadata types to define the machine learning model that’s used to generate scores for your predictive use cases.

| Available in: Lightning Experience |
| --- |
| Available when the admin settings for AI Accelerator and for the product related to the use case are enabled. The Salesforce org must have the CRM Plus license and the product’s CRM license. |


-   **[AIScoringModelDefinition](atlas.en-us.industries_reference.meta/industries_reference/meta_aiscoringmodeldefinition.htm)**
    Represents information about a machine learning model that’s used by the Scoring Framework for Industries Cloud Einstein. The machine learning model is used for scoring, including its configuration.
