# Virtual Calls Standard Objects

# Virtual Calls Standard Objects

Use these standard objects to configure virtual calls.

-   **[ServiceProviderEvent](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_serviceproviderevent.htm)**  
    Represents events generated by third-party service providers as platform events. This object is available in API version 55.0 and later.
-   **[VirtualVisitConfig](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_virtualvisitconfig.htm)**  
    Represents the configuration of an external video provider. This object is available in API version 54.0 and later.
