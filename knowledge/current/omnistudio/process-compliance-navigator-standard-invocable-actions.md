# Process Compliance Navigator Standard Invocable Actions

Learn more about the standard invocable actions available with Process Compliance Navigator.

-   **[Evaluate Compliance Validation Procedure Action](atlas.en-us.industries_reference.meta/industries_reference/actions_obj_evaluate_compliance_validation_procedure.htm)**
    Evaluate a compliance validation procedure, and create a procedure evaluation API response.
-   **[Evaluate Compliance Validation Procedure Asynchronous Action](atlas.en-us.industries_reference.meta/industries_reference/actions_obj_evaluate_compliance_validation_procedure_async.htm)**
    Asynchronously evaluate a compliance validation procedure and create a procedure evaluation job ID.
-   **[Retrieve Compliance Async Evaluation Job Detail Action](atlas.en-us.industries_reference.meta/industries_reference/actions_obj_retrieve_compliance_async_eval_job_detail.htm)**
    Get job details for an asynchronous compliance evaluation.
