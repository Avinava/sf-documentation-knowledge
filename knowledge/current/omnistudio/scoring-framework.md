# Scoring Framework

Use the Scoring Framework developer resources to define the template configurations that are used to create CRM Analytics apps. Generate scores for your predictive use cases using the preconfigured recipes and Einstein Discovery models installed with the CRM Analytics apps.

-   **[Scoring Framework Metadata Types](atlas.en-us.industries_reference.meta/industries_reference/scoring_framework_metadata_apis_parent.htm)**
    Use the Scoring Framework metadata types to define the machine learning model that’s used to generate scores for your predictive use cases.
-   **[Scoring Framework Tooling Objects](atlas.en-us.industries_reference.meta/industries_reference/scoring_framework_tooling_apis_parent.htm)**
    Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata. For more information about Tooling API objects and to find a complete reference of all the supported objects, see [Introducing Tooling API](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_tooling.meta/api_tooling/intro_api_tooling.htm "HTML (New Window)").
