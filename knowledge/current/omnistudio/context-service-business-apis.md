# Context Service Business APIs

Context Service Business APIs are RESTful APIs that are sometimes available as Apex classes and methods.

-   **[REST Reference](atlas.en-us.industries_reference.meta/industries_reference/context_service_api_rest_references.htm)**  
    You can access the Context Service APIs using REST endpoints. These REST APIs follow similar conventions as Connect REST APIs.
