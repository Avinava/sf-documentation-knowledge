# Request Bodies

Here’s a list of Context Service API request bodies.

-   **[Context Attribute Mappings Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_attribute_mappings_input.htm)**
    Input representation of context attribute mapping.
-   **[Context Attribute Hydration Details Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_attr_hydration_details_input.htm)**
    Input representation of context attribute hydration detail.
-   **[Context Attributes Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_attributes_input.htm)**
    Input representation of context attribute.
-   **[Context Definition Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_definitions_input.htm)**
    Input representation of the context definition.
-   **[Context Definition Upgrade Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_definition_upgrade.htm)**
    Input representation of context definition upgrade.
-   **[Context Definition Filter Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_definition_filter_input.htm)**
    Input representation details of context definition filter.
-   **[Context Definition Filter List Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_definition_filter_list_input.htm)**
    Input representation for a list of Context Definition Filters
-   **[Context Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_input.htm)**
    Input representation for defining a context.
-   **[Context Mappings Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_mappings_input.htm)**
    Input representation of context mapping.
-   **[Context Metadata Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_meta_data_input.htm)**
    Input representation of context metadata.
-   **[Context Node ID Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_node_id_input.htm)**
    Input representation of the list of context node IDs to create the relationship nodes for.
-   **[Context Node Mappings Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_node_mappings_input.htm)**
    Input representation of context node mapping.
-   **[Context Nodes Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_nodes_input.htm)**
    Input representation of context node.
-   **[Context Tag Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_context_tag_input.htm)**
    Input representation of the context tag.
-   **[Context Query Tags Leaner (POST)](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_context_query_lean_tag.htm)**
    Query tags and return a memory-optimized (leaner) result suitable for Apex and low-heap clients. Eliminate redundant metadata to reduce heap usage and payload size.
-   **[Node Path and Tag Values Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_node_path_and_tag_values.htm)**
    Input representation of the node path which needs to update with tag details.
-   **[Query Context Record Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_query_context_record_input.htm)**
    Input representation of query context record.
-   **[Query Record Status Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_query_data_input.htm)**
    Input representation of status and related error messages of query data records.
-   **[Tag Values Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_tag_values_input.htm)**
    Input representation of the list of Tag Names to be updated and their values.
