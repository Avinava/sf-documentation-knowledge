# Rule Results

Output representation of the result of executing a single input in an expression set.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| error | Error Result | The details of the error resulting from executing a single input in an expression set. | Small, 55.0 | 55.0 |
| results | Variable Map Output | The result of the aggregation step in an expression set. | Small, 55.0 | 55.0 |
