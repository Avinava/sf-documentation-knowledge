# ConnectApi Input Classes

Process Compliance Navigator includes these Apex input classes.

-   **[ConnectApi.ComplianceBulkUpdateInput](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_bulk_update.htm)**
    Input representation to asynchronously bulk update the status of the clause version records.
-   **[ConnectApi.ComplianceProcedureEvaluationRequest](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_procedure_evaluation_request.htm)**
    Input representation details of a compliance evaluation request.
-   **[ConnectApi.ComplianceProcedureEvaluationInfo](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_procedure_evaluation_info.htm)**
    Input representation for evaluating a compliance procedure.
-   **[ConnectApi.ComplianceProcedureBusinessContext](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_procedure_business_context.htm)**
    Input representation of a business context with the parameter name and value.
-   **[ConnectApi.ComplianceLoggingInfo](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_logging_info.htm)**
    Compliance evaluation logging request.
-   **[ConnectApi.ComplianceControllogValue](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_controllog_value.htm)**
    Compliance controllog value.
-   **[ConnectApi.ComplianceLoggingBusinessContextEvidenceValue](atlas.en-us.industries_reference.meta/industries_reference/apex_connectapi_input_compliance_logging_business_context_evidence_value.htm)**
    Input representation of the generic context and evidence value.
