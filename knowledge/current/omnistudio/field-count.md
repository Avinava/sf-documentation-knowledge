# Field Count

# Field Count

Details of fields whose values fall outside the expected or trained data range for the prediction model.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| count | Integer | Number of times the values occured. | Small, 62.0 | 62.0 |
| fieldValue | String | Out of bound field value. | Small, 62.0 | 62.0 |
