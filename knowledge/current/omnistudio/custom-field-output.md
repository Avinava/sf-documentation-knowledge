# Custom Field Output

# Custom Field Output

Output representation for creating an engagement interaction, topic, or attendee record with a custom field.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| key | String | The name of a custom field from a Salesforce object. | Small, 56.0 | 56.0 |
| value | Object | The value of a custom field from a Salesforce object. | Small, 56.0 | 56.0 |
