# Discovery Framework API Requests

# Discovery Framework API Requests

Here’s the list of the Discovery Framework Document Decision API request bodies.

-   **[Assessment Details Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_assessment_responses_assessment_details_input.htm)**  
    Input representation of the assessment details of Assessment Response request.
-   **[Assessment Links Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_assessment_links_input.htm)**  
    Associated assessments including id, category, reason and sequence.
-   **[Assessment Reasons Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_assessment_responses_assessment_reason_input.htm)**  
    Input representation of the assessment reason in Assessment Response request.
-   **[Assessment Responses Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_assessment_responses_input.htm)**  
    Input for assessment responses.
-   **[Assessment Values Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_assessment_responses_assessment_values_input.htm)**  
    Input representation of the assessment value of Assessment Response request.
-   **[Document Decision Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_document_decision_input.htm)**  
    Input representation of the Document Decision request.
-   **[Question Responses Input](atlas.en-us.industries_reference.meta/industries_reference/connect_requests_question_responses_input.htm)**  
    Input for assessment question responses.
