# Actionable List Members API Responses

Here’s the list of the Actionable List Members API response bodies.

-   **[Assign Actionable List](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_assign_actionable_list.htm)**  
    Output representation of the actionable list assignment request.
-   **[Actionable List Member Queue](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_actionable_list_member_queue.htm)**  
    Output representation of the actionable list member queue details.
-   **[Actionable List Member Queues](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_actionable_list_member_queues.htm)**  
    Output representation of the fetch request of the queues that contain the actionable list member as a supported object.
-   **[Actionable List Update Member Count Output](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_actionable_list_update_member_count_output.htm)**  
    Actionable List Member Count output.
-   **[Field](atlas.en-us.industries_reference.meta/industries_reference/ui_api_responses_field.htm)**  
    A field's metadata.
-   **[Field Value](atlas.en-us.industries_reference.meta/industries_reference/ui_api_responses_field_value.htm)**  
    The raw and displayable field values for a field in a record.
-   **[Sales Excellence Status](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_sales_excellence_status.htm)**  
    Output representation to show the status of the assignment API request.
-   **[Search Actionable List Members](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_search_actionable_list_members.htm)**  
    Output representation of the request for searching for actionable list members.
-   **[Record](atlas.en-us.industries_reference.meta/industries_reference/ui_api_responses_record.htm)**  
    The field data, API name, child relationship data, and record type information for a record.
