# Actionable Segmentation Metadata API Types

Use the Actionable Segmentation metadata types to set up actionable list definitions. Create actionable list definitions, configure visible columns for actionable lists, and configure list member statuses before activating list definitions.

-   **[ActionableListDefinition](atlas.en-us.industries_reference.meta/industries_reference/meta_actionablelistdefinition.htm)**  
    Represents the data source definition details associated with an actionable list.
