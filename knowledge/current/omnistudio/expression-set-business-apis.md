# Expression Set Business APIs

Connect REST APIs help customers, partners, and ISVs integrate with Salesforce software and UIs.

The runtime service for Expression Set is exposed through Connect APIs.

With Expression Set Connect APIs, you can:

-   [Invoke an expression set](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_bre_expression_set.htm "Invokes expression sets in Business Rule Engine.").
-   [Create an expression set](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_bre_create_expression_set.htm "Creates an expression set in Business Rules Engine. An expression set performs a series of calculations by using lookup tables and user-defined variables and constants.").

-   **[Resources](atlas.en-us.industries_reference.meta/industries_reference/expression_set_resources.htm)**
    Expression Set API includes these resources.
-   **[Request Bodies](atlas.en-us.industries_reference.meta/industries_reference/expression_set_requests.htm)**
    Expression Set API includes these request bodies.
-   **[Response Bodies](atlas.en-us.industries_reference.meta/industries_reference/expression_set_responses.htm)**
    Expression Set API includes these response bodies.
