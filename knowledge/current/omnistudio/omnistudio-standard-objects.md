# Omnistudio Standard Objects

Omnistudio data model defines the foundational objects and fields used across Omnistudio components. It enables consistent data access and management within guided experiences and process automation.

-   **[OmniComponentErrorLog](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omnicomponenterrorlog.htm)**
    For internal use only.
-   **[OmniDataPack](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omnidatapack.htm)**
    For internal use only.
-   **[OmniDataTransform](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omnidatatransform.htm)**
    For internal use only.
-   **[OmniDataTransformItem](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omnidatatransformitem.htm)**
    For internal use only.
-   **[OmniESignature](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniesignature.htm)**
    For internal use only.
-   **[OmniExtTrackingDef](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniexttrackingdef.htm)**
    Represents a connection between an OmniTrackingGroup in OmniAnalytics and a third-party Analytics system such as Google Analytics. This object is available in API version 60.0 and later.
-   **[OmniExtTrackingEventDef](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniexttrackingeventdef.htm)**
    Represents a format for FlexCard or OmniScript user interaction data that a third-party Analytics system such as Google Analytics can accept. This object is available in API version 60.0 and later.
-   **[OmniInteractionAccessConfig](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniinteractionaccessconfig.htm)**
    For internal use only.
-   **[OmniInteractionConfig](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniinteractionconfig.htm)**
    For internal use only.
-   **[OmniProcess](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniprocess.htm)**
    For internal use only.
-   **[OmniProcessCompilation](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniprocesscompilation.htm)**
    For internal use only.
-   **[OmniProcessElement](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniprocesselement.htm)**
    For internal use only.
-   **[OmniProcessTransientData](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniprocesstransientdata.htm)**
    For internal use only.
-   **[OmniScriptSavedSession](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniscriptsavedsession.htm)**
    For internal use only.
-   **[OmniTrackingComponentDef](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omnitrackingcomponentdef.htm)**
    Represents a FlexCard or OmniScript that is a member of an OmniTrackingGroup, which tracks user interactions in OmniAnalytics. This object is available in API version 60.0 and later.
-   **[OmniTrackingGroup](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omnitrackinggroup.htm)**
    Represents a group of FlexCard and OmniScript components that have their user interactions tracked together in OmniAnalytics. This object is available in API version 60.0 and later.
-   **[OmniUiCard](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_omniuicard.htm)**
    For internal use only.
