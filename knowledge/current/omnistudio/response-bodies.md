# Response Bodies

Learn more about the available response bodies of Unified Catalog APIs.

-   **[Eligibility Rule](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_eligibility_rule.htm)**  
    Output representation details of eligibility rule.
-   **[Eligibility Rule Details](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_eligibility_rule_details.htm)**  
    Output representation of eligibility rule details for qualification or disqualification decision tables.
-   **[Eligibility Rule Metadata](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_eligibility_rule_metadata.htm)**  
    Output representation details of get eligibility rule metadata request.
-   **[Eligibility Rule Parameter](atlas.en-us.industries_reference.meta/industries_reference/connect_responses_eligibility_rule_parameter.htm)**  
    Output representation of the input and output variables for the decision table.
