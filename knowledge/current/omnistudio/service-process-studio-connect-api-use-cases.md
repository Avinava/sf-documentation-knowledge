# Service Process Studio Connect API Use Cases

Use the Service Process Studio connect API use cases to invoke connect API resources to create and manage instances of particular service processes without using OmniStudio.

-   **[Transaction Dispute Management Use Case](atlas.en-us.industries_reference.meta/industries_reference/service_process_usecase_dispute_management.htm)**
    Transaction dispute management provides a streamlined process for users to capture and submit customer disputes related to financial transactions. You can use the Service Process connect API resource to create all the records related to a transaction dispute.
