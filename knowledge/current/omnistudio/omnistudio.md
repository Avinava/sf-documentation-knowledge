# Omnistudio

# Omnistudio

Omnistudio provides a suite of services, components, and data model objects that combine to create Industry Cloud applications. Use Omnistudio to create guided interactions using data from your Salesforce org and external sources. Create Omniscripts to guide users through sales and service processes with fast, personalized responses. Get and post data with Omnistudio Data Mappers and Integration Procedures.

| Available in: Lightning Experience |
| --- |
| Available in: Enterprise, Unlimited, and Developer Editions |
  

-   **[Omnistudio Standard Objects](atlas.en-us.industries_reference.meta/industries_reference/omnistudio_standard_objects_parent.htm)**  
    Omnistudio data model defines the foundational objects and fields used across Omnistudio components. It enables consistent data access and management within guided experiences and process automation.
-   **[Omnistudio Business APIs](atlas.en-us.industries_reference.meta/industries_reference/omnistudio_apis.htm)**  
    Omnistudio Business APIs are RESTful APIs that are sometimes available as Apex classes and methods.
-   **[Omnistudio Metadata API Types](atlas.en-us.industries_reference.meta/industries_reference/omnistudio_metadata_api_parent.htm)**  
    Metadata API enables you to access some types and feature settings that you can customize in the user interface.
-   **[Omnistudio Apex Reference](atlas.en-us.industries_reference.meta/industries_reference/omnistudio_apex_reference.htm)**  
    Apex is a strongly typed, object-oriented programming language that allows developers to execute flow and transaction control statements on the Salesforce Platform server, in conjunction with calls to the API. Omnistudio Apex reference includes built-in Apex classes and interfaces, grouped by namespace.
