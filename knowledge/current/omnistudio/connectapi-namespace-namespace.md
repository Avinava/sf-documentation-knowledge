# ConnectApi Namespace Namespace

# ConnectApi Namespace Namespace

The ConnectApi namespace (also called Connect in Apex) provides classes for accessing the same data available in Connect REST API. Use Connect in Apex to create custom experiences in Salesforce.

For information about working with the ConnectApi classes, see [Connect in Apex](https://developer.salesforce.com/docs/atlas.en-us.260.0.apexcode.meta/apexcode/connectAPI_overview.htm).

-   **[EngagementContainerConnect Methods](atlas.en-us.industries_reference.meta/industries_reference/engagementinteractionconnect_apex_class_methods.htm)**  
    These are the methods for EngagementContainerConnect. Methods are static.
-   **[Input Classes](atlas.en-us.industries_reference.meta/industries_reference/service_excellence_apex_input_classes.htm)**  
    These are Engagements Apex input classes.
-   **[Output Classes](atlas.en-us.industries_reference.meta/industries_reference/service_excellence_apex_output_classes.htm)**  
    These are the Engagements Apex output classes.
