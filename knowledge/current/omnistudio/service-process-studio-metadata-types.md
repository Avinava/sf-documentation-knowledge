# Service Process Studio Metadata Types

Use the Service Process Studio metadata types to set up your Service Process use case. Define attributes, OmniScripts, flows, classes, and the Integration Definition.

| Available in: Lightning Experience |
| --- |
| Available when the admin settings for Service Process Studio are enabled. The Salesforce org must have the CRM Plus license and the product’s CRM license. |


-   **[IntegrationProviderDef](atlas.en-us.industries_reference.meta/industries_reference/meta_integrationproviderdef.htm)**
    Represents an integration definition associated with a service process. Stores data for the Industries: Send Apex Async Request and Industries: Send External Async Request invocable actions.
-   **[ServiceProcess](atlas.en-us.industries_reference.meta/industries_reference/meta_serviceprocess.htm)**
    Represents a process created in Service Process Studio and its associated attributes.
