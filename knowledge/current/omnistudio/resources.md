# Resources

Here’s a list of Intelligent Document Reader API resources.

-   **[Document Reader Mappings (POST)](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_document_reader_mappings.htm)**
    Retrieve the mappings between the values of the keys extracted from the document to the Salesforce object fields with respect to a given template.
-   **[OCR Fields Mapping (GET)](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_ocr_mapping_details.htm)**
    Retrieve mapping between the values of the keys extracted from the document with respect to an OCR template.
-   **[OCR Text Extraction (GET, POST)](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_extraction_of_content_doc.htm)**
    Retrieve details of the extracted key-value pairs for a document from the latest version of the OCR document scan result. Create an OCR document scan result record using the extracted text from the latest version of a document.
-   **[OCR Templates (GET)](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_get_ocr_templates.htm)**
    Retrieve a list of active OCR templates. You can also fetch the OCR templates by document type and target object that are associated with the OCR templates.
-   **[OCR Template Details (GET)](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_get_ocr_template_details.htm)**
    Retrieve the details of an active OCR template by the template ID.
-   **[Record Creation with Content Documents](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_create_content_documents.htm)**
    Create records in Salesforce objects using a document based on a content document record.
-   **[Record Creation with Received Documents](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_create_received_documents.htm)**
    Create records in Salesforce objects using a document based on a received document record.
-   **[Record Update with Content Documents](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_update_content_documents.htm)**
    Update an existing record in a Salesforce object using a document based on a content document record.
-   **[Record Update with Received Documents](atlas.en-us.industries_reference.meta/industries_reference/intelligent_document_reader_connect_resources_update_received_documents.htm)**
    Update an existing record in a Salesforce object using a document based on a received document record.
