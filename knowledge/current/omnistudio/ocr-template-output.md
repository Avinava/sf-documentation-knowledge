# OCR Template Output

Output representation of the details of an OCR template.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| apiName | String | Developer name of the OCR template. | Big, 55.0 | 55.0 |
| documentTypeId | String | ID of the document type associated with the OCR template. | Big, 55.0 | 55.0 |
| name | String | Name of the OCR template. | Big, 55.0 | 55.0 |
| ocrTargetObjects | OCR Target Object Output[] | Details of the target objects. | Big, 55.0 | 55.0 |
| ocrTemplateId | String | ID of the OCR template record. | Big, 55.0 | 55.0 |
