# Actionable List Members API Resources

Here’s the list of the Actionable List Members API resources.

-   **[Actionable List Members Assignments](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_post_actionable_list_configuration_details.htm)**
    Update the priority and OmniScript key in the Actionable List object and update the assignee in the Actionable List Member object.
-   **[Actionable List Member Count](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_actionable_list_update_member_count.htm)**
    Update the member count of an actionable list.
-   **[Actionable List Member Queues (GET)](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_fetch_queues.htm)**
    Fetch a list of queues that contain the actionable list member as a supported object.
-   **[Actionable List Member Search by Using Actionable List ID](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_get_actionable_list_members_using_id.htm)**
    Retrieves the actionable list members for the specified actionable list ID.
-   **[Actionable List Member Search by Using Filter Criteria](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_get_actionable_list_members_using_filter_criteria.htm)**
    Get a list of actionable list members by using filter criteria.
