# Outreach List API Resources

Here’s the list of the Outreach List API resources.

-   **[Actionable Lists Assigned to an Agent (GET)](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_get_actionable_lists_assigned_to_an_agent.htm)**
    Retrieve the actionable lists that are assigned to the querying agent.
-   **[Actionable List Metadata (GET)](atlas.en-us.industries_reference.meta/industries_reference/connect_resources_get_actionable_list_metadata.htm)**
    Retrieve the actionable list member statuses, the count of actionable list members in each status, and the quick action metadata for a specified actionable list ID.
