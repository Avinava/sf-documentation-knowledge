# Stage Management Metadata API Types

Metadata API enables you to access some types and feature settings that you can customize in the user interface.

-   **[StageAssignment](atlas.en-us.industries_reference.meta/industries_reference/meta_stageassignment.htm)**
    Represents a collection of fields to automatically assign stage definitions to records based on rule criteria.
-   **[StageDefinition](atlas.en-us.industries_reference.meta/industries_reference/meta_stagedefinition.htm)**
    Represents a collection of fields to set up the states and transitions for Stage Management.
