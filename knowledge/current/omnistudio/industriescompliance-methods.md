# IndustriesCompliance Methods

These methods are for IndustriesCompliance. All methods are static.

-   **[bulkUpdateClauseVersions(bulkUpdateInputs)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_IndustriesCompliance_bulkUpdateClauseVersions.htm)**
    Asynchronously bulk update the status of all clause versions for the specified regulation version or policy version.
-   **[evaluateComplianceProcedureAsync(procedureName, evaluationInputs)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_IndustriesCompliance_evaluateComplianceProcedureAsync.htm)**
    Asynchronously evaluate a compliance procedure with the business context.
-   **[evaluateComplianceProcedure(evaluationInputs, procedureName)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_IndustriesCompliance_evaluateComplianceProcedure.htm)**
    Evaluate a compliance procedure with a business context.
-   **[getComplianceJobResults(jobId, pageSize, offset)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_IndustriesCompliance_getComplianceJobResults.htm)**
    Get the compliance job results for a specific job.
-   **[logComplianceEvaluations(loggingInputs)](atlas.en-us.industries_reference.meta/industries_reference/apex_ConnectAPI_IndustriesCompliance_logComplianceEvaluations.htm)**
    Log compliance check events for control. This API supports a maximum number of 25 logs per request. You can make up to 200,000 audit log API calls per year.
