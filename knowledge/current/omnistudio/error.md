# Error

Specifies the details of the error when the request fails.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| message | String | The message that’s displayed for the error. | Small, 56.0 | 56.0 |
| statusCode | String | The status code that indicates the error. | Small, 56.0 | 56.0 |
