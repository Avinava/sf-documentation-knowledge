# Actionable Segmentation Standard Objects

This section lists the standard objects associated with Actionable Segmentation with their standard fields. Use the standard objects to create actionable lists, capture basic information such as reference record, member status, and column details for actionable list members.

-   **[ActionableList](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelist.htm)**
    Represents an actionable list. This object is available in API version 57.0 and later.
-   **[ActionableListMember](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelistmember.htm)**
    Represents an individual actionable list item. This object is available in API version 57.0 and later.
-   **[ActionableListColumn](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelistcolumn.htm)**
    Represents information about the columns in an actionable list. This object is available in API version 57.0 and later.
-   **[ActionableListFilterCondition](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelistfiltercondition.htm)**
    Represents the filter condition for including or excluding data from the dataset associated with the actionable list. This object is available in API version 59.0 and later.
-   **[ActionableListFilterCriteria](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelistfiltercriteria.htm)**
    Represents a logical expression containing filter conditions to include or exclude data from the dataset that's associated with the actionable list. This object is available in API version 59.0 and later.
-   **[ActionableListDefFltrCond](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelistdeffltrcond.htm)**
    Represents the filter condition for including or excluding data from the dataset associated with the actionable list definition. This object is available in API version 60.0 and later.
-   **[ActionableListDefFltrCrit](atlas.en-us.industries_reference.meta/industries_reference/sforce_api_objects_actionablelistdeffltrcrit.htm)**
    Represents a logical expression containing filter conditions to include or exclude data from the dataset that's associated with the actionable list definition. This object is available in API version 60.0 and later.
