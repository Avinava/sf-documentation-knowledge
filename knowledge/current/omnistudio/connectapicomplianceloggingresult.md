# ConnectApi.ComplianceLoggingResult

Output representation of a single compliance logging.

| Property Name | Type | Description | Available Version |
| --- | --- | --- | --- |
| errorMessage | String | Brief description of the error. | 64.0 |
| statusCode | Integer | Short string that identifies the compliance logging status. | 64.0 |
| transactionKey | String | Transaction key to identify the log message. | 64.0 |
