# Introduction to Outcome Management

> Use Outcome Management to define, measure, and evaluate an organization’s outcome
  strategy. Set time-bound targets and baseline values and track results. Consistently measure the
  same indicators across different outcomes or programs. Organize outcomes into groups to reflect an
  organizational strategy, like a logic model or a set of outcomes that external stakeholders care
  about.

# Introduction to Outcome Management

Use Outcome Management to define, measure, and evaluate an organization’s outcome strategy. Set time-bound targets and baseline values and track results. Consistently measure the same indicators across different outcomes or programs. Organize outcomes into groups to reflect an organizational strategy, like a logic model or a set of outcomes that external stakeholders care about.
