# Field Representation

Output representation of the field that is returned as part of the search configuration metadata.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| displayType | String | Display type of the field. | Small, 60.0 | 60.0 |
| fieldLabel | String | Display label of the field. | Small, 58.0 | 58.0 |
| fieldName | String | API name of the field. | Small, 58.0 | 58.0 |
