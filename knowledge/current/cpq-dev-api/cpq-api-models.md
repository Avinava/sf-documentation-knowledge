# CPQ API Models

CPQ API references several CPQ data models. To use the Salesforce CPQ API, create classes for each data model in your org.

| Available in: Salesforce CPQ Spring ’17 and later |
| --- |


-   **[CPQ API QuoteLineModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quoteline_model_1.htm)**
    The Quote Line model represents a quote line data model in Salesforce CPQ.
-   **[CPQ API QuoteLineGroupModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quotelinegroup_model_2.htm)**
    The Quote Line Group model represents a quote line group data model in Salesforce CPQ.
-   **[CPQ API OptionModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_option_model_3.htm)**
    The Option model represents a product option data model in Salesforce CPQ.
-   **[CPQ API FeatureModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_feature_model_4.htm)**
    The Feature model represents a product feature data model in Salesforce CPQ.
-   **[CPQ API ConfigAttributeModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_configattr_model_5.htm)**
    The ConfigAttribute model represents the configuration attribute object in Salesforce CPQ.
-   **[CPQ API ConfigurationModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_config_model_6.htm)**
    The Configuration model represents a bundle product in Salesforce CPQ.
-   **[CPQ API ConstraintModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_constraint_model_7.htm)**
    The Constraint model represents the option constraint object in Salesforce CPQ.
-   **[CPQ API ProductModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_product_model_8.htm)**
    The Product model represents a product data model in Salesforce CPQ.
-   **[CPQ API QuoteModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quote_model_9.htm)**
    The Quote model represents a CPQ quote data model in Salesforce CPQ.
-   **[CPQ API QuoteProposalModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quoteproposal_model.htm)**
    The QuoteProposal model represents a quote document in Salesforce CPQ.
-   **[CPQ API QuoteTermModel](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quoteterm_model.htm)**
    The QuoteTerm model represents the quote term object in Salesforce CPQ.
