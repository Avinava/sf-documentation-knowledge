# CPQ Quote API

Manage quotes and quoting actions with CPQ quote API.

| Available in: Salesforce CPQ Summer ’16 and later |
| --- |


-   **[Save Quote API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_quote_api_save_final.htm)**
    The Save Quote API saves a CPQ quote.
-   **[Calculate Quote API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_quote_api_calculate_final.htm)**
    The Calculate Quote API calculates the prices of a CPQ quote.
-   **[Read Quote API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_read_quote.htm)**
    The Read Quote API reads a quote from a CPQ quote ID.
-   **[Validate Quote API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_validate_quote.htm)**
    Validate a CPQ quote and return any validation errors.
-   **[Add Products API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_add_products.htm)**
    Receive a CPQ quote, product collection, and quote group key in a request, and return a Quote model with all provided products added as quote lines.
-   **[Read Product API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_read_product.htm)**
    The Read Product API takes the request’s product ID, pricebook ID, and currency code and returns a Product model. The Product model loads the product from your catalog when the user requests it.
-   **[Create and Save Quote Proposal API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quote_document.htm)**
    Create and save a CPQ quote proposal.
-   **[Quote Term Reader API](atlas.en-us.cpq_dev_api.meta/cpq_dev_api/cpq_api_quote_term_loader.htm)**
    Retrieve quote terms for a quote.
