# Salesforce Cloud Objects

Cloud objects are used only by a specific cloud. These objects sometimes have different limitations, APIs, or querying behavior than standard common objects.

Some Salesforce objects require special licensing or purchase. If an object is included in the Object Reference for the Salesforce Platform, the special-access section of the page about the object indicates licensing or purchasing requirements. In most cases, these objects are documented in implementation guides for their cloud, not in the Object Reference.
