# Keywords

Keywords are case-sensitive and must be lowercase.

For a list of SAQL statements that are keywords, see [SAQL Statements](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_statements_concept.htm "A query is made up of statements. Each SAQL statement has an input stream, an operation, and an output stream.").
