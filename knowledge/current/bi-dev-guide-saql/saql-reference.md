# SAQL Reference

These hands-on SAQL examples walk you through writing a query to retrieve data

-   **[SAQL Basic Elements](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_basic_elements.htm)**
    Basic elements are the building blocks of your SAQL query.
-   **[SAQL Operators](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_operators.htm)**
    Use operators to perform mathematical calculations or comparisons.
-   **[SAQL Statements](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_statements_concept.htm)**
    A query is made up of statements. Each SAQL statement has an input stream, an operation, and an output stream.
-   **[SAQL Functions](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions.htm)**
    Use functions to perform complex operations on your data.
