# String Functions

Use SAQL string functions to format your measure and dimension fields.

-   **[ascii()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_ascii.htm)**
    Returns the UTF-8 code value of a character n.
-   **[chr()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_chr.htm)**
    Returns the UTF-8 character of integer n.
-   **[ends\_with()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_ew.htm)**
    Returns true if the string ends with the specified characters.
-   **[index\_of()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_io.htm)**
    Returns the location (index) of the specified characters.
-   **[len()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_len.htm)**
    Returns the number of characters in the string.
-   **[lower()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_lower.htm)**
    Returns a copy of the string with all characters in lower case.
-   **[ltrim()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_ltrim.htm)**
    Removes the specified characters from the beginning of a string.
-   **[mv\_to\_string()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_mts.htm)**
    Converts multivalue fields to string fields.
-   **[number\_to\_string](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_nts.htm)**
    Converts a number literal to a string literal.
-   **[replace()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_replace.htm)**
    Replaces a substring with the specified characters.
-   **[rtrim()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_rtrim.htm)**
    Removes the specified characters from the end of a string.
-   **[starts\_with()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_sw.htm)**
    Returns true if the string starts with the specified characters.
-   **[string\_to\_number](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_stn.htm)**
    Converts a string literal to a number literal.
-   **[substr()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_substr.htm)**
    Returns a substring that starts at the specified position. You can also specify the length of the substring to return.
-   **[trim()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_trim.htm)**
    Removes the specified substring from the beginning and the end of a string.
-   **[upper()](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_string_up.htm)**
    Returns a copy of the string with all characters in upper case.
