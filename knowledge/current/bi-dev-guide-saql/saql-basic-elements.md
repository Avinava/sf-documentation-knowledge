# SAQL Basic Elements

Basic elements are the building blocks of your SAQL query.

-   **[Statements](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_statements.htm)**
    A SAQL query loads input data, operates on it, and outputs the result data. A query is made up of statements. Each SAQL statement has an input stream, an operation, and an output stream.
-   **[Keywords](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_keywords.htm)**
    Keywords are case-sensitive and must be lowercase.
-   **[Identifiers](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_identifiers.htm)**
    SAQL identifiers are case-sensitive and must be enclosed in single quotation marks (').
-   **[Number Literals](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_number_literals.htm)**
    A number literal represents a number in your script.
-   **[String Literals](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_string_literals.htm)**
    A string is a set of characters inside double quotes (").
-   **[Boolean Literals](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_boolean_literals.htm)**
    A boolean literal represents true or false (yes or no) in your script.
-   **[Multivalue Field](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_multivalue.htm)**
    A multivalue field contains more than one value.
-   **[Quoted String Escape Sequences](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_quotedstringescapes.htm)**
    Strings can be escaped with the backslash character.
-   **[Special Characters](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_special_characters.htm)**
    Certain characters have special meanings in SAQL.
-   **[Comments](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_comments.htm)**
    To add a single-line comment in SAQL, preface your comment with two hyphens (\--). To add a multi-line comment, start your comment with a forward slash and asterisk (/\*) and end it with an asterisk and forward slash (\*/).
