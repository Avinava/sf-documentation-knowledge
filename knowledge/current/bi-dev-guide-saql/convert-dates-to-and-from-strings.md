# Convert Dates to and from Strings

You can convert dates to strings.

-   **[date\_to\_string(DateTime | DateOnly, format)](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_timezone_date2string.htm)**
    Converts a date to a string.
-   **[toDateTime(epoch)](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_timezone_datetime_epoch.htm)**
    Converts an epoch day to a DateTime type.
-   **[toDateTime(string, format)](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_timezone_datetime_string.htm)**
    Converts a date in string format to a DateTime type. “format” specifies the date format and can be any valid date format.
-   **[toDateOnly(epoch)](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_timezone_dateonly_epoch.htm)**
    Converts an epoch to a DateOnly type.
-   **[toDateOnly(string, format)](atlas.en-us.bi_dev_guide_saql.meta/bi_dev_guide_saql/bi_saql_functions_timezone_dateonly_string.htm)**
    Converts a date in string format to a DateOnly type. format specifies the date format and can be any valid date format.
