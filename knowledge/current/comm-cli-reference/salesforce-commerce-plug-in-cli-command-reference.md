# Salesforce Commerce Plug-In CLI Command Reference

The command reference contains information about the Salesforce CLI commands for the Commerce plug-in and their parameters.

This version of the command reference includes details about version 7.159.0 of the Salesforce CLI and the following plug-in versions:

-   @salesforce/commerce version 238.0.15

For information about installing Salesforce CLI, see the [Salesforce CLI Setup Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.sfdx_setup.meta/sfdx_setup/sfdx_setup_install_cli.htm "HTML (New Window)").

For information about Salesforce CLI changes, see the [Salesforce CLI Release Notes](https://github.com/forcedotcom/cli/blob/main/releasenotes/README.md "HTML (New Window)").

-   **[commerce Namespace](atlas.en-us.comm_cli_reference.meta/comm_cli_reference/comm_cli_reference_commerce.htm)**
    Commands for 1commerce developers.
-   **[Help for Salesforce CLI Commands](atlas.en-us.comm_cli_reference.meta/comm_cli_reference/comm_cli_reference_help.htm)**
    The \-h | \--help parameter shows details about Salesforce CLI topics and their commands.
-   **[CLI Deprecation Policy](atlas.en-us.comm_cli_reference.meta/comm_cli_reference/sfdx_dev_cli_deprecation.htm)**
    Salesforce deprecates CLI commands and flags when, for example, the underlying API changes.
-   **[Discover Salesforce Plugins](atlas.en-us.comm_cli_reference.meta/comm_cli_reference/sfdx_setup_other_plugins.htm)**
    Check out these other plugins that work with specific Salesforce features. These plugins are created by Salesforce.
