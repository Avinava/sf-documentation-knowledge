# Rebate Management Metadata API Types

Metadata API enables you to access some types and feature settings that you can customize in the user interface. For more information about Metadata API and to find a complete reference of existing metadata types, see **[Metadata API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/)**.

-   **[ObjectHierarchyRelationship](atlas.en-us.channel_revenue_management_dev_guide.meta/channel_revenue_management_dev_guide/rebate_objecthierarchyrelationshipsettings_metadata_api.htm)**
    Represents an organization’s custom field mappings between a reference object and eligible rebate types. Fields can be mapped from Opportunity, OpportunityLineItem, Quote, QuoteLineItem, and Contract to TransactionJournal.
