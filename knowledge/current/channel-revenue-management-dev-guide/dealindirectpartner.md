# DealIndirectPartner

Represents an indirect partner’s involvement in a deal. This object is available in API version 63.0 and later.

A DealIndirectPartner record can be created manually or through automation when a partner is associated with an opportunity, lead, or account, capturing role and contact information.

## Supported Calls

create(), delete(), describeLayout(), describeSObjects(), getDeleted(), getUpdated(), query(), retrieve(), search(), undelete(), update(), upsert()
