# Response Bodies

Here’s a list of Rebate Management Business API response bodies.

-   **[Program Rebate Type Validation Response](atlas.en-us.channel_revenue_management_dev_guide.meta/channel_revenue_management_dev_guide/connect_responses_program_rebate_type_validation_response.htm)**
    The result of validating a program rebate type. It includes validation status, messages, and any identified configuration issues.
-   **[Program Rebate Type Validation Error Response](atlas.en-us.channel_revenue_management_dev_guide.meta/channel_revenue_management_dev_guide/connect_responses_program_rebate_type_validation_error_response.htm)**
    Represents error details for program rebate type setup validation.
-   **[Program Rebate Type Validation Results Response](atlas.en-us.channel_revenue_management_dev_guide.meta/channel_revenue_management_dev_guide/connect_responses_program_rebate_type_validation_results_response.htm)**
    Represents the results details for program rebate type setup validation.
