# Salesforce Scheduler Resources

Use Salesforce Scheduler REST APIs to get appointment time slots or available service resources based on work type groups and service territories.

-   **[Scheduling](atlas.en-us.api_rest.meta/api_rest/requests_ls_scheduling.htm)**
    Returns a list of available Salesforce Scheduler REST resources and corresponding URIs. This resource is available in REST API version 45.0 and later.
-   **[Get Appointment Slots](atlas.en-us.api_rest.meta/api_rest/requests_ls_getappointmentslots.htm)**
    Returns a list of available appointment time slots for a resource based on given work type group or work type and service territories.
-   **[Get Appointment Candidates](atlas.en-us.api_rest.meta/api_rest/requests_ls_getappointmentcandidates.htm)**
    Returns a list of service resources (appointment candidates) based on work type group or work type and service territories.
