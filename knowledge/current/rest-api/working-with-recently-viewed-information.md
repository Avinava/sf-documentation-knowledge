# Working with Recently Viewed Information

The examples in this section use REST API Query and Recently Viewed resources to programmatically retrieve and update recently viewed record information.

-   **[View Recently Viewed Records](atlas.en-us.api_rest.meta/api_rest/dome_see_recently_viewed.htm)**
    Use the \*\*\* resource to get a list of recently viewed records.
-   **[Mark Records as Recently Viewed](atlas.en-us.api_rest.meta/api_rest/dome_mark_records_as_recently_viewed.htm)**
    To mark a record as recently viewed using REST API, use the Query resource with a FOR VIEW or FOR REFERENCE clause. Use SOQL to mark records as recently viewed to ensure that information such as the date and time the record was viewed is correctly set.
