# AppMenu Items

# AppMenu Items

Accesses App Menu items from the Salesforce app dropdown menu. To retrieve a list of App Menu items, use the GET method. To retrieve the headers returned by a request for App Menu items, use the HEAD method.

-   **[Get AppMenu Items](atlas.en-us.api_rest.meta/api_rest/resources_appmenu_items_get.htm)**  
    Gets a list of the App Menu items in the Salesforce Lightning dropdown menu. This resource is available in REST API version 29.0 and later.
-   **[Return Headers of App Menu Item Requests](atlas.en-us.api_rest.meta/api_rest/resources_appmenu_head.htm)**  
    Returns only the headers that are returned by a GET request for the Salesforce app dropdown menu items. Use this URI to see the header values before you retrieve the content of the resource. This resource is available in REST API version 29.0 and later.
