# Working with Object Metadata

The examples in this section use REST API resources to retrieve object metadata information. For modifying or creating object metadata information, see the [Metadata API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.260.0.api_meta.meta/api_meta/ "HTML (New Window)").

-   **[Get Metadata for an Object](atlas.en-us.api_rest.meta/api_rest/dome_sobject_basic_info.htm)**
    Use the sObject Basic Information resource to get metadata for an object.
-   **[Get Field and Other Metadata for an Object](atlas.en-us.api_rest.meta/api_rest/dome_sobject_describe.htm)**
    Use the \*\*\* resource to retrieve all the metadata for an object, including information about each field, URLs, and child relationships.
-   **[Get Object Metadata Changes](atlas.en-us.api_rest.meta/api_rest/sobject_describe_with_ifmodified_header.htm)**
    Use the sObject Describe resource and the If-Modified-Since HTTP header to determine if object metadata has changed.
