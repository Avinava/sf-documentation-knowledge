# Using Other Tools

# Using Other Tools

Other tools are available to obtain data from your Salesforce org.

If you don’t want to use CURL, you can use other tools to exercise the API. Possible choices include:

-   [Salesforce CLI.](https://developer.salesforce.com/tools/salesforcecli "HTML (New Window)")
-   Postman (a third-party tool). For more information, see the [Postman API Client Trailhead module](https://trailhead.salesforce.com/en/content/learn/modules/postman-api-client "HTML (New Window)").

These tools provide functionality that simplifies the process of creating and sending many API requests.
