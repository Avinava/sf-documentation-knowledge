# AppMenu Mobile Items

Accesses App Menu items from the Salesforce mobile app for Android and iOS and the mobile web navigation menu.

-   **[Get AppMenu Mobile Items](atlas.en-us.api_rest.meta/api_rest/resources_appmenu_items_mobile_get.htm)**
    Gets a list of the App Menu items in the Salesforce mobile app for Android and iOS and the mobile web navigation menu. This resource is available in REST API version 29.0 and later.
-   **[Return Headers of AppMenu Mobile Item Requests](atlas.en-us.api_rest.meta/api_rest/resources_appmenu_mobile_head.htm)**
    Returns only the headers that are returned by a GET request to the Salesforce mobile app for Android and iOS and the mobile web navigation menu. Use this URI to see the header values before you retrieve the content of the resource. This resource is available in REST API version 29.0 and later.
