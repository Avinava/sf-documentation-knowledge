# Get AppMenu Items

Gets a list of the App Menu items in the Salesforce Lightning dropdown menu. This resource is available in REST API version 29.0 and later.

## Syntax

URI

/services/data/vXX.X/appMenu/AppSwitcher/

Formats

JSON, XML

HTTP methods

GET

Authentication

Authorization: Bearer token

Request body

None

Request parameters

None required
