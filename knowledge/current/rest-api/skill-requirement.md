# Skill Requirement

# Skill Requirement

List of skills that are required to complete a particular task for a work type.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| skillId | String | Yes | The skill that is required. |
| SkillLevel | String | No | The level of the skill required. Skill levels can range from zero to 99.99. Depending on your business needs, you might want the skill level to reflect years of experience, certification levels, or license classes. |
