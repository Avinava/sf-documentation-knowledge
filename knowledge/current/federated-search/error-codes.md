# Error Codes

Codes from the federated search connector.

| Error Code | Description |
| --- | --- |
| 400 | Missing or invalid parameter |
| 401 | Invalid authentication |
| 403 | Unrecognized user login |
| 414 | Request-URI too long |

## Optional Codes

| Error Code | Description |
| --- | --- |
| 480 | Unidentified user or unrecognized user login |
| 481 | Missing required parameter |
| 482 | Invalid parameter value |
