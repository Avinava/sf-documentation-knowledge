# Volunteer Management Fields on Other Objects

Volunteer Management includes fields that are available on other Salesforce objects.  These fields are available only in orgs with Nonprofit Cloud when Volunteer Management is  enabled.

| Available in: Lightning Experience |
| --- |
| Available in: Unlimited and Developer Editions. |


-   **[ApplicationForm](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_applicationform.htm)**
    Represents the high level information of a submitted application. This object is available in API version 65.0 and later.
-   **[ApplicationRenderMethod](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_application_render_method.htm)**
    Represents how a part of an application can be rendered. This object is available in API version 64.0 and later.
-   **[ApplicationStageDefinition](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_obect_applicationstagedefinition.htm)**
    Represents a stage of an application. This object is available in API version 64.0 and later.
-   **[DonorGiftSummary](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_donorgiftsummary.htm)**
    Represents gift summaries for accounts and contacts. This object is available in API version 64.0 and later.
-   **[PersonCompetency](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_personcompetency.htm)**
    Represents information about a competency that a person has. This object is available in API version 64.0 and later.
-   **[Position](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_position.htm)**
    Represents a functional role that is characterized by specific duties and responsibilities, and required skills and qualifications. This object is available in API version 64.0 and later.
