# Volunteer Management Standard Objects

The Volunteer Management data model provides objects and fields to manage volunteer programs and improve the volunteer experience for your organization.

| Available in: Lightning Experience |
| --- |
| Available in: Unlimited and Developer Editions. |


Volunteer Management also uses objects from these clouds.

-   [Education Cloud](https://developer.salesforce.com/docs/atlas.en-us.260.0.edu_cloud_dev_guide.meta/edu_cloud_dev_guide/edu_cloud_standard_objects.htm)
-   [Public Sector Solutions](https://developer.salesforce.com/docs/atlas.en-us.260.0.psc_api.meta/psc_api/api_psc_overview.htm)
-   [Financial Services Cloud](https://developer.salesforce.com/docs/atlas.en-us.260.0.financial_services_cloud_object_reference.meta/financial_services_cloud_object_reference/fsc_api_access_and_usage.htm)
-   [Nonprofit Cloud for Fundraising](https://developer.salesforce.com/docs/atlas.en-us.260.0.nonprofit_cloud.meta/nonprofit_cloud/npc_fundraising_standard_objects.htm)

-   **[JobPositionAssignment](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_object_jobpositionassignment.htm)**
    Represents the assignment of a person to a specific JobPositionShift on a specific day. This object is available in API version 64.0 and later.
-   **[JobPositionQualification](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_jobpositionqualification.htm)**
    Represents a job position qualification. This object is available in API version 64.0 and later.
-   **[JobPositionShift](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_jobpositionshift.htm)**
    Represents a specific work shift associated with a job position or one of its related recurrence schedules. This object is available in API version 64.0 and later.
-   **[PersonLocationAvailability](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_personlocationavailability.htm)**
    Represents a the availability of a person at a specific location.. This object is available in API version 64.0 and later.
-   **[PositionBenefit](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objectspositionbenefit.htm)**
    Represents how a position and benefit are related. This object is available in API version 64.0 and later.
-   **[PositionQualification](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_positionqualification.htm)**
    Represents a position-based qualification. This object is available in API version 64.0 and later.
-   **[VolunteerInitiative](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/volunteer_mgmt_api_objects_volunteer_initiative.htm)**
    Represents all volunteer activities within a single volunteering initiative. This object is available in API version 64.0 and later.
