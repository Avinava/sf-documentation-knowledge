# Grantmaking

This guide provides information about the objects and APIs that Grantmaking uses.

| Available in: Lightning Experience |
| --- |
| Available in: Enterprise, Unlimited, and Developer Editions |


-   **[Grantmaking Object Reference](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/sforce_api_objects_object_reference_parent.htm)**
    The Grantmaking data model provides objects and fields to calculate and manage grants for your organization.
-   **[Grantmaking Fields on Other Objects](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/npc_grantmaking_grantmaking_fields_on_other_objects.htm)**
    Grantmaking includes fields that are available on other Salesforce objects. These fields are available only in orgs with Grantmaking enabled.
-   **[Grantmaking Tooling API Object](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/nonprofit_cloud_tooling_api_parent.htm)**
    Tooling API exposes metadata used in developer tooling that you can access through REST or SOAP. Tooling API’s SOQL capabilities for many metadata types allow you to retrieve smaller pieces of metadata.
-   **[Grantmaking Metadata API Types](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/grantmaking_metadata_api_types.htm)**
    Metadata API enables you to access some types and feature settings that you can customize in the user interface. For more information about Metadata API.
