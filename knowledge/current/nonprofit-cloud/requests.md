# Requests

Here’s a list of Program Management Business API request bodies.

-   **[Benefit Disbursements Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_benefit_disbursements_input.htm)**
    Input representation of the request to create benefit disbursements.
-   **[Benefit Schedule Participants Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_benefit_schedule_participants_input.htm)**
    Input representation of the request to add participants to a benefit schedule.
-   **[Benefit Session Participants Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_benefit_session_participants_input.htm)**
    Input representation of the request to add participants to a benefit session.
-   **[Case Programs Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_case_programs_input.htm)**
    Input representation of the request to add case programs to a benefit session.
-   **[Preview Benefit Session Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_preview_benefit_session_input.htm)**
    Input representation of the benefit scheduling session preview request.
-   **[Program Enrollment Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_program_enrollment_input.htm)**
    Input representation of the enrollee for the program.
-   **[Program Enrollments Input](atlas.en-us.nonprofit_cloud.meta/nonprofit_cloud/connect_requests_program_enrollments_input.htm)**
    Input representation of the request to enroll or update participants in a program.
