# Handling Component Events Dynamically

A component can have its handler bound dynamically via JavaScript. This is useful if a component is created in JavaScript on the client-side.

For more information, see [Dynamically Adding Event Handlers To a Component](atlas.en-us.lightning.meta/lightning/js_cb_dynamic_handler.htm "You can dynamically add a handler for an event that a component fires.").
