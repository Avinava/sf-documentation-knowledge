# Disallow Use of  __proto__ ( ( no-proto ) )

> Prevents using the obsolete __proto__ property, which was deprecated in ECMAScript 3.1. Use Object.getPrototypeOf() instead.

# Disallow Use of \_\_proto\_\_ ( ( no-proto ) )

Prevents using the obsolete \_\_proto\_\_ property, which was deprecated in ECMAScript 3.1. Use Object.getPrototypeOf() instead. This is a standard rule built into ESLint.

For complete details about this rule, including examples, see the corresponding ESLint documentation, [Disallow Use of \_\_proto\_\_ (no-proto)](https://eslint.org/docs/rules/no-proto "HTML (New Window)").
