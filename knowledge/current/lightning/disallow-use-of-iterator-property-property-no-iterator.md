# Disallow Use of \_\_iterator\_\_ Property ( Property ( no-iterator ) )

Prevents using the obsolete \_\_iterator\_\_ property. Use standard JavaScript iterators and generators instead. This is a standard rule built into ESLint.

For complete details about this rule, including examples, see the corresponding ESLint documentation, [Disallow Iterator (no-iterator)](https://eslint.org/docs/rules/no-iterator "HTML (New Window)").
