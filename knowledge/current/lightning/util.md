# Util

Util contains utility methods.

## Methods

-   **[addClass()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_addClass.htm)**
    Adds a CSS class to a component.
-   **[getBooleanValue()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_getBooleanValue.htm)**
    Coerces truthy and falsy values into a native boolean.
-   **[hasClass()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_hasClass.htm)**
    Checks whether the component has the specified CSS class.
-   **[isArray()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_isArray.htm)**
    Checks whether the specified object is an array.
-   **[isEmpty()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_isEmpty.htm)**
    Checks if the object is empty. An empty object’s value is undefined, null, an empty array, or an empty string. An object with no native properties is not considered empty.
-   **[isObject()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_isObject.htm)**
    Checks whether the specified object is a valid object. A valid object is not a DOM element, is not a native browser class (XMLHttpRequest) is not falsey, and is not an array, error, function string or a number.
-   **[isUndefined()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_isUndefined.htm)**
    Checks if the object is undefined.
-   **[isUndefinedOrNull()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_isUndefinedOrNull.htm)**
    Checks if the object is undefined or null.
-   **[removeClass()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_removeClass.htm)**
    Removes a CSS class from a component.
-   **[toggleClass()](atlas.en-us.lightning.meta/lightning/ref_jsapi_util_toggleClass.htm)**
    Toggles (adds or removes) a CSS class from a component.
