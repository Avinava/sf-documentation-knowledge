# Disallow Extending Native Objects ( no-extend-native ) )

Prevent changing the behavior of built-in JavaScript objects, such as Object or Array, by modifying their prototypes. This is a standard rule built into ESLint.

For complete details about this rule, including examples, see the corresponding ESLint documentation, [Disallow Extending of Native Objects (no-extend-native)](https://eslint.org/docs/rules/no-extend-native "HTML (New Window)").
