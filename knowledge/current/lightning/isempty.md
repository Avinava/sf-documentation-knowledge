# isEmpty()

> Checks if the object is empty. An empty object’s value is undefined, null, an
   empty array, or an empty string. An object with no native properties is not considered
   empty.

# isEmpty()

Checks if the object is empty. An empty object’s value is undefined, null, an empty array, or an empty string. An object with no native properties is not considered empty.

## Signature

isEmpty(Object obj)

## Parameters

obj

Type: Object

The object to check.

## Returns

Type: Boolean

Returns true if the object is empty, or false otherwise.
