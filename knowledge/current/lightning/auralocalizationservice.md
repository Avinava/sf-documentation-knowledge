# AuraLocalizationService

AuraLocalizationService provides methods for formatting and localizing dates. Use $A.localizationService to use the methods in AuraLocalizationService.

## Methods

-   **[UTCToWallTime()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_UTCToWallTime.htm)**
    Converts a datetime from UTC to a specified timezone.
-   **[WallTimeToUTC](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_WallTimeToUTC.htm)**
    Converts a datetime from a specified timezone to UTC.
-   **[displayDuration()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDuration.htm)**
    Displays a length of time.
-   **[displayDurationInDays()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDurationInDays.htm)**
    Displays a length of time in days.
-   **[displayDurationInHours()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDurationInHours.htm)**
    Displays a length of time in hours.
-   **[displayDurationInMilliseconds()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDurationInMilliseconds.htm)**
    Displays a length of time in milliseconds.
-   **[displayDurationInMinutes()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDurationInMinutes.htm)**
    Displays a length of time in minutes.
-   **[displayDurationInMonths()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDurationInMonths.htm)**
    Displays a length of time in months.
-   **[displayDurationInSeconds()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_displayDurationInSeconds.htm)**
    Displays a length of time in seconds.
-   **[duration()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_duration.htm)**
    Returns an object representing a length of time.
-   **[endOf()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_endOf.htm)**
    Returns a date that is the end of a unit of time for the given date.
-   **[formatCurrency()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatCurrency.htm)**
    Returns a currency number based on the default currency format.
-   **[formatDate()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatDate.htm)**
    Returns a formatted date.
-   **[formatDateTime()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatDateTime.htm)**
    Returns a formatted date time.
-   **[formatDateTimeUTC()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatDateTimeUTC.htm)**
    Returns a formatted date time in UTC.
-   **[formatDateUTC()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatDateUTC.htm)**
    Returns a formatted date in UTC.
-   **[formatNumber()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatNumber.htm)**
    Returns a formatted number with the default number format.
-   **[formatPercent()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatPercent.htm)**
    Returns a formatted percentage number based on the default percentage format.
-   **[formatTime()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatTime.htm)**

-   **[formatTimeUTC()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_formatTimeUTC.htm)**
    Returns a formatted time in UTC.
-   **[getDateStringBasedOnTimezone](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_getDateStringBasedOnTimezone.htm)**
    Gets a date string based on a time zone.

    Returns the number of days in a duration.

    Returns the default currency format.

    Returns a length of time in hours.
-   **[getLocalizedDateTimeLabels()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_getLocalizedDateTimeLabels.htm)**
    Deprecated. Do not use. Returns date time labels, such as month name, weekday name.

    Returns the number of milliseconds in a duration.

    Returns the number of seconds in a duration.
-   **[getToday](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_getToday.htm)**
    Gets today’s date based on a time zone.
-   **[getYearsInDuration()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_getYearsInDuration.htm)**
    Returns the number of years in a duration.
-   **[isAfter()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_isAfter.htm)**
    Checks if date1 is after date2.
-   **[isBefore()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_isBefore.htm)**
    Checks if date1 is before date2.
-   **[isBetween()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_isBetween.htm)**
    Checks if date is between fromDate and toDate, where the match is inclusive.
-   **[isPeriodTimeView()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_isPeriodTimeView.htm)**
    Deprecated. Do not use. Checks if a datetime pattern string uses a 24-hour or 12-hour time view.
-   **[isSame()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_isSame.htm)**
    Checks if date1 is the same as date2.
-   **[parseDateTime()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_parseDateTime.htm)**
    Parses a string and returns a JavaScript Date.
-   **[parseDateTimeISO8601()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_parseDateTimeISO8601.htm)**
    Parses a date time string in an ISO-8601 format and returns a JavaScript Date.
-   **[parseDateTimeUTC()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_parseDateTimeUTC.htm)**
    Parses a string and returns a JavaScript Date.
-   **[startOf()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_startOf.htm)**
    Returns a date that is the start of a unit of time for the given date.
-   **[toISOString()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_toISOString.htm)**
    Deprecated. Use Date.toISOString() instead.
-   **[translateFromLocalizedDigits()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_translateFromLocalizedDigits.htm)**
    Translate the localized digit string to a string with Arabic digits, if there is any.
-   **[translateFromOtherCalendar()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_translateFromOtherCalendar.htm)**
    Translates the input date from another calendar system (for example, the Buddhist calendar) to the Gregorian calendar based on the locale.
-   **[translateToLocalizedDigits()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_translateToLocalizedDigits.htm)**
    Translate the input string to a string with localized digits, if there is any.
-   **[translateToOtherCalendar()](atlas.en-us.lightning.meta/lightning/ref_jsapi_AuraLocalizationService_translateToOtherCalendar.htm)**
    Translates the input date to a date in another calendar system (for example, the Buddhist calendar) based on the locale.
